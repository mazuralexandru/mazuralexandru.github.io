01. Client-Server Model Explained

Files Modified: None

The web operates on a Client-Server Model.

  • Client: Your device (computer, phone) running a web browser. It requests information.
  • Server: A powerful computer, always connected to the internet, that stores web pages, sites, or applications. It serves information upon request.

Imagine ordering food: You (the client) ask the restaurant (the server) for a specific dish (a web page). The restaurant prepares it and sends it back to you.

This request-response cycle is the core of how you access websites.
