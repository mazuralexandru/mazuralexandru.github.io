53. CSS Colors: currentColor Keyword

CSS provides a special keyword, currentColor, which acts as a variable representing the computed value of the element's own color property. This is incredibly useful for making other properties (like borders or SVG fills) automatically match the text color.

currentColor Explained

  • Value: It's a dynamic value that resolves to whatever the element's computed color property is.
  • Inheritance: If an element doesn't have a color set directly, currentColor will represent the inherited color value.
  • Use Cases:
    • Borders: Making borders automatically match the text color (border: 1px solid currentColor;).
    • Backgrounds: Less common, but possible.
    • Shadows: box-shadow: 2px 2px 5px currentColor;
    • SVG Styling: Setting the fill or stroke of SVG icons to currentColor allows them to adopt the text color of their surrounding context, making them easy to theme.
    • Gradients: Using it within linear-gradient().

In Our Example:

  • We set the button's text color to mediumturquoise.
  • We set border: 2px solid currentColor;. This means the border color will automatically become mediumturquoise.
  • In the :hover state (a pseudo-class we'll cover later, it styles the element when the mouse is over it):
    • We change the text color to darkslategray.
    • Because the border uses currentColor, the border color will automatically update to darkslategray as well, without us needing to explicitly redeclare border-color: darkslategray;!
    • We also set the hover background-color to the original text color (mediumturquoise) for effect.
