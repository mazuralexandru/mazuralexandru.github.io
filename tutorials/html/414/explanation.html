414. filter Property: Syntax

Files Modified: None

The filter property accepts one or more filter functions as its value.

  • Syntax: filter: none | <filter-function>+ ;
    • none: (Default) No filter is applied.
    • <filter-function>: One or more filter functions, separated by spaces.
  • Filter Function Syntax: Generally function-name(value). The value type depends on the function (e.g., length, percentage, angle, number).
  • Multiple Filters: If multiple filter functions are listed, they are applied sequentially in the order they appear.

Example:

/* Apply a single filter */
img {
  filter: grayscale(100%); 
}

/* Apply multiple filters */
img:hover {
  filter: brightness(1.2) contrast(110%) sepia(30%); 
}

/* Remove filters */
img.no-filter {
  filter: none;
}
