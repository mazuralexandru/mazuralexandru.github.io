75. HTML Structural Elements: <br> and <hr>

<br> and <hr> Explained

  • <br> (Line Break):
    • Semantic Meaning: Represents a line break within text where the division is significant (e.g., lines of a poem, parts of an address). It's an empty element (no closing tag needed, though <br/> is also valid).
    • Misuse: Do not use multiple consecutive <br> tags (<br><br><br>) simply to create vertical space between paragraphs or other elements. This is semantically incorrect and bad practice. Use CSS margin or padding for spacing.
    • Styling: Generally not styled directly with CSS.
  • <hr> (Thematic Break / Horizontal Rule):
    • Semantic Meaning: Represents a thematic break between paragraph-level elements within an HTML section (e.g., a change of topic, a scene transition in a story). It implies a shift but doesn't introduce a new section heading. It's also an empty element.
    • Default Styling: Browsers typically render it as a horizontal line with some vertical margin.
    • Styling: Highly customizable with CSS. Common techniques involve removing the default border, setting a height, and using background-color or background-image to create the visual line, as shown in our example with a simple gradient.
    • Decoration vs. Semantics: If you just need a decorative line, it's often better to use CSS border-top or border-bottom on an existing element rather than adding a purely decorative <hr>. Use <hr> when there's a genuine thematic shift in the content.
