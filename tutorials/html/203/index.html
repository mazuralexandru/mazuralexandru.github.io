Link Content & Accessibility

Compare descriptive vs. generic link text:

Bad Examples (Avoid):

  • To learn more about accessibility, click here.
  • Our latest blog post is available. Read more.
  • Download the file: report.pdf

Good Examples (Prefer):

  • Learn more about web accessibility guidelines.
  • Read our latest blog post on responsive design.
  • Download the Quarterly Sales Report (PDF).

Good examples are clear even out of context.
