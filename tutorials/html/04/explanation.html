04. Adding the Page Title

Let's give our page a proper title that appears in the browser tab or window. This is done using the <title> element.

What's New?

We added a <title> element inside the <head> section.

<!-- index.html snippet -->
<head>
  <meta charset="UTF-8">
  <title>My First Web Page</title> <!-- Added title tag -->
</head>
<body>

The <title> Element

  • Purpose: Defines the title of the HTML document. This is crucial metadata.
  • Placement: It must be placed inside the <head> element. A document should have exactly one <title> element.
  • Content: The text between the opening <title> and closing </title> tags is the title itself. Only plain text is allowed here (no other HTML tags).
  • Where it Appears:
    • Browser tab or window title bar.
    • Browser bookmarks/favorites.
    • Search engine results pages (SERPs) - often used as the main clickable link.
  • Importance:
    • Usability: Helps users identify the page among multiple tabs.
    • Accessibility: Screen readers announce the title when the page loads, providing context.
    • SEO: Search engines weigh the title heavily for understanding the page's topic.

Best Practices for Titles:

  • Be descriptive and concise.
  • Make it unique for each page on your website.
  • Consider including relevant keywords, but prioritize clarity for users.
