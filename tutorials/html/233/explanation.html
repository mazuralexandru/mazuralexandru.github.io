233. Writing Effective alt Text

Writing good alt text is more art than science, but crucial for accessibility and usability. The key is context. Ask yourself: Why is this image here? What information does it convey that a user needs to understand the surrounding content?

Key Principles:

  1. Be Accurate and Concise: Describe what the image is or shows efficiently. Avoid unnecessary words.

    • Okay: alt="A photograph showing a brown dog running on green grass"
    • Better: alt="Brown dog running on grass"

  2. Focus on Content/Function, Not Appearance: Describe the information or purpose, not just the visual details, unless those details are the relevant information.

    • Image: A red "Warning" icon.
    • Bad: alt="Red octagon with exclamation mark" (Describes appearance)
    • Good: alt="Warning:" (Describes function/meaning)

  3. Consider Context: The same image might need different alt text depending on the surrounding content.

    • Image: Graph showing rising sales.
    • Context 1 (Caption below describes details): alt="Bar chart showing quarterly sales." (Concise description)
    • Context 2 (No caption, graph illustrates point in text): alt="Bar chart demonstrating significant sales increase in Q4." (Summarizes the conveyed info)

  4. Avoid Redundancy: If the surrounding text fully describes the image, repeating that information in alt can be annoying for screen reader users. In such cases, a decorative alt="" might be appropriate (see next lesson).

  5. Skip "Image of...", "Picture of...": Screen readers announce the element as an image. Adding these phrases is usually redundant.

    • Bad: alt="Picture of the Eiffel Tower"
    • Good: alt="Eiffel Tower at sunset"

  6. Transcribe Text within Images: If an image contains important text (e.g., a graphic button, a diagram label), the alt text must include that text verbatim. However, avoid using images of text whenever possible. Use actual HTML text and style it with CSS for better accessibility, SEO, and maintainability.

  7. Functional Images (Links/Buttons): As covered in Module 4, if the image is the only content of a link or button, the alt text must describe the link's destination or the button's action.

    • <a href="search.html"><img src="magnifying-glass.svg" alt="Search Site"></a>

  8. Use Punctuation: End alt text with a period or appropriate punctuation. This helps screen readers pause correctly, improving the listening experience.

✅ Observation: No code changes. Review the principles above. Look at the alt text in our current example: alt="Placeholder image: 150x75 pixels, sky blue background, white text saying Sample Image". Is this good alt text? It accurately describes what the image shows, which is appropriate for a placeholder. If it were a real image (e.g., a company logo), the alt text would need to describe that content instead (e.g., alt="Example Corp Logo").

✨ Key Takeaway: Writing effective alt text requires describing the image's content and purpose accurately and concisely within its specific context, avoiding redundancy, and focusing on function for interactive images.
