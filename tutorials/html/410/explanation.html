410. Decorative Styling: border-radius

Files Modified: style.css

We can use border-radius to round the corners of images, just like other elements.

  • Effect: Rounds the corners formed by the border. Also clips the background and content to the rounded shape.
  • Circular Images: A common technique is to use border-radius: 50%; on a square image (or an image container with equal width and height) to create a circular frame.

✅ Observe style.css:

  • We've added border-radius: 15px; to the .block-image rule.
.block-image {
    display: block; 
    max-width: 100%; 
    height: auto; 
    width: 300px; 
    margin: 2em auto 3em auto; 
    border: 4px solid steelblue; 
    padding: 10px; 
    background-color: #444; 
    border-radius: 15px; /* Round the corners */
}

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview: The corners of the image's border (and the dark gray background within the padding) should now be rounded with a 15px radius.
