110. HTML Comments: Nesting

Files Modified: None

According to the HTML standard, HTML comments cannot be nested.

This means you cannot put one comment block inside another one:

<!-- Outer Comment Starts
  <!-- Inner Comment Starts -->  <!-- This is INVALID -->
Outer Comment Ends -->

The browser will likely interpret the first --> it encounters as the end of the outer comment, leaving the rest (Outer Comment Ends -->) as potentially visible, invalid content on the page.

Workaround: If you need to comment out a block of code that already contains comments, you might need to temporarily remove or modify the inner comments, or use server-side logic or build tools to handle conditional inclusion instead of relying on nested HTML comments.

Rule: Avoid nesting HTML comments.
