337. Removing Default List Styles

Files Modified: style.css

While browsers provide default styling for lists (bullets, numbers, indentation), you'll often want to remove these defaults to apply your own custom styling, especially for navigation menus or other non-standard list presentations.

Common Reset: A common starting point is to remove the default markers and indentation:

ul, ol {
  list-style: none; /* Removes bullets/numbers */
  padding-left: 0;  /* Removes default indentation (often padding) */
  margin-top: 0;    /* Removes default top margin */
  margin-bottom: 0; /* Removes default bottom margin */
}

/* You might also reset margin on li if needed */
li {
  margin: 0; 
}
  • list-style: none;: This property (or specifically list-style-type: none;) removes the bullet points or numbers.
  • padding-left: 0;: Browsers typically use left padding on the <ul> or <ol> element for indentation. Setting this to 0 removes that space.
  • margin-top: 0; / margin-bottom: 0;: Removes default vertical spacing often applied to list containers.
