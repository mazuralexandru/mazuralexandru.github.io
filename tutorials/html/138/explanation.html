138. <link rel="stylesheet"> Explained

Files Modified: None

The rel="stylesheet" attribute is the key that tells the browser how to treat the file linked by the href attribute.

  • rel: Stands for "relationship". It describes the relationship between the linking document (the HTML file) and the linked resource (the file specified in href).
  • stylesheet: This specific value indicates that the linked resource is a Cascading Style Sheet file containing rules that should be applied to the current document.

Without rel="stylesheet", the browser might fetch the file specified in href, but it wouldn't know to interpret it as CSS and apply its rules to style the page. Other rel values exist for different purposes (e.g., icon for favicons, preload for performance optimization), but stylesheet is essential for CSS.
