349. Controlling List Indentation: Customizing padding-left

Files Modified: style.css

To customize the indentation of your lists, you simply override the default browser padding by setting your own padding-left value on the <ul> or <ol> element.

  • Units: You can use any valid length unit (px, em, rem, etc.). Using relative units like em or rem is often preferred so the indentation scales appropriately with the font size.
  • margin-left: While padding-left controls the indentation relative to the markers (when list-style-position: outside), you might also use margin-left on the list element if you need to control the spacing between the entire list block (including its padding area) and the edge of its container.

✅ Observe style.css:

  • We already set padding-left: 2em; for the ul elements.
  • Let's add a different padding to the ol elements to restore some indentation, but less than the ul. We'll also restore a default marker type.
/* Style Ordered Lists */
ol {
   /* list-style: none; <- Removed reset */
   list-style-type: decimal; /* Restore default numbers */
   padding-left: 1.5em; /* Add custom indentation */
   margin-bottom: 1em; 
}

✨ Observe the Preview:

  • The ordered lists ("Morning Routine", "Top 3") should now be indented again, but likely slightly less than the unordered list ("Favorite Fruits") because we used 1.5em vs 2em.
  • The "Morning Routine" list should have decimal numbers (1, 2, 3).
  • The "Top 3" list should still show Roman numerals (III, IV, V) because the HTML attribute overrides the CSS list-style-type.

This shows how padding-left is the primary way to control list indentation.
