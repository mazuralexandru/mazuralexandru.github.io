90. Applying CSS - Method 1: Inline Styles

Files Modified: None

Before we add rules to our external style.css file (which is the preferred method), let's look at another way to apply CSS: Inline Styles.

Inline styles apply CSS declarations directly to a single HTML element using the style attribute.

  • Attribute: style is a global HTML attribute, meaning it can be added to almost any HTML element.
  • Value: The value of the style attribute is a string containing one or more CSS declarations (property: value;), separated by semicolons, just like in a CSS declaration block, but without the curly braces or selector.
