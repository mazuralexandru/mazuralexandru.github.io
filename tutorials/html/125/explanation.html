125. Applying CSS - Method 2: Internal Stylesheets

Files Modified: None

We've seen how to apply styles directly to an element using the style attribute (inline styles), and we know that linking an external .css file is the preferred method for larger projects.

There's a third way: Internal Stylesheets. This method involves embedding CSS rules directly within the HTML document itself, but unlike inline styles, they are grouped together in one place and can target multiple elements using selectors.

This approach sits between inline styles and external stylesheets in terms of organization and scope. Let's explore how it works.
