152. Reset: Removing Placeholder Links

We've established the basic <a> structure, the essential role of the href attribute, and the difference between placeholder anchors and placeholder links (href="#").

Now, to move on to creating functional links that navigate to actual destinations, let's clear out the placeholder examples we just created.

Why Remove Code?

  • Focus: Shift from understanding placeholders to understanding actual URL types (Absolute, Relative, Fragment).
  • Clarity: Remove potentially confusing non-functional or minimal-function links before introducing links that cause real navigation.
  • Clean Slate: Prepare the <body> for examples using specific URL formats in the href attribute.

Code Change:

We have removed the paragraphs containing the placeholder <a> examples (both with and without href) from index.html.

<body>
  <h1>Links and Navigation</h1>
  <p>Preparing to add functional links with actual URLs.</p>

  <!-- <p>2. Placeholder Link (href="#"): <a href="#">Goes Nowhere (Top)</a></p> -->
  <!-- <p>3. Functional Link (href="..."): </p> -->

  <p>Next: Absolute URLs.</p>

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>
