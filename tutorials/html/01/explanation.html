01. Introduction: What is HTML? (Part 1: Definition)

Welcome to the world of web development! Our journey begins with the absolute foundation: HTML.

What is HTML?

HTML stands for HyperText Markup Language. Let's break that down:

  • HyperText: This refers to text that contains links, called hyperlinks, allowing you to jump non-linearly to other text, documents, or resources on the web. Think of how you navigate websites by clicking links â€“ that's hypertext in action!
  • Markup: This means using special codes, called tags, to "mark up" or describe different pieces of content. These tags tell the web browser about the structure and meaning of the content. For example, a tag might indicate "this text is a main heading" or "this text is a paragraph."
  • Language: HTML is a formal language with specific rules (syntax) that computers, particularly web browsers, understand.

Key Distinction:

It's crucial to understand that HTML is a markup language, not a programming language.

  • Markup Languages (like HTML): Describe the structure and meaning of content. They define what something is (e.g., a heading, a list, an image).
  • Programming Languages (like JavaScript): Define logic and behavior. They tell the computer how to do things (e.g., calculate a sum, react to a button click, fetch data).
