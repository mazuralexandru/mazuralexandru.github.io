443. background-size: Syntax

Files Modified: None

Let's review the syntax options for background-size:

  • One Value:
    • auto: Uses the image's intrinsic dimensions.
    • <length>: Sets the width; height becomes auto (maintains aspect ratio).
    • <percentage>: Sets the width relative to the background positioning area; height becomes auto.
    • cover: Scales to cover the area.
    • contain: Scales to fit within the area.
  • Two Values:
    • auto auto: Same as single auto.
    • <length> <length>: Sets explicit width and height.
    • <percentage> <percentage>: Sets explicit width and height relative to the background positioning area.
    • <length> auto: Sets width; height maintains aspect ratio.
    • auto <length>: Sets height; width maintains aspect ratio.
    • <percentage> auto: Sets width relative to container; height maintains aspect ratio.
    • auto <percentage>: Sets height relative to container; width maintains aspect ratio.
  • Multiple Backgrounds: Use comma-separated values, one for each layer defined in background-image.
    • background-size: auto, 50px, cover; (Applies to first, second, third layers respectively).

Choosing the right value depends on whether you need fixed dimensions, proportional scaling relative to the container, or full covering/containment.
