324. overflow Value: hidden

Files Modified: style.css

Setting overflow: hidden; changes how overflowing content is handled.

  • Behavior: Any content that extends beyond the element's padding edge is clipped and becomes invisible. No scrollbars are provided to access the hidden content.
  • Effect: Prevents content from spilling out and overlapping other elements, but makes the overflowing content inaccessible to the user.
  • Use Case: Useful when you intentionally want to hide overflow without providing scrolling, perhaps for decorative elements with fixed sizes or in certain layout techniques. Also required (along with white-space: nowrap;) for text-overflow: ellipsis; to work.

✅ Observe style.css:

  • We've added overflow: hidden; to the .overflow-box rule.
.overflow-box {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid orange;
  background-color: rgba(0, 0, 100, 0.2);
  margin-bottom: 1em; 
  overflow: hidden; /* Clip overflowing content */
}

✨ Observe the Preview: The text that was previously spilling out of the bottom of the orange-bordered box should now be cut off (clipped) at the bottom edge of the box's padding area. The hidden text is no longer visible.
