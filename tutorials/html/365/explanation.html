365. Special Link Types: mailto: & tel:

Files Modified: index.html

The href attribute can use special URL schemes beyond http or https to trigger specific browser or operating system actions. Two common ones are mailto: and tel:.

  • mailto: (Email Links):

    • Purpose: Creates a link that, when clicked, typically opens the user's default email client with the specified address pre-filled.
    • Syntax: href="mailto:email@example.com"
    • Optional Parameters: You can add parameters to pre-fill other fields using a ? followed by key=value pairs, separated by &. Spaces and special characters in values need to be URL-encoded (e.g., space becomes %20).
      • cc=address@example.com
      • bcc=address@example.com
      • subject=Your%20Subject%20Here
      • body=Your%20message%20body%20here.
    • Example: <a href="mailto:info@example.com?subject=Website%20Inquiry&body=Hello,">Email Us</a>

  • tel: (Telephone Links):

    • Purpose: Creates a link that, when clicked on a device with telephony capabilities (like a smartphone), initiates a call to the specified phone number.
    • Syntax: href="tel:+1-555-123-4567" (Use the international dialing format with + and country code).
    • Example: <a href="tel:+18005551212">Call Toll-Free: +1-800-555-1212</a>
