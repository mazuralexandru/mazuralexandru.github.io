343. List Marker Position (list-style-position): inside

Files Modified: style.css

Let's change the marker position for the unordered list.

  • Value: inside
  • Behavior:
    • The marker is treated as if it were part of the list item's inline content, appearing at the very beginning of the first line.
    • If the list item's text wraps onto multiple lines, subsequent lines will align with the start of the marker itself, not the start of the text after the marker. The marker effectively becomes part of the text block.

✅ Observe style.css:

  • We've added list-style-position: inside; to the ul rule.
/* Style Unordered Lists */
ul {
  list-style-type: square; 
  padding-left: 2em; 
  margin-bottom: 1em; 
  list-style-position: inside; /* Place marker inside text flow */
}

✨ Observe the Preview:

  • Look at the "Favorite Fruits" list. The square markers should now appear further to the right, seemingly indented along with the first line of text for each item.
  • If an item were long enough to wrap, the second line would start directly underneath the square marker, not underneath the first letter of the text.
  • This is less common for standard bulleted lists but can be useful in specific layout scenarios.
