Spacing Properties Explained

  • letter-spacing:
    • Purpose: Adjusts the space between individual characters of text (also known as "tracking" in typography).
    • Values: Accepts <length> units (px, em, rem).
      • Positive values increase space, making text wider.
      • Negative values decrease space (kerning), making text tighter.
    • Use Cases: Can be used subtly for headings or logos, but excessive positive or negative spacing drastically reduces readability for body text. normal is the default.
  • word-spacing:
    • Purpose: Adjusts the space in addition to the default space between words.
    • Values: Accepts <length> units (px, em, rem).
      • Positive values add extra space between words.
      • Negative values reduce space between words.
    • Use Cases: Generally used sparingly. Can sometimes be adjusted subtly for justified text or specific design effects. normal is the default (represents the standard space character width).
