187. Commonly Inherited Properties

Files Modified: None

Not all CSS properties are inherited. Inheritance primarily applies to properties related to text and visibility.

Commonly Inherited Properties:

  • Font Properties:
    • font-family
    • font-size
    • font-weight
    • font-style
    • font-variant
    • font (shorthand)
  • Text Properties:
    • color
    • line-height
    • letter-spacing
    • word-spacing
    • text-align
    • text-indent
    • text-transform
    • white-space
  • List Properties:
    • list-style (and its longhand properties like list-style-type, list-style-image, list-style-position)
  • Table Borders:
    • border-collapse
    • border-spacing
  • Other:
    • visibility
    • cursor
    • quotes
