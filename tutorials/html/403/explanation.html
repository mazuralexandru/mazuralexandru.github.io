403. Images as Inline Elements & vertical-align

Files Modified: index.html, style.css

By default, <img> is an inline-level element (specifically, a replaced inline element). This means it flows within lines of text. However, its default vertical alignment can sometimes look awkward.

  • Default Alignment: Usually aligns the bottom of the image with the baseline of the surrounding text. This often leaves a small gap below the image.
  • vertical-align Property: Controls the vertical positioning of an inline-level element relative to its line box. We saw this briefly with the link icon earlier. Common values include baseline (default), middle, top, bottom, text-top, text-bottom.

✅ Observe style.css:

  • We've removed the previous .sample-image styles.
  • We've added a new rule for img.inline-example setting vertical-align: middle; and a fixed height.
/* Universal Box Sizing Reset */
html {
  box-sizing: border-box; 
}
*, *::before, *::after { 
  box-sizing: inherit; 
}

/* Basic H1 style for context */
h1 {
    text-align: center;
    margin-bottom: 1em;
}

/* Inline Image Alignment */
img.inline-example {
    height: 1.2em; /* Size relative to text */
    vertical-align: middle; /* Align vertically with text */
    margin: 0 0.25em; /* Add small horizontal space */
    /* Note: width and vertical margins are mostly ignored on inline */
}

✅ Switch to index.html:

  • We've added a paragraph containing text interspersed with <img> elements using the inline-example class.
    <h1>Image Alignment</h1>

    <p>
        Text flows around inline images. This image 
        <img src="icon1.png" alt="Icon 1" class="inline-example" width="16" height="16"> 
        should align nicely with the text using vertical-align. Here's another one 
        <img src="icon2.png" alt="Icon 2" class="inline-example" width="16" height="16"> 
        to show how they sit on the line.
    </p>

✨ Observe the Preview:

  • You should see the paragraph text with two (broken) image icons within it.
  • Because <img> is inline and we've set vertical-align: middle;, the icons should appear roughly vertically centered within the line of text, rather than sitting awkwardly on the baseline.
  • Notice that setting width or vertical margin/padding on these inline images would have little to no effect.
