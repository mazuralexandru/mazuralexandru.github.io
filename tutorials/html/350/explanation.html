350. Styling List Items (<li>): Spacing

Files Modified: style.css

Often, you'll want to control the vertical spacing between individual list items (<li>). The default spacing is usually minimal or non-existent.

  • Method: The standard way to add space between list items is to apply margin-top or (more commonly) margin-bottom to the <li> elements themselves.
  • Units: Again, relative units like em or rem are often good choices so the spacing scales with the text size.

✅ Observe style.css:

  • We've added a new rule targeting all li elements (inside both ul and ol).
  • We're adding a small bottom margin to each list item.
/* Style List Items */
li {
    margin-bottom: 0.5em; /* Add space below each list item */
}

✨ Observe the Preview:

  • You should now see a small amount of vertical space appear between each item in all the lists ("Favorite Fruits", "Morning Routine", "Top 3", and the nested apple list). This generally improves readability compared to having the items tightly packed together.
