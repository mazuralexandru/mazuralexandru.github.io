293. Rounded Corners (border-radius): Shorthand Values

Files Modified: None

Let's quickly review the shorthand logic for border-radius, as it follows the same pattern as padding and margin:

  • border-radius: A; -> All corners = A
  • border-radius: A B; -> Top-Left/Bottom-Right = A, Top-Right/Bottom-Left = B
  • border-radius: A B C; -> Top-Left = A, Top-Right/Bottom-Left = B, Bottom-Right = C
  • border-radius: A B C D; -> Top-Left = A, Top-Right = B, Bottom-Right = C, Bottom-Left = D (Clockwise)

This allows for creating various shapes, from uniformly rounded rectangles to pill shapes or elements with only specific corners rounded.
