170. The target Attribute: Introduction

So far, when we clicked our links (like the absolute URL example), the linked page replaced the current page in the same browser tab or window. This default behavior can be changed using the target attribute on the <a> element.

The target Attribute:

  • Definition: Specifies the browsing context (window, tab, or iframe) in which the linked document should be opened when the hyperlink is followed.
  • Purpose: To control whether a link opens in the current tab/window or a new one, or potentially targets a specific frame.
  • Syntax: target="value"
    • Added inside the opening <a> tag.
    • The value is typically a keyword starting with an underscore (_) or the name of a frame.
