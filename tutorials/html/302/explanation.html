302. Margin Shorthand Property (margin)

Files Modified: style.css

Just like padding, the margin shorthand property allows setting margins on all four sides concisely.

  • Purpose: A more concise way to define margins.
  • Syntax: The number of values provided determines how they are applied (exactly the same logic as padding):
    • One Value: margin: value; (All four sides)
    • Two Values: margin: value1 value2; (Top/Bottom, Left/Right)
    • Three Values: margin: value1 value2 value3; (Top, Left/Right, Bottom)
    • Four Values: margin: value1 value2 value3 value4; (Top, Right, Bottom, Left - clockwise)

✅ Observe style.css:

  • We've replaced the four longhand margin-* properties with a single margin shorthand using four values to achieve the same result as before.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  /* Replaced longhand with shorthand */
  margin: 2em 50px 3em 10px; /* Top, Right, Bottom, Left */
  border-bottom: 4px solid red; 
  border-radius: 15px; 
}

✨ Observe the Preview: The spacing around the box should be identical to the previous step. The margin shorthand simply provides a more compact way to write the same rules.
