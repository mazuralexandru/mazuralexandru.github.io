241. <small>, <sup>, <sub>: CSS Styling

Files Modified: style.css

You can, of course, override the default styling of these elements using CSS.

✅ Observe style.css:

  • We've added rules to customize small, sup, and sub.
  • We make small slightly opaque.
  • We change the color and slightly adjust the default vertical alignment for sup and sub using position: relative and top/bottom.
/* Styling Other Inline Semantics */
small {
  opacity: 0.8; /* Make small text slightly faded */
}

sup {
  color: skyblue;
  font-size: 0.75em; /* Ensure it's smaller */
  vertical-align: baseline; /* Reset default */
  position: relative; /* Allow offset */
  top: -0.5em; /* Adjust vertical position */
}

sub {
  color: lightcoral;
  font-size: 0.75em; /* Ensure it's smaller */
  vertical-align: baseline; /* Reset default */
  position: relative; /* Allow offset */
  bottom: -0.3em; /* Adjust vertical position */
}

✨ Observe the Preview:

  • The "small print" text should now be slightly faded (opacity).
  • The superscript "st" should now be sky blue and potentially positioned slightly differently due to the explicit offset.
  • The subscript "2" should now be light coral and potentially positioned slightly differently.

This demonstrates customizing the appearance beyond the browser defaults while still using the semantically appropriate HTML tags.
