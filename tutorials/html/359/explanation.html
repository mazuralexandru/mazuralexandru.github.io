359. href Attribute Recap

Files Modified: index.html

The href (Hypertext Reference) attribute is the most crucial attribute for the <a> element. It specifies the destination URL of the hyperlink.

  • Purpose: Defines where the link points to. Without a valid href, the <a> element represents a placeholder link and isn't interactive.
  • Value Types:
    • Absolute URL: A full web address, including the protocol (http:// or https://). Used for linking to external websites.
      • Example: <a href="https://www.google.com">Google</a>
    • Relative URL: A path relative to the current HTML document's location. Used for linking to other pages or files within the same website.
      • Example: <a href="about.html">About</a> (links to about.html in the same folder)
      • Example: <a href="/images/logo.png">Logo</a> (links to logo.png in the images folder at the site root)
      • Example: <a href="../contact.html">Contact</a> (links to contact.html in the parent folder)
    • Fragment Identifier: A hash (#) followed by the id of an element on the same page. Used for linking to specific sections within the current document.
      • Example: <a href="#section-two">Go to Section 2</a> (links to <div id="section-two">)
    • Other Protocols: Can also use mailto:, tel:, or even javascript: (though the latter is generally discouraged for accessibility and maintainability reasons).
