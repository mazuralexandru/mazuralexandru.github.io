113. Ordered Lists (<ol>): Combining start and reversed

The start and reversed attributes can be used together on an <ol> element.

Interaction:

  • When both reversed and start are present, the start attribute specifies the number for the first item in the list (which will be the highest number in the countdown).
  • The list then counts down sequentially from that start value.

Example Scenario:

Imagine you want to list the Top 3 items, but you want to start the numbering explicitly from 3 and count down (3, 2, 1). You would use start="3" and reversed. If you wanted to show items ranked 10, 9, and 8, you would use start="10" and reversed.

Code Change:

We have added both reversed and start="10" attributes to the <ol> tag in index.html.

<p>Combining 'start' and 'reversed':</p> <!-- Adjusted P -->

<ol reversed start="10"> <!-- Added reversed and start -->
  <li>Item numbered 10</li> <!-- Text updated -->
  <li>Item numbered 9</li>
  <li>Item numbered 8</li>
</ol>

<p>Starts countdown from the 'start' value.</p> <!-- Adjusted P -->

✅ Observation:

  1. Examine the <ol> tag in index.html with both reversed and start="10".
  2. Look at the browser preview. The list markers should now count down starting from 10: 10., 9., 8..

✨ Key Takeaway: When reversed and start are used together on <ol>, the list counts down starting from the value specified in the start attribute.
