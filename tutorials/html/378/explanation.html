378. Basic Structure: Building a Table with Multiple Rows

Now that we have one complete row (<tr> containing <td> elements), we can build a table with multiple rows by simply adding more <tr> elements within the <table>.

Structure:

  • Place subsequent <tr>...</tr> blocks one after another, directly inside the <table> (or later, inside <thead>, <tbody>, <tfoot>).
  • Each <tr> represents a new horizontal record in the table.

Code Change:

We've added a second <tr> element, also containing two <td> cells, below the first <tr> in index.html.

<!-- index.html snippet -->
  <table>
    <tr>
      <td>Cell 1, Row 1</td>
      <td>Cell 2, Row 1</td>
    </tr>
    <tr> <!-- Added second row -->
      <td>Cell 1, Row 2</td>
      <td>Cell 2, Row 2</td>
    </tr>
  </table>

✅ Observation:

  1. Examine the index.html code. You now have two <tr> elements inside the <table>, each containing two <td> elements.
  2. Look at the browser preview. You should see:
    • "Cell 1, Row 1" "Cell 2, Row 1" on the first line.
    • "Cell 1, Row 2" "Cell 2, Row 2" on the second line, directly below the first row.
    • The browser has rendered a simple 2x2 grid of data cells.

✨ Key Takeaway: Add multiple <tr> elements within a <table> (or tbody, etc.) to create multiple rows of data. Each <tr> starts a new line in the table grid.
