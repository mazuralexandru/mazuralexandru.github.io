28. CSS Selectors: Introduction to ID Selectors

While classes are for styling multiple elements similarly, ID selectors target one specific, unique element on the page.

The HTML id Attribute

  • Purpose: Assigns a unique identifier to an element within an HTML document.
  • Uniqueness Constraint: Unlike classes, an id value must be unique on the entire page. No two elements can have the same id. This is crucial.
  • Syntax: <element id="unique-name">.
  • Naming Conventions: Similar to classes (lowercase, hyphens), but must be unique. main-content, primary-navigation, error-summary are common examples.
  • Primary Uses:
    • Linking: Creating internal page links (e.g., <a href="#main-content">Go to main content</a>).
    • JavaScript Hooks: Providing a reliable way for JavaScript to find and manipulate a specific element (document.getElementById('main-content')).
    • CSS Targeting (Use Carefully): Targeting a unique element for specific styling.

We've added <div id="main-content"> containing a paragraph. Like with classes, adding the id attribute alone doesn't apply style.
