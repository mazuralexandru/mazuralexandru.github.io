11. Controlling Font Size

Let's add another common style property to our inline style: font-size.

What's New?

We added a third declaration, font-size: 1.2em;, to the style attribute on the <p> tag.

<!-- index.html snippet -->
<body>
  <!-- This is our first paragraph -->
  <p style="color: lightcoral; background-color: #444; font-size: 1.2em;">Hello, World!</p> <!-- Added font-size -->

  <!-- Commented out paragraph remains -->
  <!-- <p>This paragraph is commented out and won't appear.</p> -->
</body>

CSS font-size Property

  • The font-size property controls the size of the text within an element.

  • It accepts various types of values:

    • Keywords: small, medium, large, x-large, etc. (Relative to the browser's default settings, less precise).
    • Absolute Units:
      • px (pixels): Fixed size, common but can cause accessibility issues if users need to resize text. 16px is often the browser default.
    • Relative Units (Generally Preferred):
      • % (percentage): Relative to the parent element's font size. 120% is similar to 1.2em.
      • em: Relative to the parent element's font size (or the element's own font size for some properties). 1em is the default size, 2em is double, 0.8em is smaller. Can compound if nested.
      • rem (root em): Relative to the font size of the root element (<html>). Often preferred over em for consistency, as it avoids compounding issues.

  • In this example, 1.2em makes the font size 1.2 times the size it would otherwise inherit (likely from the <body>, which inherits the browser default).
