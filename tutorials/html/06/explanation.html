06. Adding HTML Comments

As code gets more complex, it's helpful to leave notes for yourself or other developers. HTML comments allow you to do this.

What's New?

We've added text enclosed in <!-- and -->.

<!-- index.html snippet -->
<body>
  <!-- This is our first paragraph -->
  <p>Hello, World!</p>

  <!-- We can add comments anywhere outside of tags -->
  <!-- <p>This paragraph is commented out and won't appear.</p> -->
</body>

HTML Comments (<!-- ... -->)

  • Purpose:
    • Add explanatory notes within the HTML code that are ignored by the browser during rendering.
    • Temporarily disable ("comment out") sections of HTML code without deleting them, useful for testing or debugging.
  • Syntax: Comments start with <!-- and end with -->. Everything in between is considered part of the comment.
  • Multi-line: Comments can span multiple lines.
  • Placement: You can place comments almost anywhere in your HTML document, except inside HTML tags themselves.
  • Visibility: Comments are not displayed on the web page itself, but they are visible to anyone who views the page source code (Right-click -> View Page Source in most browsers). Therefore, never put sensitive information (like passwords or private data) in HTML comments.
  • Nesting: Technically, you cannot nest comments according to the HTML standard (e.g., <!-- A <!-- B --> C --> is problematic).
