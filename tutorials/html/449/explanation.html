449. background-attachment: local

Files Modified: index.html, style.css

The local value is specifically for elements that themselves have scrollbars.

  • Behavior: The background image is positioned relative to the element's content. If the element has scrollbars (e.g., due to overflow: scroll or auto), the background image will scroll along with the content inside the element.
  • Contrast with scroll: With scroll (the default), if an element has scrollbars, its background image remains fixed relative to the element's border box and does not scroll with the content within the element. local makes the background scroll with the internal content.

/* Background Attachment: local Example */
.scroll-box {
    height: 200px; /* Fixed height */
    width: 80%;
    margin: 2em auto;
    overflow: scroll; /* Force scrollbars */
    border: 1px solid cyan;
    padding: 1em;

    background-image: url('images/placeholder-pattern.png'); /* Repeating pattern */
    background-size: 40px; /* Make pattern visible */
    background-repeat: repeat; /* Let it repeat */
    background-attachment: local; /* Background scrolls WITH content */
    /* Try changing to 'scroll' (default) in DevTools */
}

.scroll-box p {
    margin-bottom: 1em;
    /* Making text transparent to see background clearly */
    color: transparent; 
    text-shadow: 0 0 2px rgba(255,255,255,0.5); /* Slight shadow to see text exists */
}
  • We made the text transparent with a slight shadow just to make the background scrolling more obvious.
