58. Component 4: <body> Element

Files Modified: None

The final component of the minimal HTML structure is the <body> element.

  • Purpose: This element contains all the visible content of the web page – the text, images, headings, lists, links, videos, forms, etc., that the user actually sees and interacts with in the browser window.
  • Syntax: Requires an opening <body> and closing </body> tag.
  • Placement: It must be the second direct child of the <html> element, immediately following the </head> closing tag. There must be exactly one <body> element in a document.

If the <head> is the information desk, the <body> is the main stage where the actual show happens.
