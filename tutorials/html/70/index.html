Presentational Elements Demo (i, b, u)

Use i tags for text offset from normal prose, like a technical term, an idiomatic phrase from another language, a thought, or a ship name. Do not use it simply for italics if em or CSS is more appropriate.

Use b tags to stylistically offset text without conveying extra importance, like keywords in a document abstract, product names in a review, or article lead sentences. Do not use it if strong or CSS is more appropriate.

Use u tags generally to represent a non-textual annotation, like labeling a word as a misspelling or marking proper names in Chinese text. Avoid using u where it could be confused with a hyperlink. Prefer CSS `text-decoration` for underlines.
