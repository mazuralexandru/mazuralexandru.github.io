38. Purpose of Void Elements

Files Modified: None

Why do void elements exist?

They typically represent a specific instruction, insertion point, or self-contained object that doesn't naturally "contain" other content in the same way a paragraph contains text or a <div> contains other elements.

  • <br>: Inserts a line break. It doesn't contain the break; it is the break.
  • <hr>: Represents a thematic break between paragraph-level elements. It is the divider.
  • <img>: Embeds an image. The image source is specified via an attribute (src), but the <img> tag itself doesn't enclose image data or other elements.
  • <input>: Creates a form control. Its type and value are set by attributes. It doesn't contain other elements.
  • <link> / <meta>: Provide information about the document (metadata or links) rather than representing content within the document flow.

They are concise ways to represent these specific concepts without needing unnecessary closing tags.
