128. Internal Stylesheets: Syntax

Files Modified: None

The syntax for using an internal stylesheet is straightforward:

  1. Open the <head> section of your HTML document.
  2. Add an opening <style> tag.
  3. Write standard CSS rulesets (selectors and declaration blocks) between the <style> tags.
  4. Add a closing </style> tag.
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="base_styles.css">
    <link rel="stylesheet" href="style.css">
    <title>My Page Title</title>

    <style>
      /* CSS Rules Go Here */
      h1 {
        color: green;
      }
      p {
        font-style: italic;
      }
    </style>

</head>

The browser will parse the content within the <style> tags as CSS and apply it to the elements in the <body> of the same document.
