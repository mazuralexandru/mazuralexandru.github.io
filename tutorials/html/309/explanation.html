309. Centering with Auto Margins: Technique

Files Modified: index.html, style.css

A very common layout task is horizontally centering a block-level element within its container. The standard technique for this uses auto margins.

  • Technique:

    1. The element to be centered must be a block-level element (or have display: block;).
    2. The element must have an explicitly defined width that is less than the width of its container (e.g., width: 80%; or width: 600px;). It cannot be width: auto;.
    3. Set both margin-left and margin-right to auto. This can be done using the shorthand: margin: 0 auto; (sets top/bottom margin to 0, left/right to auto) or margin: 1em auto; (sets top/bottom to 1em, left/right to auto).

  • How it Works: When left and right margins are set to auto on an element with a defined width, the browser calculates the available horizontal space in the container (container width - element width) and divides it equally between the left and right margins, effectively centering the element.

✅ Switch to style.css:

  • We define a width for .centered-box.
  • We apply margin-left: auto; and margin-right: auto;.
  • We add background and padding for visibility.
/* Centering with Auto Margins */
.centered-box {
  width: 60%; /* MUST have an explicit width less than container */
  margin-left: auto;
  margin-right: auto;
  /* Or use shorthand: margin: 1em auto; */

  background-color: teal;
  padding: 20px;
  height: 100px; /* Added height for visibility */
}

✨ Observe the Preview: You should see the teal box horizontally centered within the page body. The space to its left and right should be equal. If you resize the window, it will remain centered while maintaining its 60% width.
