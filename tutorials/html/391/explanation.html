391. alt Attribute (Accessibility Deep Dive)

Files Modified: None

The alt (alternative text) attribute is essential for accessibility and SEO. It provides a textual description of the image's content and function.

  • Purpose:
    • Read aloud by screen readers for visually impaired users.
    • Displayed if the image fails to load (broken link, slow connection).
    • Used by search engines to understand the image content.
  • Requirement: According to WCAG (Web Content Accessibility Guidelines), all <img> elements must have an alt attribute. The only exception is for purely decorative images where the alt text should be explicitly empty (alt="").
  • Writing Good Alt Text:
    • Be Concise & Descriptive: Describe the image's content and purpose accurately.
    • Context is Key: The description should make sense within the surrounding text.
    • Avoid Redundancy: Don't start with "Image of..." or "Picture of...". Screen readers already announce it as an image.
    • Informative Images: Describe the information conveyed (e.g., alt="Bar chart showing Q3 sales growth").
    • Functional Images (Links/Buttons): Describe the action or destination (e.g., alt="Search products", alt="View shopping cart").
    • Decorative Images: Images purely for visual flair that don't convey information should have an empty alt attribute: alt="". This tells screen readers to ignore the image. Never omit the alt attribute entirely.
    • Complex Images (Charts, Diagrams): Provide a brief summary in alt and link to or provide a longer description nearby.
  • Examples:
    • Good: alt="Golden retriever puppy playing with a red ball"
    • Bad: alt="image"
    • Bad: alt="puppy" (Too brief)
    • Good (Functional): alt="Print this page"
    • Good (Decorative): alt="" (For a background pattern image used in content)

Providing meaningful alt text is a fundamental aspect of creating accessible web content.
