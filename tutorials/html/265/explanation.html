265. width Applicability

Files Modified: None

It's important to remember which elements respect the width property by default:

  • Block-level Elements: <div>, <p>, <h1>-<h6>, <ul>, <ol>, <li>, <form>, <article>, <section>, <header>, <footer>, etc. These elements naturally occupy their own line and can have their width controlled.
  • Replaced Elements: Elements whose content is replaced by an external resource, like <img>, <video>, <iframe>, or form elements like <input>, <textarea>, <select>. These often behave like inline-block elements and respect width.
  • Elements with display set to block or inline-block: You can change an element's default display type using the display property. If you set an inline element like <span> to display: block; or display: inline-block;, it will then respect the width property.

Elements that IGNORE width by default:

  • Non-replaced Inline Elements: <span>, <a>, <em>, <strong>, <i>, <b>, etc. Their width is determined solely by their content.
