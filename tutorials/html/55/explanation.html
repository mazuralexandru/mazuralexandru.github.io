55. <head>: Syntax

Files Modified: None

The syntax involves the <head> and </head> tags wrapping the metadata elements:

<!DOCTYPE html>
<html lang="en">
  <head> 
    <!-- Metadata elements go here -->
    <!-- e.g., <meta charset="UTF-8"> -->
    <!-- e.g., <title>Page Title</title> -->
    <!-- e.g., <link rel="stylesheet" href="style.css"> -->
  </head> 
  
  <!-- <body> element follows here -->

</html>
  • It uses standard opening (<head>) and closing (</head>) tags.
  • It must be nested directly inside the <html> element.
  • It must come before the <body> element.

Only specific metadata elements are allowed inside the <head>. You cannot place visible content elements like <p> or <h1> directly inside the <head>.
