229. <em> & <strong>: Default Styling

Files Modified: index.html

As mentioned, browsers apply default styling to these elements as visual hints:

  • <em> is typically rendered using font-style: italic;.
  • <strong> is typically rendered using font-weight: bold;.

✅ Observe index.html:

  • We've added <em> around the word "standard" in the first paragraph.
  • We've added <strong> around the word "follows" in the second paragraph.
        <p class="highlight-text" title="Standard Paragraph">
            This is a <em>standard</em> paragraph with a verylongandunbreakablewordlikethisoneaddedtoit. 
            <span>This is inside the paragraph.</span> 
        </p> 

        <p>This paragraph <strong>follows</strong> the heading.</p>

✨ Observe the Preview:

  • The word "standard" in the first paragraph should now appear in italics (inheriting the yellow text color).
  • The word "follows" in the second paragraph should now appear in bold (inheriting the light green text color).

This default styling reflects the semantic meaning, but remember you can override it with CSS if needed.
