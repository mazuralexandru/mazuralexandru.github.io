60. Text Styling: text-decoration

The text-decoration property is used to add lines to text, such as underlines or line-throughs.

.custom-deco-text {
  /* Using sub-properties for more control */
  text-decoration-line: overline; /* Could be underline, line-through, overline, or multiple */
  text-decoration-color: red;
  text-decoration-style: wavy; /* solid, double, dotted, dashed, wavy */
  text-decoration-thickness: 2px; /* or 'auto', 'from-font', percentage, length */
  margin-bottom: 10px;
}

.no-deco-link {
  text-decoration: none; /* Common use case: removing default link underlines */
  color: cyan; /* Make link visible */
}

text-decoration Explained

  • Shorthand Property: text-decoration is a shorthand for several sub-properties:
    • text-decoration-line: Specifies the type of line(s). Values: none, underline, overline, line-through, or multiple space-separated values (e.g., underline overline).
    • text-decoration-color: Sets the color of the line(s). Accepts any valid color value. Defaults to currentColor if not set.
    • text-decoration-style: Sets the style of the line(s). Values: solid (default), double, dotted, dashed, wavy.
    • text-decoration-thickness: Sets the thickness of the line(s). Values: auto (default), from-font, or <length>/<percentage>.
  • Using the Shorthand: You can specify one, two, or three values for line, style, and color (order doesn't strictly matter for these three in the shorthand). Thickness must be set via its specific property. Example: text-decoration: underline wavy red;
  • Common Use Case: Removing the default underline from links (a { text-decoration: none; }). Be sure to provide other visual cues (like color) if you remove underlines, for accessibility.

In Our Example:

  • .underline-text uses the basic shorthand text-decoration: underline;.
  • .strike-text uses text-decoration: line-through;.
  • .custom-deco-text uses the individual sub-properties to create a 2px thick, wavy, red overline.
  • .no-deco-link explicitly sets text-decoration: none; to remove the browser's default underline on the link.
