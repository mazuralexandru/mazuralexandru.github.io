224. The src Attribute - Definition & Purpose

To make an <img> element actually display an image, the src attribute is essential and mandatory.

The src (Source) Attribute:

  • Definition: Specifies the URL (Uniform Resource Locator), or path, to the image file that should be embedded.
  • Purpose: It tells the browser where to find the image file to download and display on the page. Without a valid src, the browser doesn't know which image to show.
  • Requirement: The src attribute is mandatory for every <img> element to be valid and functional.
