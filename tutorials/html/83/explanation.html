83. CSS Rulesets

Files Modified: None

A ruleset is the fundamental building block of CSS. It's the combination of a selector and a declaration block.

/* This whole thing is ONE ruleset */
h1 {                               /* Selector */
  color: cornflowerblue;           /* Declaration Block starts { */
  text-decoration: underline;      /*   Declaration 1 */
  font-size: 2em;                  /*   Declaration 2 */
}                                  /* Declaration Block ends } */
  • Purpose: A ruleset connects specific HTML elements (targeted by the selector) with specific style instructions (defined in the declarations).
  • Multiple Rulesets: A CSS file typically contains many rulesets, targeting different elements or groups of elements to build up the overall page styling.

Understanding this basic structure selector { property: value; } is key to writing any CSS.
