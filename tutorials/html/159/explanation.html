159. href URL Types: Relative URL (Parent Directory)

Now, let's see how to link to a file located in the parent directory (the directory above the current one) using a relative URL.

Syntax:

You use two dots followed by a forward slash (../) to represent moving up one level in the directory structure.

  • To go up one level and link to filename.html in the parent directory: href="../filename.html"
  • To go up two levels: href="../../filename.html"
  • To go up one level, then into a sibling directory named other, and link to file.html: href="../other/file.html"
