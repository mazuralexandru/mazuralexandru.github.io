15. Minimal HTML Document: The <html> Element

After the DOCTYPE, the next essential part is the <html> element.

Purpose:

  • Acts as the root element for the entire HTML document.
  • It's the main container that encloses all other HTML elements on the page (except for the <!DOCTYPE html> declaration itself).

Syntax:

  • It uses an opening tag <html> and a closing tag </html>.
  • All other page content and structure (like the <head> and <body> elements we'll add next) must be nested inside the <html> element.

Placement:

  • The <html> tag should appear immediately after the <!DOCTYPE html> declaration.
  • The </html> closing tag will be the very last thing in the HTML file.

Code Change:

We've added the <html> opening and closing tags after the DOCTYPE in index.html. We've also moved the comment inside the <html> element, as comments should reside within the structure.

<!DOCTYPE html>
<html> <!-- Added opening tag -->

<!-- Moved comment inside -->
<!-- We will build the standard document structure soon -->

</html> <!-- Added closing tag -->

Note: Indentation here is for readability; HTML ignores most whitespace.
