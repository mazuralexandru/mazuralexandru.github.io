175. Descendant Combinator ( Space)

Files Modified: style.css

The Descendant Combinator is represented by one or more spaces between two selectors.

  • Syntax: ancestor descendant
  • Meaning: Selects any element matching descendant that is located anywhere inside an element matching ancestor in the HTML structure. The descendant doesn't have to be a direct child; it can be nested multiple levels deep.

✅ Observe style.css:

  • We've added a new rule: .container span.
/* Descendant Combinator Example */
.container span {
  font-weight: bold; /* Make spans inside .container bold */
}

✨ Observe the Preview:

  • Look at the first paragraph ("This is a standard paragraph..."). The text "This is inside the paragraph." should now be bold.
  • This happens because:
    • The <span> element is a descendant of the <div> which has class="container".
    • The selector .container span matches this <span>.
    • The font-weight: bold; style is applied.

If we had a <span> outside the <div class="container">, this rule would not affect it.
