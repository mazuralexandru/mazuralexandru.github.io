193. Email Links (mailto:): Adding CC and BCC

Besides the subject, you can also pre-fill the Carbon Copy (cc) and Blind Carbon Copy (bcc) fields using parameters.

Combining Multiple Parameters:

  • Use an ampersand (&) character to separate multiple key=value parameters in the mailto: link.
  • The first parameter still starts with ?, and subsequent parameters are joined with &.

The cc and bcc Parameters:

  • cc: Pre-fills the "CC" (Carbon Copy) field. Multiple addresses can sometimes be added, separated by commas (,), but support may vary between email clients. The comma itself doesn't usually need encoding here, but email addresses within parameters should be valid.
  • bcc: Pre-fills the "BCC" (Blind Carbon Copy) field. Addresses here are hidden from other recipients. Same comma separation note applies.
  • Syntax: mailto:to@example.com?subject=Subj&cc=cc@example.com&bcc=bcc@example.com

Code Change:

We've added cc and bcc parameters to our mailto: link in index.html, connecting them to the existing subject parameter using &.

<p>
  <!-- Note: Use '&' to separate multiple parameters -->
  <a href="mailto:contact@example.com?subject=Website%20Inquiry&cc=manager@example.com&bcc=admin@example.com"> <!-- Added &cc=...&bcc=... -->
    Email with Subject, CC, BCC
  </a>
</p>

✅ Observation:

  1. Examine the href value. Note the structure: mailto:...?subject=...&cc=...&bcc=...
  2. Click the link in the preview.
  3. If your email client opens, check the pre-filled fields:
    • "To:" should be contact@example.com.
    • "Subject:" should be Website Inquiry.
    • "CC:" should contain manager@example.com.
    • "BCC:" should contain admin@example.com.

✨ Key Takeaway: Use & to chain multiple parameters like cc=address and bcc=address onto a mailto: link after the initial ?. Remember URL encoding if needed for parameter values (less common for simple email addresses).
