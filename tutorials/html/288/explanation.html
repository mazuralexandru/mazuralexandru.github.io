288. The title Attribute - Fundamentals & Syntax

The title global attribute provides a mechanism for adding advisory information to an element.

Purpose:

  • The title attribute offers supplementary, non-essential information about the element it's applied to.
  • It's intended for hints, explanations, definitions, or clarifications that aren't part of the main content flow but might be helpful to some users.

Core Concept:

  • It associates a piece of text (the attribute's value) with an element, which browsers typically reveal through a specific user interaction (usually hovering).

Syntax:

  • title="Supplementary advisory text here"
  • The attribute is added to the opening tag of almost any HTML element.
  • The value is a string of plain text. While you can put HTML entities (like &amp;amp;) inside, browsers generally don't interpret HTML tags within the title value when displaying the tooltip. Line breaks might be preserved by some browsers in the tooltip.

Code Change:

We've added a title attribute to the <h1> element in index.html.

<!-- index.html -->
<body>
  <h1 title="This is the main heading for Module 6">Global Attributes - Title</h1> <!-- Added title -->
  <p>Let's explore the 'title' attribute.</p>

  <p>It provides supplementary information, often shown as a tooltip.</p>

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Locate the title="..." attribute added to the <h1> tag.
  2. Look at the browser preview. Visually, the heading appears unchanged.
  3. Hover your mouse pointer over the "Global Attributes - Title" heading for a second or two. A small tooltip should appear, displaying the text "This is the main heading for Module 6". (Tooltip appearance varies by browser/OS).
  4. Move the mouse away, and the tooltip disappears.

✨ Key Takeaway: The title attribute provides advisory text associated with an element. Browsers typically display this text as a tooltip when the user hovers over the element.
