105. HTML Comments

Files Modified: index.html

Comments are parts of your code that are ignored by the browser but are visible to developers reading the source code. They are essential for explaining code, leaving notes, or temporarily disabling parts of the code.

  • Purpose:
    • Add explanations or notes for yourself or other developers.
    • Clarify complex sections of code.
    • Temporarily remove code from being processed without deleting it (called "commenting out").
  • Syntax: HTML comments start with <!-- and end with -->. Everything in between is considered part of the comment.
