31. Text: Paragraphs (<p>) - Fundamentals

Welcome to Module 2! We'll now explore specific HTML elements used to structure the text content within the <body>. We'll start with the most fundamental text block: the paragraph.

The Paragraph Element (<p>)

  • Purpose: The <p> element is used to define a paragraph of text. It groups related sentences into a distinct block.
  • Semantics: It semantically marks the content as being a paragraph within the document's structure. This is important for readability, accessibility, and styling.
  • Type: It's a block-level element. This means it typically starts on a new line and takes up the full width available to it (though this can be changed with CSS).

Code Change:

We already had one paragraph in our index.html. To better illustrate its block nature and separation, we've added a second <p> element below the first one inside the <body>.

<body>
  <h1>Hello World!</h1>
  <p>This is the body content of my first web page. It represents a single, logical block of text.</p>
  <p>This is a second paragraph. Notice how it starts on a new line and has some space separating it from the first paragraph.</p> <!-- Added second paragraph -->

</body>

✅ Observation:

  1. Examine the index.html file and locate the new, second <p> element.
  2. Look at the browser preview pane. You should see:
    • The heading ("Hello World!").
    • The first paragraph.
    • The second paragraph, appearing below the first one.
    • Notice the vertical whitespace (margin) between the heading and the first paragraph, and between the first and second paragraphs. This visual separation is a default browser behavior for <h1> and <p> elements.

✨ Key Takeaway: Use the <p> element to wrap distinct paragraphs of text. Browsers render them as separate blocks with default spacing.
