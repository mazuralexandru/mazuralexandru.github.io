318. display Value: inline-block

Files Modified: style.css

inline-block offers a hybrid behavior, combining aspects of both inline and block elements.

  • Behavior:
    1. Flows Inline: Like inline elements, inline-block elements do not force line breaks before or after themselves. They sit alongside other inline or inline-block elements on the same line if space allows.
    2. Respects Dimensions/Spacing: Like block elements, inline-block elements do respect explicit width, height, vertical padding, and vertical margin properties. These properties affect the element's size and its spacing relative to other elements on the same line and surrounding lines.
  • Use Cases:
    • Creating grid-like layouts of items (like navigation links, buttons, or small cards) that should sit next to each other but also have controllable dimensions and vertical spacing.
    • Positioning small elements like icons or badges next to text while controlling their size and alignment precisely.

✅ Observe style.css:

  • We've changed the display property for .inline-span from its default (inline) to inline-block.
/* Display Inline-Block Example */
.inline-span {
  display: inline-block; /* Change display type */

  background-color: rgba(255, 192, 203, 0.4); 
  border: 1px solid pink;
  padding: 10px; /* Vertical padding now respected */
  margin: 10px; /* Vertical margin now respected */
  width: 100px; /* Now respected */
  height: 50px; /* Now respected */
  text-align: center; /* Center text inside the box */
}
  • We also reduced the width to 100px and added text-align: center for better visualization.

✨ Observe the Preview:

  • The spans still flow inline next to each other (and the surrounding text) as space allows.
  • However, each span now behaves like a mini-block:
    • The width: 100px; and height: 50px; are now respected, giving each span a fixed size.
    • The vertical padding (10px) and margin (10px) are now fully effective, creating space above and below each span box and potentially increasing the overall line height.
    • The text inside each span is centered due to text-align: center;.
