59. <body>: Purpose Explained

Files Modified: None

The <body> element's role is simple but vital: it defines the main content area of the HTML document that will be rendered visually by the browser.

  • Visible Content: Everything you want the user to see goes inside the <body> tags. This includes text (<p>, <h1>-<h6>), lists (<ul>, <ol>, <li>), links (<a>), images (<img>), tables (<table>), forms (<form>), structural divisions (<div>, <section>, <article>, etc.), and more.
  • Document Body: It represents the body of the document in the browser's internal DOM structure. CSS styles are applied to elements within the body to control their appearance, and JavaScript often manipulates elements within the body to create dynamic behavior.

Essentially, anything that isn't metadata belonging in the <head> belongs in the <body>.
