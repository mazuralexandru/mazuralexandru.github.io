402. Image Alignment & Spacing Introduction

Files Modified: None

Beyond controlling the image's size and how its content fits within its box, we often need to control how the image box itself is positioned relative to surrounding content or its container.

We'll explore several ways to achieve this:

  1. Default Inline Behavior: Understanding how images behave inline with text and using vertical-align.
  2. Block-Level Behavior: Using display: block; and techniques like margin: auto; for centering.
  3. Using Layout Containers: Leveraging Flexbox or Grid on parent elements for sophisticated alignment.
  4. Basic Spacing: Applying margin directly to the image.
