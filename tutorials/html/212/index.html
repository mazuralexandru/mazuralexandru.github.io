Styling Links with CSS (Preview)

CSS pseudo-classes style links based on their state:

Example Link 1 Example Link 2
  • :link - Unvisited link
  • :visited - Visited link (styling limited by privacy)
  • :hover - Mouse pointer is over the link
  • :active - Link is currently being clicked
  • :focus - Link has keyboard focus (Tab key) - CRUCIAL for accessibility!
