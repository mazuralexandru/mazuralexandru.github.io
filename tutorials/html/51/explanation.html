51. <html> Essential Attribute: lang

Files Modified: None

While optional according to the strict syntax, adding the lang attribute to the <html> tag is a critical best practice.

  • Purpose: Specifies the primary language of the text content within the element (and, by inheritance, the whole page unless overridden).
  • Syntax: <html lang="language-code">
  • Example: <html lang="en"> (for English), <html lang="es"> (for Spanish), <html lang="fr"> (for French).

Why is it important?

  1. Accessibility: Screen readers use the lang attribute to pronounce the content correctly using the appropriate language profile (accent, pronunciation rules). Missing or incorrect lang can make content unintelligible for screen reader users.
  2. Search Engines (SEO): Helps search engines understand the language of your page, which can aid in targeting the right audience and ranking for language-specific searches.
  3. Translation Tools: Browser translation features may use this attribute to determine the original language.
  4. Styling: CSS selectors can target elements based on their language (e.g., :lang(fr)).

Finding Language Codes: Use standard codes defined by BCP 47. Common codes include en (English), es (Spanish), fr (French), de (German), ja (Japanese), zh (Chinese). You can also specify regional variations like en-US (US English) or en-GB (British English).

Rule: Always include a lang attribute on your <html> tag.
