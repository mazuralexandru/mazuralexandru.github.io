19. Code Formatting

Files Modified: None

Consistent code formatting (indentation, spacing, line breaks) makes code much easier to read and maintain, especially when working in teams. Manually formatting code is tedious and error-prone.

Code Formatters are tools that automatically reformat your code according to predefined style rules.

  • Prettier: A very popular, opinionated code formatter that supports HTML, CSS, JavaScript, and more. It enforces a consistent style with minimal configuration.
  • VS Code Integration: VS Code has built-in formatting capabilities and integrates seamlessly with formatters like Prettier (often via an extension). You can usually format a file on save or by using a keyboard shortcut (e.g., Shift+Alt+F or Shift+Option+F).

Why use it?

  • Consistency: Ensures all code follows the same style guide.
  • Readability: Properly indented and spaced code is easier to follow.
  • Reduces Arguments: Eliminates debates about subjective style preferences within teams.
  • Focus: Lets you focus on the logic, not the layout of the code.

Setting up a formatter like Prettier is highly recommended.
