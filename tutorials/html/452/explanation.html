452. background-origin: Values

Files Modified: style.css

Let's explicitly see the difference between the values.

✅ Observe style.css:

  • We've changed background-origin to content-box.
.origin-box {
    width: 300px;
    height: 150px;
    padding: 25px; 
    border: 10px dashed orange; 
    margin: 2em auto;
    background-color: rgba(0, 100, 0, 0.2); 

    background-image: url('images/placeholder-pattern.png');
    background-repeat: no-repeat;
    background-size: 40px; 
    background-position: top left; 
    /* background-origin: border-box; */
    background-origin: content-box; /* Position relative to content edge */
}

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview:

  • Compared to the border-box example, the small background pattern image should now be positioned further inside the box, aligning with the top-left corner of the actual content area (inside the padding).
  • Changing this property alters the reference point for background-position. The default, padding-box, aligns with the inside edge of the border.
