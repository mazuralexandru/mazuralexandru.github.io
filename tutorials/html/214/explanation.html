214. text-align

Files Modified: None

The text-align property controls the horizontal alignment of inline content (like text) within its containing block element.

  • Purpose: To align text to the left, right, center, or justify it within its container.
  • Applies To: Block-level elements (like <p>, <h1>, <div>). It affects the inline content inside the block element.
  • Inherited: Yes. Child block elements will inherit the text alignment unless they specify their own.
  • Common Values:
    • left: Aligns text to the left edge (default for left-to-right languages like English).
    • right: Aligns text to the right edge (default for right-to-left languages like Arabic/Hebrew).
    • center: Centers the text horizontally within the container.
    • justify: Stretches the lines of text so that both the left and right edges align (like in newspaper columns). Can sometimes create awkward spacing between words. Use with consideration for readability.
  • text-align-last: A related property that controls the alignment of the last line of text within a block, which is especially relevant when text-align: justify; is used.

Example:

p { text-align: left; } /* Default */
h1 { text-align: center; }
.quote { text-align: right; }
.newspaper-column { text-align: justify; }
