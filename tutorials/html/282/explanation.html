282. Padding Shorthand Property (padding)

Files Modified: style.css

Writing out all four longhand padding properties can be verbose. CSS provides the padding shorthand property to set padding on all four sides with a single declaration.

  • Purpose: A more concise way to define padding.
  • Syntax: The number of values provided determines how they are applied:
    • One Value: padding: value;
      • Applies the value to all four sides (top, right, bottom, left).
      • Example: padding: 15px; (15px on all sides)
    • Two Values: padding: value1 value2;
      • value1 applies to top and bottom.
      • value2 applies to left and right.
      • Example: padding: 10px 20px; (10px top/bottom, 20px left/right)
    • Three Values: padding: value1 value2 value3;
      • value1 applies to top.
      • value2 applies to left and right.
      • value3 applies to bottom.
      • Example: padding: 10px 20px 5px; (10px top, 20px left/right, 5px bottom)
    • Four Values: padding: value1 value2 value3 value4;
      • Applies values in clockwise order: value1 (Top), value2 (Right), value3 (Bottom), value4 (Left). (TRBL - "Trouble")
      • Example: padding: 10px 20px 5px 15px; (10px top, 20px right, 5px bottom, 15px left)

✅ Observe style.css:

  • We've replaced the four longhand padding-* properties with a single padding shorthand using four values to achieve the same result as before.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  /* Replaced longhand with shorthand */
  padding: 20px 5px 15px 30px; /* Top, Right, Bottom, Left */
  margin: 1em auto; 
}

✨ Observe the Preview: The appearance of the box and its internal spacing should be identical to the previous step, but the CSS code used to achieve it is now more concise using the padding shorthand.
