190. The initial Keyword

Files Modified: None

The initial keyword is used to explicitly reset a CSS property on an element to its initial value, as defined by the CSS specification for that property.

  • Purpose: To undo any inherited styles or styles applied by the cascade and force a property back to its standard default state.
  • Syntax: property: initial;
  • What is the "Initial Value"? Every CSS property has a defined initial value in the CSS standards (e.g., the initial value for color is browser/element dependent but often black, for background-color it's transparent, for margin it's 0, for display it's inline). It's not necessarily the browser's default stylesheet value, but the value defined in the spec.

Example:

p {
  color: blue; /* All paragraphs blue by default */
}

.reset-color {
  color: initial; /* Reset color to its CSS initial value (likely black) */
}

<p>This paragraph is blue.</p>
<p class="reset-color">This paragraph's color is reset.</p>

Using initial ensures you get the standard default, regardless of inherited values or browser defaults.
