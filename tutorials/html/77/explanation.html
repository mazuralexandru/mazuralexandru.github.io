77. Linking Base Stylesheet

Files Modified: index.html

Before we dive into CSS theory, let's link our base stylesheet (base_styles.css) to our HTML document. This file will eventually contain foundational styles for a consistent look and feel (like the dark theme and grid background).

We use the <link> element within the <head> section.

  • Element: <link> (a void element).
  • rel attribute: Specifies the relationship. stylesheet indicates it's a CSS file.
  • href attribute: Specifies the URL (path) to the CSS file. Here, it's base_styles.css (assuming it's in the same directory).

    <link rel="stylesheet" href="base_styles.css"> <!-- Link base styles -->
