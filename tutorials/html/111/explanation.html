111. Ordered Lists (<ol>): The start Attribute

Another useful attribute for <ol> is start, which allows you to specify the starting number for the list sequence.

The start Attribute:

  • Purpose: Specifies the ordinal value (the number) from which the list should start counting.
  • Syntax: <ol start="integer">
  • Value: An integer (whole number). Importantly, the value provided is always interpreted as a decimal number, even if you have also used a type attribute like "a" or "I". The browser then converts this starting decimal number to the appropriate letter or Roman numeral if needed.
  • Behavior: The first <li> in the list will receive the marker corresponding to the start value, and subsequent items will increment sequentially from that value.

Use Cases:

  • Continuing a list that was interrupted by other content.
  • Starting a list from a specific number that has semantic meaning (e.g., "Step 5: ...").
  • Creating lists like "Top 10 starting from number 5".

Code Change:

We've added the start="5" attribute to the opening <ol> tag in index.html.

<p>The 'start' attribute specifies the starting number:</p> <!-- Adjusted P -->

<ol start="5"> <!-- Added start="5" -->
  <li>This item starts at 5</li>
  <li>This item is 6</li>
  <li>This item is 7</li>
</ol>

<p>The value is always a number, regardless of 'type'.</p> <!-- Adjusted P -->

✅ Observation:

  1. Locate the start="5" attribute in the <ol> tag in index.html.
  2. Look at the browser preview. The list markers should now start numbering from 5: 5., 6., 7..

✨ Experiment: Try changing the start value to "10" or "3" in index.html and observe how the numbering changes in the preview. (Remember to set it back to 5 if following along).

✨ Key Takeaway: Use the start attribute on <ol> to begin the list's sequence from a specific number other than 1.
