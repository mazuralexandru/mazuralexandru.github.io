61. <body>: Placement

Files Modified: None

The placement rules for the <body> element are strict:

  1. It must be a direct child of the <html> element.
  2. It must appear after the closing </head> tag.
  3. There must be only one <body> element in the entire document.
<!DOCTYPE html>
<html lang="en">

  <head>
    <!-- Metadata -->
  </head> 

  <body>  <!-- CORRECT: Inside <html>, after </head> -->
    <!-- Visible Content -->
  </body>

</html>

Placing <body> tags inside the <head>, having multiple <body> sections, or placing content outside both <head> and <body> (but within <html>) is invalid HTML.
