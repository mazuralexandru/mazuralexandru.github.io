438. background-position

Files Modified: style.css

When a background image is not repeated (no-repeat) or repeated only along one axis, the background-position property controls its initial placement within the background positioning area (which defaults to the padding box).

  • Purpose: To set the starting position of a background image.
  • Syntax: background-position: value;
    • Accepts one or two values.
    • If one value is given:
      • If it's a keyword like top, bottom, center, it sets that position along one axis, and the other axis defaults to center. (background-position: top; is same as top center;)
      • If it's left or right, it sets the horizontal position, and vertical defaults to center.
      • If it's a length or percentage, it sets the horizontal position (offset from left), and vertical defaults to 50% (center).
    • If two values are given:
      • The first sets the horizontal position (left, center, right, length, or percentage).
      • The second sets the vertical position (top, center, bottom, length, or percentage).
  • Default: 0% 0% or top left.
  • Multiple Backgrounds: Use comma-separated values to position each background layer independently.

✅ Observe style.css:

  • We've added background-position: center top, center center; to the .bg-box rule.
    • The first value (center top) applies to the first image layer (the gradient).
    • The second value (center center) applies to the second image layer (the pattern image).
.bg-box {
    background-color: transparent; 
    background-image: 
        linear-gradient(to bottom, rgba(0,0,0,0.5), transparent), 
        url('images/placeholder-pattern.png'); 
    background-repeat: no-repeat; 
    background-position: center top, center center; /* Position layers */
    padding: 2em;
    margin-bottom: 1em;
    border: 1px dashed #fff; 
    min-height: 150px; 
}
