09. Applying Inline Styles

While external stylesheets (style.css) are preferred for overall styling, HTML allows you to apply styles directly to a single element using the style attribute. This is called an "inline style".

What's New?

We added a style attribute directly to the <p> tag.

<!-- index.html snippet -->
<body>
  <!-- This is our first paragraph -->
  <p style="color: lightcoral;">Hello, World!</p> <!-- Added inline style -->

  <!-- Commented out paragraph remains -->
  <!-- <p>This paragraph is commented out and won't appear.</p> -->
</body>

The style Attribute (Inline CSS)

  • Purpose: Allows you to embed CSS declarations directly within an HTML element's opening tag.
  • Syntax: <element style="property: value; property2: value2;">
    • The value of the style attribute is a string containing one or more standard CSS declarations (property: value), separated by semicolons, just like inside a CSS file's declaration block.
  • Specificity: Inline styles have very high specificity, meaning they often override styles defined in external stylesheets or within <style> tags in the <head>. We'll cover specificity in more detail later.
  • Use Cases:
    • Applying a unique style to one specific element that won't be reused.
    • Quick testing or debugging.
    • Sometimes used by JavaScript to dynamically change an element's style.
  • Drawbacks (Why Avoid Generally):
    • Mixes Presentation and Structure: Violates the principle of separating concerns.
    • Poor Maintainability: Styles are scattered throughout the HTML, making updates difficult.
    • No Reusability: The style only applies to that single element.
    • Overrides: Can make stylesheets harder to manage due to high specificity.

CSS color Property

  • We used color: lightcoral; inside the style attribute.
  • The color property sets the foreground color of an element's text content.
  • lightcoral is a predefined color keyword. CSS supports many such keywords, as well as other color formats like hexadecimal (#F08080) or RGB (rgb(240, 128, 128)).
