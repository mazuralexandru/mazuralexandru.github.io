29. Content

Files Modified: None

The content is everything that sits between the opening tag and the closing tag of an HTML element.

  • Nature: Content can be various things:
    • Plain text (e.g., <p>This is paragraph text.</p>)
    • Other HTML elements (this is called nesting, e.g., <div><p>Nested paragraph.</p></div>)
    • A combination of text and other elements.
    • Sometimes, elements are designed to be empty and have no content (void elements).
  • Purpose: The content is the actual information or structure that the element represents (the text of the paragraph, the items in a list, the sections within a division).

The tags define the type and boundaries of the element, while the content is the substance within those boundaries.
