92. Inline Styles: Syntax Example

Files Modified: index.html

Let's add a paragraph to our <body> and apply some inline styles using the style attribute.

✅ Observe index.html! A new <p> element has been added inside the <body>.

<p style="color: tomato; font-size: 22px;">This paragraph uses inline styles.</p>

✨ Observe the Preview: You should now see the text "This paragraph uses inline styles." displayed. Notice its appearance:

  • The text color is tomato red (color: tomato;).
  • The text size is larger than the default (font-size: 22px;).

These styles are applied directly by the style attribute on this specific <p> tag. No changes were made to style.css.
