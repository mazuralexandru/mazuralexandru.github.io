236. <s>, <del>, <ins>, <mark>: HTML Recap

Files Modified: None

Let's recap the semantics of these edit/annotation tags:

  • <s>: Content no longer accurate/relevant. (Default: Strikethrough)
  • <del>: Content deleted from the document. (Default: Strikethrough)
  • <ins>: Content inserted into the document. (Default: Underline)
  • <mark>: Content highlighted for contextual relevance. (Default: Yellow background)

Use these tags when the specific semantic meaning applies. For purely visual strikethrough or underlining not related to edits, use CSS text-decoration. For general highlighting not related to contextual relevance, use a <span> with a background color class.
