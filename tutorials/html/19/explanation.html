19. Adding Rules to Internal Stylesheets

Now that we have the <style> element set up in the <head>, let's add our first CSS rule within it to style the <h1> element.

What's New?

  • We've added a CSS ruleset inside the <style> tags in index.html.
<!-- index.html snippet -->
<head>
  <meta charset="UTF-8">
  <title>My First Web Page</title>
  <link rel="stylesheet" href="style.css">
  <style>
    /* Internal CSS rules will go here */
    h1 { /* Added H1 rule */
      color: lightseagreen;
      text-decoration: underline;
    }
  </style>
</head>
<body>

Internal CSS Syntax

  • Inside the <style> tags, you write CSS exactly as you would in an external .css file.
  • Ruleset: Consists of a selector and a declaration block.
    • Selector (h1): Targets the element(s) to style. Here, it targets all <h1> elements on this specific page.
    • Declaration Block ({ ... }): Contains the style definitions.
    • Declarations (property: value;):
      • color: lightseagreen; sets the text color.
      • text-decoration: underline; adds an underline. Remember the semicolon ; separates declarations.
