49. Controlling Inheritance with Keywords

What if you want a child element to explicitly take on a parent's value for a normally non-inherited property? Or what if you want to reset a property on a child element back to its inherited value or its initial default? CSS provides keywords for this.

Inheritance Control Keywords

  • inherit: Forces an element to adopt the computed value of the property from its direct parent element. This works even for properties that don't normally inherit (like border, padding, background-color).
  • initial: Resets a property on the element to its initial default value as defined by the CSS specification for that property (e.g., the initial value for color is often black, for background-color it's transparent, for border it's effectively none). It ignores any inherited value.
  • unset: Acts like a combination. If the property is naturally inherited (like color), unset behaves like inherit. If the property is not naturally inherited (like border), unset behaves like initial. It's often used in CSS resets to undo styles.
  • revert: A newer value that resets the property to the value defined by the user-agent (browser) stylesheet for that element, overriding author and user styles. Less commonly used than the others for basic styling.
