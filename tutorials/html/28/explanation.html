28. Preparing for Styling: Linking the CSS File

We've discussed the "Web Trio" (HTML, CSS, JS) and the importance of separating concerns. Our HTML provides the structure, but it looks very plain using only the browser's default styles.

To apply our own custom styles using CSS, we first need to link our (currently empty) style.css file to our index.html document.

The <link> Element:

  • Purpose: Specifies relationships between the current HTML document and external resources. A primary use is to link external CSS stylesheets.
  • Placement: Must be placed within the <head> section of the HTML document.
  • Syntax: It's a void element (no closing tag). It uses attributes to define the link.
    <link rel="stylesheet" href="style.css">
  • Key Attributes:
    • rel="stylesheet": This attribute is crucial. rel stands for "relationship". Setting it to stylesheet tells the browser that the linked file is a Cascading Style Sheet containing style rules to be applied to the current document.
    • href="style.css": This attribute provides the Hypertext Reference, or the path/URL, to the external resource. In this case, style.css tells the browser to look for a file named style.css in the same directory as the index.html file.

Code Change:

We have added the <link> element inside the <head> section of index.html to connect it to style.css.

<head>
  <meta charset="UTF-8">
  <title>My First Web Page</title>
  <link rel="stylesheet" href="style.css"> <!-- Added link to CSS -->

  <!-- This is an HTML comment. It's ignored by the browser -->
  <!-- Comments can span
       multiple lines. -->
</head>

✅ Observation:

  1. Verify the new <link> tag is present in the <head> of index.html.
  2. Look at the browser preview. It should still look exactly the same as before. Why? Because although we've linked the style.css file, that file is currently empty! We haven't defined any styles yet.
  3. The browser has acknowledged the link and would apply styles from style.css if there were any.

✨ Key Takeaway: Use the <link rel="stylesheet" href="..."> tag within the <head> to connect an external CSS file to your HTML document. This is the standard way to apply styles while maintaining separation of concerns.
