87. Multiple CSS Declarations

Files Modified: None

You almost always want to apply more than one style to an element. You can include multiple declarations within a single declaration block.

  • Rule: Simply list each declaration one after another, making sure each one ends with a semicolon (;).
p { /* Selector */
  /* Declaration Block */
  color: navy;              /* Declaration 1; */
  font-size: 14px;          /* Declaration 2; */
  line-height: 1.5;         /* Declaration 3; */
  margin-bottom: 10px;      /* Declaration 4; */
}

The semicolon after the last declaration in a block is technically optional according to the CSS specification, but it is strongly recommended to always include it. Forgetting it means if you later add another declaration below it, the previous one might become invalid.

Best Practice: Always terminate every declaration with a semicolon.
