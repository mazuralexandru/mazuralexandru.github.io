152. Class Selectors: Example

Files Modified: index.html, style.css

Let's apply a class selector. We'll add a class to our paragraph and create a corresponding CSS rule.

✅ Observe index.html:

  • The <p> element now has a class attribute with the value "highlight-text".
<p class="highlight-text">This is a standard paragraph.</p> <!-- Added class -->

✅ Now, switch to style.css:

  • We've added a new CSS ruleset using the class selector .highlight-text.
  • This rule sets the background-color to a dark yellow and the color to almost black.
  • The previously commented-out p rule has been removed for clarity, as we are now targeting the paragraph via its class.
/* Added class selector rule */
.highlight-text {
  background-color: #4d4d00; /* Dark yellow */
  color: #1a1a1a; /* Near black */
  padding: 5px; /* Add some padding */
}

/* Removed p selector rule */

✨ Observe the Preview: The paragraph ("This is a standard paragraph.") now has a dark yellow background and dark text, because the .highlight-text rule in style.css is targeting the element with class="highlight-text" in index.html. The <h1> remains unchanged as it doesn't have this class. This demonstrates targeted styling using classes.
