361. Link Attribute: target

Files Modified: index.html

The target attribute specifies where to open the linked document when the link is clicked.

  • Purpose: Controls the browsing context (window or tab) for the link destination.

  • Common Values:

    • _self: (Default) Opens the linked document in the same frame or window where the link was clicked.
    • _blank: Opens the linked document in a new unnamed window or tab. This is commonly used for external links to keep the user on your site.
    • _parent: Opens the linked document in the parent frame (used with HTML framesets, which are largely obsolete).
    • _top: Opens the linked document in the full body of the window, cancelling out all frames (used with framesets).
    • <frame-name>: Opens the linked document in a specifically named frame (used with framesets).

  • Security Note for _blank: When using target="_blank", it's strongly recommended to also add rel="noopener noreferrer" for security and privacy reasons (explained next).
