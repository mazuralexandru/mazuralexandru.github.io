225. text-shadow

Files Modified: style.css

The text-shadow property adds one or more shadows to text.

  • Purpose: To create visual effects like drop shadows, glows, or outlines for text.
  • Syntax: text-shadow: h-offset v-offset blur-radius color [, h-offset v-offset blur-radius color]*;
    • h-offset: Horizontal offset of the shadow. Positive values move it right, negative values move it left. (Required)
    • v-offset: Vertical offset of the shadow. Positive values move it down, negative values move it up. (Required)
    • blur-radius: (Optional) The blur distance. Larger values create bigger, fuzzier shadows. 0 means a sharp, unblurred shadow. Defaults to 0 if omitted.
    • color: (Optional) The color of the shadow. Defaults to the element's color if omitted.
    • Multiple Shadows: You can specify multiple shadows by separating them with commas. They are applied front-to-back (the first shadow listed is topmost).
  • Value: none (Default) - No shadow.

✅ Observe style.css:

  • We've added a text-shadow to the #main-heading.
#main-heading {
    /* ... other styles ... */
    text-transform: uppercase; 
    /* Add a simple black drop shadow */
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.7); 
}
  • This adds a shadow offset 2px right, 2px down, with a 4px blur radius, and a semi-transparent black color.

✨ Observe the Preview: The "MAIN HEADING" text should now have a noticeable drop shadow effect behind it, making it appear slightly raised or offset from the background.
