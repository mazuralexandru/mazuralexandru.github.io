370. Link State: :active

Files Modified: style.css

The :active pseudo-class styles a link during the brief moment it is being activated by the user (e.g., while the mouse button is pressed down on the link).

  • :active:
    • Selects: An element (commonly links or buttons) during the time the user activates it.
    • Purpose: Provides brief visual feedback confirming the activation action.

✅ Observe style.css:

  • We've added a rule for a:active.
  • We're changing the color to bright red.
/* Style links when active/clicked */
a:active { 
  color: red; /* Change color while actively clicking */
}

✨ Observe the Preview:

  • Click and hold down the mouse button on any of the links in index.html.
  • While the mouse button is held down, the link's text color should briefly turn red.
  • When you release the mouse button, the link will either navigate (if it's a real link) or revert to its :hover (if the cursor is still over it) or :link/:visited state.
  • This :active state provides immediate feedback that the click is being registered.
