142. Linking Multiple Stylesheets

Files Modified: None

You can link multiple external stylesheets to a single HTML document by including multiple <link rel="stylesheet" href="..."> tags in the <head>.

<head>
  <meta charset="UTF-8">
  <title>Multi-Style Page</title>
  <link rel="stylesheet" href="base.css">      <!-- Base/Reset Styles -->
  <link rel="stylesheet" href="layout.css">    <!-- Layout Rules -->
  <link rel="stylesheet" href="theme.css">     <!-- Visual Theme -->
  <link rel="stylesheet" href="components.css"> <!-- Component Styles -->
</head>

Order Matters: The order in which you link stylesheets is important because of the CSS Cascade. If multiple stylesheets define conflicting rules for the same element with the same specificity, the rule from the stylesheet linked later (further down in the HTML) will generally take precedence.

Common practice is to link stylesheets in this order:

  1. Resets/Normalizers: (e.g., reset.css, normalize.css) - Establish a consistent baseline across browsers.
  2. Base Styles/Frameworks: (e.g., base.css, bootstrap.css) - Define site-wide defaults, typography, basic layout.
  3. Layout Styles: (e.g., layout.css) - Define major page structure (header, footer, sidebar, grid).
  4. Component Styles: (e.g., buttons.css, cards.css) - Styles for reusable UI elements.
  5. Page-Specific Styles: (e.g., about-page.css) - Styles unique to a particular page.
  6. Theme/Overrides: (e.g., dark-theme.css) - Styles that modify or override previous rules.
