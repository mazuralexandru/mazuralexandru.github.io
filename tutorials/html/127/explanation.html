127. Internal Stylesheets: Purpose Explained

Files Modified: None

The primary purpose of using an internal stylesheet with the <style> element is to embed CSS rules directly within the HTML <head>.

This keeps the styles associated with a single HTML page contained within that page's file. Unlike inline styles, you can use standard CSS selectors (h1, p, classes, IDs, etc.) to target multiple elements efficiently from one location within the document.

It offers better organization than scattering inline styles throughout the <body>, but lacks the reusability and caching benefits of external stylesheets.
