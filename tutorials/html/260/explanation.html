260. The Four Components of the Box: Border Area

Files Modified: None

Outside the padding area lies the Border Area.

  • Purpose: The border is the visual line drawn around the element's padding area. It acts like the frame itself around the picture and matting.
  • Control: Set using the border shorthand property or the longhand properties (border-width, border-style, border-color). You need to define at least the style (e.g., solid) for a border to be visible.
  • Dimensions: The border's thickness (defined by border-width) adds to the overall size of the element when using the default box-sizing: content-box;.
  • Appearance: You can control its color, thickness, and style (solid, dotted, dashed, etc.).

Borders are used to visually outline elements or create dividing lines.
