137. <link> Placement

Files Modified: None

The placement rule for the <link> element when used for stylesheets is simple and strict:

  • It must be placed within the <head> ... </head> section of the HTML document.

Placing <link rel="stylesheet"> inside the <body> is invalid HTML and can lead to unpredictable rendering behavior or styles not being applied correctly. Browsers expect to find stylesheet links in the head so they can parse and apply the CSS rules before rendering the body content (or at least concurrently).

You can have multiple <link> elements in the <head> to link several different CSS files.

<head>
    <meta charset="UTF-8">
    <title>My Page</title>
    <link rel="stylesheet" href="reset.css"> <!-- First stylesheet -->
    <link rel="stylesheet" href="layout.css"> <!-- Second stylesheet -->
    <link rel="stylesheet" href="theme.css"> <!-- Third stylesheet -->
    <!-- Internal styles could also go here -->
    <style> /* ... */ </style>
</head>
