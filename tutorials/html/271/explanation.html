271. Minimum and Maximum Dimensions: max-width

Files Modified: style.css

Conversely, the max-width property prevents an element from becoming too wide.

  • Purpose: Sets the maximum width an element can expand to. If the content or the calculated width (like width: 100% or width: auto) would normally make the element wider than the max-width value, the element's width will be capped at the max-width.
  • Syntax: max-width: value;
  • Values: Length units (px, em, rem), percentages (%), or keywords like none (default, no maximum), min-content, max-content, fit-content.
  • Interaction: max-width overrides width if width calculates to a value greater than max-width. min-width still takes precedence over max-width.
  • Responsive Design: max-width: 100%; is extremely common for images (<img>) to make them scale down proportionally within their container on smaller screens without exceeding their original size on larger screens. It's also used on containers to prevent them from becoming excessively wide on large monitors.

✅ Observe style.css:

  • We've added max-width: 400px; to the .box rule.
  • We've also changed width back to auto for this demonstration.
.box {
  /* width: 75%; */
  width: auto; /* Let width be determined by content/container */
  height: auto; 
  min-width: 200px; 
  max-width: 400px; /* Prevent box from getting wider than 400px */
  background-color: cornflowerblue; 
  padding: 10px; 
  margin: 1em auto; /* Center the box for better visibility */
}
  • We added margin: 1em auto; to center the box horizontally, making the width constraints easier to see.

✨ Observe the Preview:

  • The box is now centered.
  • If the preview window is wide, the box's width should be capped at 400px (plus padding/border).
  • If you make the window narrower, the box will shrink (because width is auto), but it won't go below 200px (due to min-width).
  • max-width provides an upper limit to prevent elements from becoming too large.
