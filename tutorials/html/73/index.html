Code & Preformatted Text Demo

Use the <code> tag for inline code snippets like document.getElementById('id') or CSS properties like font-family.

Use the <pre> tag for blocks of preformatted text, often code:

/* style.css snippet */
body {
  color: #333; /* Example Color */
  font-size: 16px;
}

.highlight {
  background-color: yellow;
}
