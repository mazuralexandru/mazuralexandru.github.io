460. background Shorthand: Example

Files Modified: None

Let's review the shorthand example we created for the .origin-box:

.origin-box {
  width: 300px;
  height: 150px;
  padding: 25px; 
  border: 10px dashed orange; 
  margin: 2em auto;
  
  background: 
      rgba(0, 100, 0, 0.2) /* color */
      url('images/placeholder-pattern.png') /* image */
      repeat /* repeat */
      scroll /* attachment */
      top left / 40px /* position / size */
      padding-box /* origin */
      content-box; /* clip */
}

This single background declaration sets:

  • background-color: rgba(0, 100, 0, 0.2)
  • background-image: url('images/placeholder-pattern.png')
  • background-repeat: repeat
  • background-attachment: scroll
  • background-position: top left
  • background-size: 40px
  • background-origin: padding-box
  • background-clip: content-box

It effectively combines multiple longhand properties into one line. Remember that any properties not included in the shorthand are reset to their initial values.
