371. Introduction to Module 8: Tables

Welcome to Module 8! Having covered document structure, text formatting, lists, links, images, metadata, and global attributes, we now turn our attention to structuring tabular data using HTML tables.

Purpose of This Module:

This module will teach you how to use the various HTML table elements (<table>, <tr>, <td>, <th>, <caption>, <thead>, <tbody>, <tfoot>) to represent data that is logically organized in rows and columns. We will emphasize semantic correctness and accessibility, clearly distinguishing the appropriate use of tables for data from the outdated and harmful practice of using tables for page layout.
