189. The inherit Keyword

Files Modified: style.css

What if you want an element to inherit a property value that it doesn't normally inherit, or you want to explicitly state that it should inherit? You can use the inherit keyword.

  • Purpose: Forces an element to take on the computed value of a specific property from its parent element.
  • Syntax: property: inherit;
  • Use Case: Can be useful for properties that don't inherit by default (like border or background-color) if you want a child to explicitly match its parent. It can also be used to override a specific style on an element and force it back to inheriting from its parent.

✅ Observe style.css:

  • We've added a rule targeting the <span> inside .container.
  • We're setting its border property to inherit.
/* inherit keyword example */
.container span {
  font-weight: bold; 
  border: inherit; /* Inherit border style from parent <p> */
}

✨ Observe the Preview:

  • The <span> ("This is inside the paragraph.") should now also have the 1px dotted gray border that its parent <p> has (due to the p[title] rule).
  • Normally, border is not inherited. By setting border: inherit; on the <span>, we explicitly tell it to copy the computed border value from its parent <p>.

The inherit keyword provides explicit control over the inheritance mechanism.
