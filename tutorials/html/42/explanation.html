42. CSS Combinators: Adjacent Sibling Combinator (+)

The Adjacent Sibling Combinator (+) selects an element that immediately follows another specific element, provided both elements share the same parent.

What's New?

  • CSS Pruning: Removed the ul > li child combinator rule.
  • CSS Addition: Added a new rule using the adjacent sibling combinator: h2 + p.

Adjacent Sibling Combinator (+)

  • Syntax: Two selectors separated by a plus sign (+): element1 + element2.
  • Meaning: Selects any element matching element2 that comes immediately after an element matching element1 in the HTML source code, and both elements must share the same parent element. It only selects the single element immediately following.
  • Use Cases: Applying specific spacing or styling to the first element after a certain type (e.g., removing the top margin from the first paragraph after a heading), styling pairs of elements.

In our example:

  • The rule h2 + p { ... } targets any <p> element whose immediately preceding sibling element is an <h2>.
  • It applies a yellow color, bold font weight, and removes the top margin.
