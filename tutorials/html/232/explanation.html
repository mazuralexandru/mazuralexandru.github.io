232. The alt Attribute - Syntax & Basic Example

Let's add a valid <img> element that includes both src and alt.

Syntax:

  • The alt attribute is added inside the <img> tag, alongside src.
  • Its value is a string of text enclosed in quotes, describing the image.
    <img src="path/to/image.png" alt="Description of the image content or function">

Requirement:

  • As previously stated, alt must always be present.

Code Change:

We've added a complete and valid <img> element to index.html, including both src (using the placeholder service again for a visible example) and a descriptive alt attribute.

A screen reader would announce this text when encountering the image.

✨ Key Takeaway: A valid <img> element requires both src and alt attributes. The alt text should accurately describe the image's content or function.
