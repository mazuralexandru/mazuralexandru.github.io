154. href URL Types: Absolute URL Example

Let's create a functional link using an absolute URL.

Purpose & Use Case:

Absolute URLs are primarily used when linking to resources that are external to your own website – pages on different domains.

Code Change:

We have added an anchor (<a>) element to index.html. The href attribute contains a complete absolute URL pointing to the MDN Web Docs page for the anchor element itself. We've also added a <br> for formatting the introductory text.

<p>
  Example linking to an external site (MDN):<br>
  <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a"> <!-- Absolute URL -->
    MDN Documentation for &lt;a&gt; <!-- Link text -->
  </a> <!-- Added link with absolute URL -->
</p>

Let's break down this absolute URL: https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a

  • scheme: https
  • domain: developer.mozilla.org
  • path: /en-US/docs/Web/HTML/Element/a

✅ Observation:

  1. Examine the new <a> element in index.html, paying close attention to the complete URL in the href attribute.
  2. Look at the browser preview. You should see a standard hyperlink with the text "MDN Documentation for ".
  3. Try clicking the link! (You might need to right-click and choose "Open Link in New Tab" depending on your preview environment to avoid navigating away entirely). It should take you to the corresponding MDN documentation page in your browser.

✨ Key Takeaway: Use absolute URLs in the href attribute, including the scheme (https) and full domain name, to create links to external websites.
