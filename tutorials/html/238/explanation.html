238. The width and height Attributes - Definition & Syntax

Besides src and alt, two other very important attributes for the <img> element are width and height.

The width and height Attributes:

  • Definition: Specify the intrinsic dimensions (the actual size) of the image file.
  • Purpose: Primarily used by the browser to reserve space in the page layout before the image file finishes downloading. This helps prevent content jumping around as images load (Layout Shift).
  • Syntax:
    <img src="..." alt="..." width="value" height="value">
    • The value for both attributes should be an integer representing pixels.
    • Crucially, do NOT include units like px. Just the number is required (e.g., width="300", not width="300px").

Why Pixels?

  • Even though CSS allows various units, the HTML width and height attributes historically (and currently) expect unitless integer values representing pixels. The browser uses these pixel dimensions to calculate the image's aspect ratio for layout reservation.
