432. background-image: Multiple Backgrounds

Files Modified: style.css

You can layer multiple background images (including gradients) on the same element by providing a comma-separated list to background-image.

  • Layering Order: The first image/gradient listed is drawn on top, the second is drawn below the first, and so on. The background-color is always drawn underneath all image layers.
  • Transparency: If upper layers have transparency (e.g., PNGs with alpha, gradients with transparent stops, rgba() colors used in gradients), the layers below will show through.

✅ Observe style.css:

  • We've added a linear-gradient() before the url() in the background-image property. This gradient goes from semi-transparent black at the top to fully transparent at the bottom.
.bg-box {
    background-color: transparent; 
    /* Layer 1: Gradient (topmost) */
    /* Layer 2: Image URL */
    background-image: 
        linear-gradient(to bottom, rgba(0,0,0,0.5), transparent), 
        url('images/placeholder-pattern.png'); 
    padding: 2em;
    margin-bottom: 1em;
    border: 1px dashed #fff; 
    min-height: 150px; 
}

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview:

  • You should still see the repeating pattern image in the background of the box.
  • However, there should now be a darkening effect at the top of the box, fading out towards the bottom. This is the semi-transparent black linear gradient being drawn on top of the pattern image.
  • This demonstrates how multiple backgrounds can be layered for combined effects. You can apply other background properties (like background-repeat, background-position, etc.) independently to each layer using comma-separated values.
