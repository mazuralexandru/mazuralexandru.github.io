97. CSS Property: color

Files Modified: None

The color property sets the color of an element's foreground content, which is primarily its text.

Applies to: Text, list markers, generated content, etc. Inherited: Yes (child elements will typically inherit the text color of their parent unless overridden).

Common Value Types:

  1. Color Keywords: Predefined names like red, blue, black, white, green, tomato, cornflowerblue, transparent (fully see-through). There are many standard keywords.
  2. Hexadecimal (#RRGGBB or #RGB): Represents colors using Red, Green, and Blue components.
  3. RGB / RGBA (rgb(r,g,b) / rgba(r,g,b,a)): Also uses Red, Green, Blue values (0-255), with an optional Alpha channel (0-1) for transparency in RGBA.

We'll explore color values in more detail shortly.

Example Usage:

p { 
  color: navy; /* Using a keyword */
} 
h1 {
  color: #333333; /* Using hexadecimal */
}
span {
  color: rgba(255, 0, 0, 0.8); /* Using RGBA for semi-transparent red */
}
