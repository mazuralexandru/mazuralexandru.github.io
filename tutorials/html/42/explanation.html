42. Inline Semantics: Strong Importance (<strong>)

Similar to <em>, the <strong> element adds semantic meaning to inline text, but its purpose is different.

The Strong Importance Element (<strong>)

  • Purpose: Used to indicate that its content has strong importance, seriousness, or urgency compared to surrounding text. It implies the content is significant.
  • Semantics: Marks the enclosed text as having high importance. Think of warnings, key terms essential for understanding, or text that needs to stand out due to its weightiness.
  • Type: Like <em>, it's an inline element.
  • Syntax: Uses an opening <strong> and a closing </strong> tag.

Default Rendering:

Browsers typically render text within <strong> tags in a bold font weight by default. Again, this is a visual convention tied to the semantic meaning of importance. CSS can change this appearance.

Code Change:

We've now wrapped the word "importance" in the first paragraph and the word "second" in the second paragraph with <strong> tags in index.html.

<p>This is the introductory paragraph. We will add <em>emphasis</em> and <strong>importance</strong> to text within paragraphs like this one.</p> <!-- Added <strong> -->

<p>This is another paragraph. It serves as a <strong>second</strong> block of text for our examples.</p> <!-- Added <strong> -->

  2. Look at the browser preview. You should see:

✨ Key Takeaway: Use the <strong> element to mark text that has strong importance, seriousness, or urgency. It's an inline element, typically rendered as bold by default.
