162. Attribute Selectors: Syntax

Files Modified: index.html, style.css

Attribute selectors use square brackets [] around the attribute criteria. Here are the common forms:

  1. Presence [attribute]: Selects elements that simply have the specified attribute, regardless of its value.
    • Example: [title] selects any element with a title attribute.
  2. Exact Value [attribute="value"]: Selects elements where the attribute has an exact matching value.
    • Example: [type="submit"] selects elements with type="submit".
  3. Whitespace-separated list contains [attribute~="value"]: Selects elements where the attribute value is a list of space-separated words, and one of them exactly matches the value. (Often used with class, but .value is usually simpler).
    • Example: [data-tags~="news"] selects elements where data-tags contains the word "news".
  4. Starts with value-hyphen [attribute|="value"]: Selects elements where the attribute value is exactly "value" or starts with "value" immediately followed by a hyphen (-). Primarily used for language subcodes.
    • Example: [lang|="en"] selects elements with lang="en" or lang="en-US".
  5. Starts with value [attribute^="value"]: Selects elements where the attribute value begins with the specified string.
    • Example: a[href^="https://"] selects links whose href starts with "https://".
  6. Ends with value [attribute$="value"]: Selects elements where the attribute value ends with the specified string.
    • Example: img[src$=".png"] selects images whose src ends with ".png".
  7. Contains substring [attribute*="value"]: Selects elements where the attribute value contains the specified substring anywhere within it.
    • Example: a[href*="example.com"] selects links whose href contains "example.com".
