340. list-style-type Values for <ol>

Files Modified: style.css

Now let's apply some specific marker types to our ordered lists (<ol>).

Common values for list-style-type suitable for ordered lists include:

  • decimal: Standard decimal numbers (1, 2, 3...). (Default for <ol>)
  • decimal-leading-zero: Numbers padded with leading zeros (01, 02, ... 09, 10...).
  • lower-roman: Lowercase Roman numerals (i, ii, iii...).
  • upper-roman: Uppercase Roman numerals (I, II, III...).
  • lower-alpha or lower-latin: Lowercase letters (a, b, c...).
  • upper-alpha or upper-latin: Uppercase letters (A, B, C...).
  • lower-greek: Lowercase Greek letters (α, β, γ...).
  • Others exist for different scripts (Hebrew, Armenian, Georgian, etc.).
  • none: No marker.

✅ Observe style.css:

  • We've updated the rule for ol.
  • We're setting list-style-type: lower-alpha;.
  • We're also adding padding-left for indentation.
/* Keep unordered lists styled */
ul {
  list-style-type: square; 
  padding-left: 2em; 
  margin-bottom: 1em; 
}

/* Style Ordered Lists */
ol {
   /* list-style: none; <- Removed reset */
   list-style-type: lower-alpha; /* Use lowercase letters */
   padding-left: 2em; /* Add indentation back */
   margin-bottom: 1em; 
}

✨ Observe the Preview:

  • The "Morning Routine" list should now display markers a., b., c..
  • The "Top 3" list still uses Roman numerals (III, IV, V) because the type="I" attribute set directly on the <ol> tag in the HTML overrides the lower-alpha style set in the CSS (HTML attributes often have higher precedence for this specific property).
  • Both ordered lists should now be indented.
