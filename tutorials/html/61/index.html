Text Align Demo

Left Aligned: This is the default alignment for left-to-right languages like English. The text aligns to the left edge.

Center Aligned: Each line of text is centered horizontally within the container.

Right Aligned: This text aligns to the right edge of the container.

Justified: Text is stretched out so that each line (except potentially the last) fills the full width of the container by adjusting spacing between words. Use with caution, can create uneven gaps.
