226. Vertical Alignment (vertical-align)

Files Modified: index.html, style.css

The vertical-align property affects the vertical positioning of an inline-level element (like <span>, <img>, <a>) or a table-cell element (<td>, <th>) relative to its line box or table row.

  • Purpose: To control the vertical alignment of elements within a line of text or within a table cell. It does NOT vertically align block-level elements within their container (that requires other techniques like Flexbox, Grid, or positioning).
  • Applies To: Inline-level and table-cell elements.
  • Inherited: No.
  • Common Values (for inline elements):
    • baseline: (Default) Aligns the baseline of the element with the baseline of its parent.
    • sub: Aligns the element as a subscript.
    • super: Aligns the element as a superscript.
    • top: Aligns the top of the element with the top of the line box.
    • middle: Aligns the vertical midpoint of the element with the baseline plus half the x-height of the parent. Often used to vertically center images or icons alongside text.
    • bottom: Aligns the bottom of the element with the bottom of the line box.
    • text-top: Aligns the top of the element with the top of the parent's font.
    • text-bottom: Aligns the bottom of the element with the bottom of the parent's font.
    • <length>: Raises or lowers the element by a fixed amount relative to the baseline (e.g., 5px, -0.2em).
    • <percentage>: Raises or lowers the element relative to the line-height of the element itself.
