336. Nested List Example

Files Modified: index.html

Let's add a nested list to our "Favorite Fruits" example to show varieties.

✅ Observe index.html:

  • Inside the <li>Apples</li> item, we have added a nested <ul> containing specific apple varieties.
    <!-- Unordered List Example -->
    <h2>Favorite Fruits</h2>
    <ul>
        <li>Apples
            <!-- Nested List -->
            <ul>
                <li>Granny Smith</li>
                <li>Fuji</li>
            </ul>
        </li>
        <li>Bananas</li>
        <li>Oranges</li>
    </ul>

✨ Observe the Preview:

  • Under "Apples", you should now see an indented sub-list containing "Granny Smith" and "Fuji".
  • Notice that the browser likely uses a different bullet point style (e.g., circles instead of discs) for the nested list items, indicating the hierarchy visually.
