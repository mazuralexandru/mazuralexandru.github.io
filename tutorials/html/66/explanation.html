66. <title>: Placement

Files Modified: None

The placement rule for the <title> element is:

  1. It must be placed inside the <head> element.
  2. A document must contain exactly one <title> element.
<head>
  <!-- Other head elements like <meta>, <link> can go here -->
  <title>Correct Placement</title>
  <!-- Other head elements can also follow -->
</head>

Placing the <title> outside the <head> (e.g., in the <body>) or having multiple <title> elements is invalid HTML.
