327. Directional Overflow Properties: overflow-x & overflow-y

Files Modified: style.css

Instead of using the overflow shorthand (which applies the same behavior to both horizontal and vertical overflow), you can control overflow independently for each axis using overflow-x and overflow-y.

  • overflow-x: Controls overflow along the horizontal (x) axis.
  • overflow-y: Controls overflow along the vertical (y) axis.
  • Values: Both properties accept the same values as overflow: visible, hidden, scroll, auto.
  • Use Case: Allows different handling for horizontal vs. vertical overflow. For example, you might want vertical scrolling (overflow-y: auto;) but want to clip any horizontal overflow (overflow-x: hidden;).

✅ Observe style.css:

  • We've removed the overflow: auto; shorthand.
  • We've added overflow-y: scroll; (force vertical scrollbar always).
  • We've added overflow-x: hidden; (clip any horizontal overflow).
.overflow-box {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid orange;
  background-color: rgba(0, 0, 100, 0.2);
  margin-bottom: 1em; 
  /* overflow: auto; <- Removed */
  overflow-y: scroll; /* Always show vertical scrollbar */
  overflow-x: hidden; /* Clip horizontal overflow */
}

✨ Observe the Preview:

  • The box should now always have a vertical scrollbar (due to overflow-y: scroll;), even if the content technically fits.
  • There should be no horizontal scrollbar, and if any content did overflow horizontally, it would be clipped (due to overflow-x: hidden;).
