345. List Marker Image (list-style-image): Syntax

Files Modified: None

CSS allows you to replace the standard list marker (bullet or number) with a custom image using the list-style-image property.

  • Purpose: To use custom graphics as list markers for unique visual styling.
  • Syntax: list-style-image: url('path/to/your-image.png');
    • The value is the url() function, containing the path (relative or absolute) to the image file you want to use.
    • Common image formats like PNG, JPG, GIF, SVG can be used. Small, simple icons usually work best.
  • Placement: Like list-style-type, this property is typically set on the list container (<ul> or <ol>) or specific list items (<li>).
  • Fallback: If the image fails to load, the browser will typically fall back to displaying the marker specified by list-style-type. Therefore, it's good practice to set both list-style-image and a fallback list-style-type.
