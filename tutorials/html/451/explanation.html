451. background-origin: Syntax

Files Modified: None

The syntax is straightforward:

background-origin: <box> [, <box>]*;

Where <box> is one of the keywords:

  • border-box
  • padding-box (Default)
  • content-box

If multiple background images are defined, use a comma-separated list to specify the origin for each layer independently.
