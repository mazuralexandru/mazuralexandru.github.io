34. Example: Correct vs. Incorrect Nesting

Files Modified: None

Let's reinforce the proper nesting rule with clear examples:

Correct Nesting:

<article>
  <h1>Article Title</h1>
  <p>
    This is the first paragraph. 
    <strong>Important text</strong> inside the paragraph.
  </p>
</article>
  • <strong> is inside <p>, so </strong> comes before </p>.
  • <h1> and <p> are inside <article>, so </h1> and </p> come before </article>.

Incorrect Nesting (Overlapping Tags):

<article>
  <h1>Article Title
    <p>This is the first paragraph. 
  </h1> <!-- Incorrect: Closing h1 before p which is inside it -->
    <strong>Important text</strong> inside the paragraph.
  </p>
</article> <!-- Closing tags might be misinterpreted by browser -->

Always double-check that your closing tags match the corresponding opening tags in the correct reverse order.
