164. Linking to Fragments: Adding Target id Attributes

Before we can create links to page fragments, we need to define the target locations within the page. We do this by adding unique id attributes to the elements we want to link to.

Choosing Target Elements:

  • Commonly, headings (<h2>, <h3>, etc.) that mark the beginning of sections are given id attributes.
  • You can also add ids to paragraphs (<p>), divisions (<div>), or almost any other element to make it a linkable target.

id Attribute Rules Recap:

  • Uniqueness: The id value must be unique within the entire page.
  • Syntax: Should start with a letter, can contain letters, numbers, hyphens, underscores. Avoid spaces.

Code Change:

To make our page scrollable and provide targets, we've:

  1. Updated the <h1> and <title>.
  2. Added <style> tags within the <head> for temporary CSS to create vertical space (.spacer class) – purely for demonstrating the scrolling effect of fragment links. Normally, styles belong in the external style.css file.
  3. Added two <h2> elements, each with a unique id attribute (id="section1" and id="section2").
  4. Added some placeholder content and spacer <div>s.
  5. Added a basic nav structure where we will place the links later.
<head>
  <meta charset="UTF-8">
  <title>HTML Links - Fragments</title> <!-- Updated title -->
  <link rel="stylesheet" href="style.css">
  <style>
    /* Add some temporary height/spacing for scrolling demo */
    .spacer { height: 80vh; border-bottom: 1px dashed #555; margin-bottom: 20px;}
    h2 { margin-top: 40px; }
  </style>
</head>
<body>
  <h1>Linking to Page Fragments</h1>
  <p>We need target elements with unique 'id' attributes.</p>

  <!-- Links will go here -->
  <nav>
      <p>Navigation:</p>
      <!-- Add links later -->
  </nav>

  <div class="spacer">Scroll down...</div>

  <h2 id="section1">Section One</h2> <!-- Added id="section1" -->
  <p>This is the content of the first section. Lots of text here...</p>
  <p>More content for section one...</p>

  <div class="spacer">Scroll down further...</div>

  <h2 id="section2">Section Two</h2> <!-- Added id="section2" -->
  <p>This is the content of the second section. Even more text here...</p>
  <p>More content for section two...</p>

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Examine index.html. Locate the <h2> elements and their id attributes. Note the temporary CSS in <style> used only for this demo.
  2. The browser preview should now have more content and significant vertical space. You might need to scroll down to see "Section One" and "Section Two". These headings are now targetable locations because they have unique ids.

✨ Key Takeaway: To create destinations for fragment links, add unique id attributes to the target HTML elements (like headings or sections).
