565. Multimedia: Video Tracks (<track>) - WebVTT Format Overview

The external files linked via the <track> element's src attribute need to be in a specific format. The standard format is WebVTT (Web Video Text Tracks).

WebVTT Format:

  • File Extension: .vtt
  • MIME Type: text/vtt
  • Purpose: Defines timed text cues (text + start/end times) and associated metadata.
  • Simple Text File: You can create/edit .vtt files using a basic text editor.

Basic Structure:

  1. Signature: Must start with the line WEBVTT optionally followed by a space and text header.
  2. Blank Line: Requires at least one blank line after the signature.
  3. Cues: Consists of one or more text track cues. Each cue typically has:
    • (Optional) Cue Identifier: A unique name for the cue (e.g., chapter-1). Followed by a line break.
    • Timings: Start time --> End time (e.g., 00:00:05.000 --> 00:00:08.500). Format is HH:MM:SS.sss or MM:SS.sss. Followed by a line break.
    • (Optional) Settings: Cue settings like align:start, line:90%, position:10%, size:50%, vertical:lr. Appear after timings on the same line, space-separated.
    • Payload Text: The actual text to be displayed during the cue's time range. Can span multiple lines. Must be followed by a blank line to separate it from the next cue.

Example .vtt File (captions_en.vtt):

WEBVTT - English Captions for Example Video

NOTE This is a comment and will be ignored.

1 - intro
00:00:03.500 --> 00:00:07.200 align:middle
Welcome to this demonstration
of HTML5 video captions!

2
00:00:08.000 --> 00:00:11.500
Here we see the speaker talking.
[Background music fades]

00:00:12.100 --> 00:00:14.900
Now showing the code example...

Key Points:

  • Strict formatting is required (blank lines between cues are essential).
  • Comments start with NOTE.
  • Timestamps define when text appears and disappears.
  • Payload text is what gets displayed or read.

✅ Observation: No changes to index.html. Examine the example .vtt structure above. Understand that this is the kind of file you would create and link to using the <track src="..."> attribute. Creating accurate, synchronized VTT files often requires specialized captioning tools or services.

✨ Key Takeaway: Timed text tracks for <video>/<audio> use the WebVTT (.vtt) format, which defines text cues with specific start and end times. Correct formatting is crucial.
