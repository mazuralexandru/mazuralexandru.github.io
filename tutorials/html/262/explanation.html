262. The width Property: Purpose

Files Modified: index.html, style.css

Now that we understand the box model components, let's look at controlling the size of the content area specifically. The width property is used for this.

  • Purpose: By default (box-sizing: content-box;), the width property defines the width of the element's content area. It does not include padding, border, or margin.
  • Applicability: It primarily affects block-level elements (like <div>, <p>) and replaced elements (like <img>). It generally has no effect on non-replaced inline elements (like <span>, <em>) unless their display property is changed (e.g., to inline-block or block).
