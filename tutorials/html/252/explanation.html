252. Structural Elements (<br>, <hr>)

Files Modified: None

Finally, let's look at two simple but important elements related to text structure: <br> and <hr>.

  • <br> (Line Break):
    • Semantic Meaning: Represents a line break within a block of text. It's used when a division of lines is significant, such as in addresses or poetry, but where creating separate paragraphs (<p>) would be semantically incorrect.
    • Usage: Insert <br> where you want the line to break. It's a void element (no content, no closing tag).
    • Caution: Do not use multiple <br> tags simply to create vertical space between paragraphs or other elements. Use CSS margin or padding for that purpose. Overusing <br> for layout is bad practice.
    • Example:
      <p>123 Main Street<br>Anytown, CA 91234</p>
  • <hr> (Thematic Break / Horizontal Rule):
    • Semantic Meaning: Represents a thematic break between paragraph-level content. It signifies a shift in topic or scene within a section. It is not just a decorative line.
    • Default Styling: Browsers typically render it as a horizontal line across the available width, often with some vertical margin.
    • Usage: Place <hr> between blocks of content (like paragraphs) to indicate a thematic separation. It's also a void element.
    • Example:
      <p>This is the first topic.</p>
<hr>
