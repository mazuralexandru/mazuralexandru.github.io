256. <figure> Example: Multiple Images, Single Caption

A <figure> can contain more than one core content element, such as multiple related images, all described by a single <figcaption>.

Use Case: Grouping related visuals (e.g., different views of a product, steps in a visual process, a collection of related photos) under one common caption.

Code Change:

We've modified the <figure> in index.html to include two <img> elements (using different placeholder URLs/text) and updated the <figcaption>.

<body>
  <h1>Figures and Captions</h1>
  <p>A figure can group multiple related images under one caption:</p> <!-- Adjusted P -->

  <figure>
    <img src="https://placehold.co/150x100/FFB6C1/FFFFFF.png?text=Image+A"
         alt="Placeholder A: 150x100, light pink."
         width="150" height="100">
         <!-- First image -->
    <img src="https://placehold.co/150x100/87CEFA/FFFFFF.png?text=Image+B"
         alt="Placeholder B: 150x100, sky blue."
         width="150" height="100">
         <!-- Second image -->
    <figcaption>Fig. 1: Two related placeholder images (A and B).</figcaption>
    <!-- Updated figcaption for both images -->
  </figure>

  <p>The single figcaption applies to the whole group.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Examine the <figure> block in index.html. Note the two <img> tags followed by the single <figcaption>.
  2. Look at the browser preview. You should see the two placeholder images displayed side-by-side (or wrapping, depending on container width) followed by the single caption ("Fig. 1: Two related placeholder images...").
  3. This demonstrates how <figure> acts as a container for the group, and <figcaption> labels the entire group.

✨ Key Takeaway: You can place multiple <img> elements (or other content) within a single <figure> and use one <figcaption> to describe the collection as a whole.
