197. Relative Unit: em

Files Modified: None

The em (em) unit is relative to the computed font-size of the element itself. If used for the font-size property, it's relative to the font-size of the parent element.

  • Meaning: 1em equals the current font size. 2em is twice the current font size, 0.5em is half.
  • Use Cases:
    • Setting padding, margin, width, height relative to the text size within the element. This ensures spacing scales proportionally if the font size changes.
    • Setting font-size relative to the parent (though rem is often preferred for this now).
  • Pros: Creates scalable components where spacing is tied to text size.
  • Cons: Compounding Effect: When used for font-size on nested elements, the effect multiplies. If a parent has font-size: 1.2em and its child also has font-size: 1.2em, the child's actual size will be 1.2 * 1.2 = 1.44 times the grandparent's font size. This can become difficult to manage in deep structures.

Example:

div { font-size: 20px; }
p { 
  font-size: 0.8em; /* 0.8 * 20px = 16px */
  padding: 1em; /* 1 * 16px = 16px padding */
  margin-bottom: 0.5em; /* 0.5 * 16px = 8px margin */
}
p span {
  font-size: 1.5em; /* 1.5 * 16px = 24px (compounding!) */
}
