21. File & Project Management

Files Modified: None

Websites consist of multiple files (HTML, CSS, JS, images). A good code editor provides tools to manage these files effectively within a project structure.

  • Explorer/Sidebar: VS Code's Explorer view (usually in the left sidebar) shows the folder structure of your project.
  • Actions: You can easily:
    • Create new files and folders.
    • Rename, move, or delete files/folders.
    • Open files for editing.
    • Search within files and across the entire project.

How it helps:

  • Organization: Keeps your project files structured and easy to navigate.
  • Efficiency: Provides quick access to all project resources without leaving the editor.

You'll typically open the main folder containing your website files in VS Code to work on it as a project.
