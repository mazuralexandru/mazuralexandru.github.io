342. List Marker Position (list-style-position): outside (Default)

Files Modified: None

The list-style-position property controls whether the list marker appears inside or outside the main content block of the list item (<li>).

The default value is outside.

  • Behavior:
    • The marker (bullet, number) is placed to the left of the list item's text content, within the padding-left area of the parent <ul> or <ol>.
    • If the list item's text wraps onto multiple lines, subsequent lines will align vertically with the start of the first line's text, effectively creating a hanging indent where the marker sits outside the main text block.
  • Appearance: This is the standard list appearance most people are familiar with.
