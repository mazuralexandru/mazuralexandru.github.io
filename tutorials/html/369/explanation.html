369. Link State: :hover

Files Modified: style.css

The :hover pseudo-class is essential for providing interactive feedback when a user positions their mouse pointer over a link.

  • :hover:
    • Selects: An element (not just links, but commonly used on them) when the user's pointing device (like a mouse cursor) is hovering over it.
    • Purpose: To visually indicate that an element is interactive and currently under the user's pointer.

✅ Observe style.css:

  • We've added a rule for a:hover.
  • We're changing the color and adding back the text-decoration: underline;.
/* Style links on hover */
a:hover { 
  color: lightcoral; /* Change color on hover */
  text-decoration: underline; /* Add underline on hover */
}

✨ Observe the Preview:

  • Move your mouse cursor over any of the links in index.html.
  • When the cursor is over a link, its text color should change to light coral, and an underline should appear.
  • When the cursor moves away, the link should revert to its :link (cyan) or :visited (medium purple) style.
  • This immediate visual feedback is crucial for usability.
