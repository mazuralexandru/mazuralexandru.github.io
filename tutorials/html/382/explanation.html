382. Horizontal Nav (Inline Block): display: inline-block

Files Modified: style.css

The key step in the inline-block technique for horizontal navigation is to change the display property of the list items (<li>) from their default list-item (which behaves like block) to inline-block.

  • Effect: Setting display: inline-block; on the <li> elements causes them to:
    • Flow horizontally next to each other, like inline elements.
    • Respect width, height, padding, and vertical margins, like block elements.

✅ Observe style.css:

  • We've added display: inline-block; to the nav li rule.
nav li {
    margin: 0; /* Reset li margin */
    display: inline-block; /* Make list items flow horizontally */
}

✨ Observe the Preview:

  • The list items ("Home", "About", "Services", "Contact") should now appear horizontally next to each other, centered within the faint dark background of the <ul>.
  • They no longer stack vertically.
  • Because they are now inline-block, they respect the padding we added to the <a> tags inside them, creating clickable areas.
