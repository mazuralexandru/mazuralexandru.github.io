506. Reset: Preparing for Landmark Elements (<header>, <footer>, etc.)

We've established the difference between semantic and non-semantic elements and the appropriate use of generic containers (<div>, <span>).

Now, let's dive into the specific HTML5 elements designed to define the major structural regions or landmarks of a typical web page: <header>, <footer>, <nav>, and <main>.

To focus on these elements, we will remove the <div> and <span> examples and the temporary CSS used to style them.

Why Remove Code?

  • Focus: Isolate the concept of landmark elements and their specific semantic roles.
  • Clean Slate: Provide an empty <body> structure where we can build a basic page layout using semantic landmarks.
  • Remove Demo Styles: Eliminate the temporary CSS associated with the div/span examples.

Code Change:

  1. Removed the <div class="content-wrapper"> block (and its contents) from the <body> of index.html.
  2. Removed the temporary <style> block from the <head>.
  3. Updated the <h1> and paragraph text.
