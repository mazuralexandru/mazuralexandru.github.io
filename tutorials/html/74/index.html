Quotation Elements

The blockquote element is used for long quotations. For shorter, inline quotes, use the q element, like this: He said, That's fascinating! right in the middle of the sentence.

This is a longer quotation, potentially spanning multiple paragraphs, taken from another source. It indicates that this content is not the original author's own words.

Blockquotes are typically indented by browsers by default.
