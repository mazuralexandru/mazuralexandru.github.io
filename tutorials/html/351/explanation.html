351. Styling List Items (<li>): Backgrounds, Borders, Padding

Files Modified: style.css

Since <li> elements are block-level by default within their list container, you can style them individually just like other block elements using properties like background-color, border, and padding.

This allows you to create visually distinct list items, perhaps for task lists, feature comparisons, or styled navigation.

✅ Observe style.css:

  • We've added some background, padding, and border-radius specifically to the <li> elements within the ordered list (ol li).
/* Style specific list items */
ol li {
    background-color: rgba(144, 238, 144, 0.1); /* Faint light green */
    padding: 0.5em;
    border-radius: 4px;
}

✨ Observe the Preview:

  • Each item in the "Morning Routine" and "Top 3" ordered lists should now have a faint light green background, some internal padding, and slightly rounded corners.
  • The unordered list items remain unstyled in this way.

This demonstrates how you can target and style individual list items to achieve various visual effects.
