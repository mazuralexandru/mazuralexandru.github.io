99. CSS Color Values: Hexadecimal

Files Modified: None

Hexadecimal color values are a very common way to specify colors on the web. They represent colors using the RGB (Red, Green, Blue) color model.

  • Syntax: Starts with a hash symbol (#) followed by either three or six hexadecimal digits (0-9 and A-F).

    • Six-digit (#RRGGBB): Two digits for Red, two for Green, two for Blue. Each pair ranges from 00 (none) to FF (maximum intensity).
    • Three-digit shorthand (#RGB): Can be used when each RR, GG, BB pair has identical digits. #RGB is equivalent to #RRGGBB.

  • Examples:

    • #FF0000 (Pure Red: Max Red, no Green, no Blue)
    • #00FF00 (Pure Green)
    • #0000FF (Pure Blue)
    • #FFFFFF (White: Max Red, Max Green, Max Blue)
    • #000000 (Black: No Red, No Green, No Blue)
    • #CCCCCC or #CCC (A light gray)
    • #FA8072 (Salmon color - same as keyword salmon)

Pros: Allows specifying any color within the RGB gamut (millions of colors). Widely supported and understood. Compact. Cons: Less intuitive to read than keywords without a color picker tool. Doesn't directly support transparency (though an 8-digit #RRGGBBAA format exists, it has less browser support than rgba()).
