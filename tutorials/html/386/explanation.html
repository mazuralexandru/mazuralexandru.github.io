386. Reset: Preparing for Table Structure Elements (<thead>, <tbody>, <tfoot>)

We've established the core table grid (<tr>, <td>) and the importance of header cells (<th>) with explicit scope (scope).

Now, we'll introduce elements that add further semantic structure by grouping rows into logical sections: <thead>, <tbody>, and <tfoot>.

To focus on these grouping elements, we'll reset our table slightly, removing the row headers for simplicity, and keeping just the column headers and a couple of data rows.

Why Reset?

  • Focus: To clearly demonstrate how <thead>, <tbody>, and <tfoot> wrap existing <tr> elements.
  • Clarity: A simpler table makes the grouping structure more obvious.

Code Change:

  1. Modified the table in index.html back to a structure with only column headers (scope="col") and standard data rows (<td>).
  2. Updated the paragraph text to introduce the grouping elements.

  <p>Using thead, tbody, and tfoot to group rows semantically.</p> <!-- Adjusted P -->
