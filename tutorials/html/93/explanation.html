93. Unordered Lists (<ul>): Core Mechanics (Syntax & Type)

Let's look at the basic syntax and element type for <ul>.

Syntax:

  • The unordered list is created using an opening <ul> tag and a closing </ul> tag.
    <ul>
  <!-- Content (specifically <li> elements) goes here -->
</ul>
  • These tags act as the container for the entire list.

Element Category: Block-Level

  • The <ul> element is a block-level element.
  • This means, by default:
    • It starts on a new line.
    • It typically occupies the full width available to it within its parent container (though its content might not fill that width).
    • It creates vertical separation from preceding and succeeding elements (usually via default browser margins).
