50. <html>: Syntax

Files Modified: None

The syntax for the <html> element is straightforward:

<!DOCTYPE html>
<html>  <!-- Opening tag -->
  
  <!-- Content goes here: <head> and <body> elements -->

</html> <!-- Closing tag -->
  • It uses standard opening (<html>) and closing (</html>) tags.
  • It directly follows the <!DOCTYPE html> declaration.
  • All other elements that make up your page will be nested inside these tags.
  • As mentioned, it usually contains attributes like lang.

Remember to always include the closing </html> tag at the very end of your document content.
