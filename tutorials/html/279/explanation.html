279. The class Attribute - Behavior & Interactions: JavaScript

Classes are also essential for JavaScript interaction, providing a flexible way to select and manipulate groups of elements.

JavaScript Selection Methods:

  1. document.getElementsByClassName('classname')

    • Purpose: Selects all elements in the document that have the specified class name.
    • Return Value: Returns a live HTMLCollection. This is an array-like object containing all matching elements. "Live" means if elements with this class are added or removed from the DOM after the collection was created, the collection updates automatically.
    • Iteration: You typically need to loop through the HTMLCollection to access individual elements (e.g., using a for...of loop or Array.from().forEach()).
    • Example: let highlightedItems = document.getElementsByClassName('highlight');

  2. document.querySelectorAll('.classname')

    • Purpose: Selects all elements matching the provided CSS selector (in this case, a class selector).
    • Return Value: Returns a static NodeList. This is also an array-like object, but "static" means it's a snapshot of the elements that matched at the time the query was run. It does not automatically update if matching elements are later added or removed.
    • Iteration: Also typically requires looping (e.g., NodeList.forEach() is available directly, or use for...of).
    • Versatility: Can use more complex CSS selectors (e.g., document.querySelectorAll('p.info-text')). Generally preferred over getElementsByClassName due to its static nature (often less surprising behavior) and versatility with selectors.
    • Example: let infoParagraphs = document.querySelectorAll('.info-text');

  3. Scoped Selection: You can also use these methods on a specific parent element to find descendants with a certain class:

    • parentElement.getElementsByClassName('classname')
    • parentElement.querySelectorAll('.classname')

JavaScript Class Manipulation (element.classList)

  • As discussed previously (when comparing id and class), the element.classList API is the modern and best way to dynamically add, remove, toggle, or check for classes on an element using JavaScript.
    • element.classList.add('is-active');
    • element.classList.remove('is-hidden');
    • element.classList.toggle('highlighted');
    • element.classList.contains('error');
  • Use Case: This is fundamental for creating dynamic user interfaces. For example, clicking a button might add an is-active class to a menu using JavaScript, and CSS rules defined for .is-active would then make the menu visible.
