123. DevTools: Inspecting Elements

Files Modified: None

The Elements panel makes it easy to pinpoint specific elements on the page and see their corresponding HTML code.

Methods:

  1. Selector Tool:
    • Click the "Inspect Element" icon, which usually looks like a square with a mouse pointer arrow (it might be in the top-left corner of the DevTools panel).
    • Move your mouse pointer over the rendered web page. As you hover over different parts (like the heading or paragraph), they will be highlighted on the page, and the corresponding HTML code will be highlighted simultaneously in the Elements panel.
    • Click on an element on the page to select it permanently in the Elements panel.
  2. Hovering/Clicking in DOM Tree:
    • As you hover over different lines of code (elements) within the Elements panel itself, the corresponding element will usually be highlighted on the rendered page.
    • Clicking on a line of code in the Elements panel selects that element.

This direct link between the visual page and the underlying code is incredibly useful for understanding how the page is built and for finding the code responsible for a specific visual part.
