56. <head>: Placement

Files Modified: None

The placement of the <head> element within the overall HTML structure is fixed:

  1. It must be a direct child of the <html> element.
  2. It must appear before the <body> element.
<!DOCTYPE html>
<html lang="en">

  <head>  <!-- CORRECT: Inside <html>, before <body> -->
    <meta charset="UTF-8">
    <title>Document</title>
  </head>

  <body> <!-- <body> follows <head> -->
    <!-- Visible content -->
  </body>

</html>

Placing the <head> after the <body>, or outside the <html> tags, is invalid HTML and will lead to errors or unpredictable behavior. There should only be one <head> element per document.
