The <form> Element: Enctype

Summary of enctype values (for POST):

  • application/x-www-form-urlencoded (Default): Encodes spaces (+/%20), special chars (%XX). Use for text data.
  • multipart/form-data: **Required** for <input type="file">. Sends data in parts, no encoding.
  • text/plain: Spaces to +, no other encoding. Debugging only.
