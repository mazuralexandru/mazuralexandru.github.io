05. Adding Visible Content with Paragraphs

Our page structure is set up, but it's still visually empty. Let's add our first piece of visible content inside the <body> element using a paragraph tag.

What's New?

We added a <p> element containing the text "Hello, World!" inside the <body>.

<!-- index.html snippet -->
</head>
<body>
  <p>Hello, World!</p> <!-- Added paragraph content -->
</body>
</html>

The <body> and <p> Elements

  • <body> Recap: As mentioned before, the <body> element is the container for all the content that users will actually see on the page.
  • <p> Element (Paragraph):
    • Purpose: Represents a paragraph of text. Browsers typically render paragraphs with some vertical space separating them from adjacent elements.
    • Syntax: It requires an opening tag (<p>) and a closing tag (</p>). The text content goes in between.
    • Semantic Meaning: Using <p> tells browsers (and screen readers, search engines) that this block of text is a distinct paragraph, giving it structural meaning beyond just being plain text.
