378. Image Links Styling

Files Modified: None

When an image (<img>) is wrapped in a link (<a>), you might want to apply visual feedback on hover or focus. Since the <img> itself isn't directly interactive in the same way text is, you typically style either the containing <a> tag or apply effects directly to the <img> using CSS.

Common Techniques:

  1. Styling the <a> Container:

    • Give the <a> tag display: inline-block; so it respects padding/borders.
    • Apply :hover or :focus-visible styles to the <a> tag, such as:
      • Adding an outline or border.
      • Changing the background-color (if appropriate).
      • Applying a subtle box-shadow.
    • Example:
      a.image-link { display: inline-block; }
a.image-link:hover,
a.image-link:focus-visible { outline: 2px solid orange; }

  2. Styling the <img> Directly:

    • Target the <img> inside the link on hover/focus states of the link.
    • Apply effects like:
      • opacity: Make the image slightly transparent or opaque.
      • filter: Use CSS filters like brightness(), contrast(), saturate(), or drop-shadow().
      • transform: Apply scaling (scale()) or slight movement (translate()). Remember to add transition for smoothness.
    • Example:
      a:hover img,
a:focus-visible img {
  opacity: 0.8;
  transform: scale(1.05);
  transition: opacity 0.2s ease, transform 0.2s ease;
}

Choosing the best technique depends on the desired visual effect. Applying styles to the <a> container often provides clearer focus indication, while styling the <img> directly allows for more visual effects on the image itself.
