109. Ordered Lists (<ol>): Exploring type Attribute Values

Let's quickly demonstrate another value for the type attribute to reinforce the concept. We'll change it from lowercase letters (a) to uppercase Roman numerals (I).

Recall type Values:

  • 1 (Default)
  • a
  • A
  • i
  • I

Code Change:

We have changed the value of the type attribute in the opening <ol> tag from "a" to "I" in index.html.

<ol type="I"> <!-- Changed type to "I" -->
  <li>First step (now 'I')</li>
  <li>Second step (now 'II')</li>
  <li>Third step (now 'III')</li>
</ol>

✅ Observation:

  1. Verify the change type="I" in the <ol> tag in index.html.
  2. Look at the browser preview. The list markers should now be uppercase Roman numerals: I., II., III. (or similar formatting).

✨ Experiment: Try changing the type attribute in index.html yourself to "A" or "i" (remembering to revert back to "I" or remove it afterwards if following along sequentially) to see the uppercase letter and lowercase Roman numeral markers.

✨ Reminder: While useful for demonstration, remember that CSS (list-style-type) is the preferred method for controlling marker appearance in modern web development.
