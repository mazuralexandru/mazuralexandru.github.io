135. Linking with <link> Element (HTML Metadata Element)

Files Modified: None

To make an external stylesheet apply to an HTML document, you must connect them using the <link> element.

  • Purpose: The <link> element defines relationships between the current HTML document and external resources. Its most common use is linking CSS stylesheets.
  • Placement: It's a void element (no closing tag) and must be placed within the <head> section of the HTML document.
  • Key Attributes:
    • rel="stylesheet": This attribute is crucial. It specifies the relationship of the linked resource to the HTML document. stylesheet tells the browser that the linked file is a CSS stylesheet that should be applied.
    • href="path/to/your/file.css": This attribute provides the Hypertext Reference (URL or path) to the external .css file. The browser uses this path to locate and download the stylesheet.
