391. Reset: Preparing for <caption> Element

We have now structured our table using row grouping elements (<thead>, <tbody>, <tfoot>).

The next step is to add a proper title or caption to the table using the <caption> element.

To focus clearly on the <caption> element, its placement, and its importance, we will simplify the table structure slightly by removing the <tfoot> section for now. We'll keep the <thead> and <tbody>.

Why Reset?

  • Focus: Isolate the <caption> element and its relationship to the <table> tag.
  • Clarity: A slightly simpler table structure makes the placement rule for <caption> (must be first child of <table>) easier to demonstrate.

Code Change:

  1. Removed the <tfoot>...</tfoot> block from index.html.
  2. Updated the paragraph text to introduce <caption>.
<!-- index.html snippet -->
<body>
  <h1>Adding a Table Caption</h1> <!-- Updated H1 -->
  <p>Preparing to add a caption using &lt;caption&gt;.</p> <!-- Adjusted P -->

  <table>
    <thead>
      <tr>
        <th scope="col">Header 1</th>
        <th scope="col">Header 2</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Data A1</td>
        <td>Data B1</td>
      </tr>
      <tr>
        <td>Data A2</td>
        <td>Data B2</td>
      </tr>
    </tbody>
    <!-- Removed tfoot -->
  </table>

  <p>The caption provides a title or description for the entire table.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Verify the removal of the <tfoot> element from index.html. The table now consists only of <thead> and <tbody> sections.
  2. The browser preview shows the table without the summary row at the bottom.

✨ Key Takeaway: Removing the <tfoot> simplifies the example, preparing for the focused introduction of the <caption> element and its specific placement rules.
