91. The style Attribute

Files Modified: None

The style attribute allows you to embed CSS declarations directly within an HTML tag.

Syntax: <element style="property: value; property2: value2;"> ... </element>

Example: <p style="color: blue; margin-left: 20px;">This paragraph has inline styles.</p>

In this example:

  • The style attribute is added to the <p> tag.
  • Its value contains two CSS declarations:
    • color: blue;
    • margin-left: 20px;
  • These styles apply only to this specific paragraph element.

This method directly mixes presentation rules into the HTML structure.
