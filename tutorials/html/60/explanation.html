60. <body>: Syntax

Files Modified: None

The syntax uses standard opening and closing tags:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Document</title>
  </head>
  <body>  <!-- Opening tag -->

    <!-- All visible page content goes here -->
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
    
  </body> <!-- Closing tag -->
</html>
  • It uses <body> and </body>.
  • It must be nested directly inside the <html> element.
  • It must come after the closing </head> tag.
  • The closing </body> tag typically comes just before the final closing </html> tag (though sometimes <script> tags are placed just before </body>).

All the content elements we'll learn about will be placed within these <body> tags.
