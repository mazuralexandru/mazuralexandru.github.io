196. Relative Unit: Percentage (%)

Files Modified: None

The percentage (%) unit defines a size relative to a corresponding value on the parent element. What it's relative to depends on the property being set:

  • width, height (and related like padding, margin in block direction): Percentage is relative to the width of the containing block (usually the parent element's content area). This is very common for creating fluid layouts where elements take up a proportion of their container.
    .container { width: 800px; }
.column { width: 50%; } /* 50% of 800px = 400px */
  • font-size: Percentage is relative to the font size of the parent element. font-size: 120%; is equivalent to font-size: 1.2em;.
  • line-height: Percentage is relative to the font size of the element itself. line-height: 150%; is equivalent to line-height: 1.5;.
  • top, bottom (for positioned elements): Percentage is relative to the height of the containing block.
  • left, right (for positioned elements): Percentage is relative to the width of the containing block.

Pros: Useful for fluid layouts, scaling relative to parent dimensions or font sizes. Cons: Can be confusing what the percentage is relative to for different properties. Can lead to compounding effects with font-size similar to em.
