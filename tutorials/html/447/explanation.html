447. background-attachment: scroll (Default)

Files Modified: None

This is the default behavior.

  • Behavior: The background image is positioned relative to the element's border box. When the page (or the element itself, if it's scrollable) is scrolled, the background image moves along with the element.
  • Effect: The background appears "attached" to the element and scrolls normally with the page flow.
