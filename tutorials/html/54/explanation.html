54. Structural Formatting: Line Breaks (<br>)

Now let's look at an element that directly affects the visual formatting within a block of text: the line break element, <br>.

The Line Break Element (<br>)

  • Purpose: Inserts a hard line break. It forces the content that follows it to start on the next line.
  • Type: It's a void (empty) element, meaning it has no content and no closing tag. It's also considered inline in terms of its effect within the flow of text, even though it causes a break.
  • Syntax (HTML5): <br> (The XHTML-style <br /> is also valid but less common in HTML5).
  • Placement: Used inside block-level elements where a line break is semantically appropriate (e.g., within <p>, <li>, table cells).

Behavior:

  • Visual Effect: Creates a line break without the extra vertical margin typically associated with starting a new paragraph (<p>).
  • Use Cases: Primarily used when the division of lines is intrinsic to the content, such as:
    • Lines of poetry.
    • Postal addresses.
    • Separating short, related pieces of information within a single logical block.

Code Change:

  2. Replaced the second paragraph's content with a postal address example using multiple <br> tags.

✨ Key Takeaway: Use <br> (a void, inline element) to force a line break within block-level content where required by the content's structure (like addresses or poetry).
