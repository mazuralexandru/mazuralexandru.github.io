431. background-image: Syntax (url())

Files Modified: style.css

The most common way to specify a background image is using the url() function.

  • Syntax: background-image: url('path/or/url/to/image.format');
    • The path inside url() follows the same rules as href or src (can be relative or absolute).
    • Quotes around the path are optional but recommended for clarity and to handle paths with special characters.

✅ Observe style.css:

  • We've added a background-image property to .bg-box, pointing to a placeholder image URL.
  • We've also temporarily set background-color: transparent; so the image is clearly visible against the body background.
.bg-box {
    background-color: transparent; /* Temporarily remove solid color */
    background-image: url('images/placeholder-pattern.png'); /* Add image */
    padding: 2em;
    margin-bottom: 1em;
    border: 1px dashed #fff; 
    min-height: 150px; /* Ensure box is tall enough to see pattern */
}
  • We added a min-height to ensure the box is tall enough to see the background effect.
