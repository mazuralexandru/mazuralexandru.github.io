75. Quotations: The cite Attribute for <q>

Just like <blockquote>, the inline <q> element also supports the optional cite attribute.

The cite Attribute on <q>:

  • Purpose: Provides a URL pointing to the source document or context for the inline quotation.
  • Syntax: <q cite="URL_of_source">...quoted text...</q>
  • Value: A valid URL.
  • Behavior: Identical to its behavior on <blockquote>:
    • Not visually displayed by default.
    • Provides machine-readable source information for assistive technologies, scripts, or browser tools.

Usage:

While perhaps less common than using cite on <blockquote>, it can be useful for providing direct source links for short inline quotes where applicable.
