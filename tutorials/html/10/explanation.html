10. Multiple Inline Styles & Background Color

You can apply multiple CSS declarations within a single style attribute by separating them with semicolons. Let's add a background color specifically to our paragraph.

What's New?

We added a second declaration, background-color: #444;, inside the existing style attribute on the <p> tag, separated from the first declaration by a semicolon.

<!-- index.html snippet -->
<body>
  <!-- This is our first paragraph -->
  <p style="color: lightcoral; background-color: #444;">Hello, World!</p> <!-- Added background-color -->

  <!-- Commented out paragraph remains -->
  <!-- <p>This paragraph is commented out and won't appear.</p> -->
</body>

Multiple Declarations in style Attribute

  • Just like in a CSS file's declaration block ({ ... }), you list property: value pairs separated by semicolons (;) within the style attribute's quotes.
  • The order generally doesn't matter unless you declare the same property twice (in which case the later one usually wins, though this is bad practice).

CSS background-color Property

  • The background-color property sets the background color of an HTML element.
  • It accepts the same types of color values as the color property (keywords, hex codes, RGB/RGBA, etc.).
  • Here, #444 is a hexadecimal color code representing a darker shade of gray than the body background, but lighter than pure black. The # indicates a hex code, followed by three pairs of hexadecimal digits (0-9, A-F) representing Red, Green, and Blue intensity. #444 is shorthand for #444444.
