250. <blockquote>, <q>, <cite>: Default Styling

Files Modified: index.html

Browsers apply default styles to visually distinguish these elements:

  • <blockquote>: Typically indented from both left and right margins.
  • <q>: Automatically adds language-appropriate quotation marks around the content (e.g., double quotes “” for English). Nested <q> elements often get different quote styles (e.g., single quotes ‘’).
  • <cite>: Typically rendered in italics.

✅ Observe index.html:

  • We've added a <blockquote> containing a paragraph and a <footer> with a <cite> element.
  • We've added a paragraph containing a <q> element.
    <hr>
    <blockquote cite="http://example.com/source">
      <p>This is a block quotation. It is typically indented by the browser to set it apart from the surrounding text.</p>
      <footer>— Some Author, <cite>Some Work Title</cite></footer>
    </blockquote>

    <p>She said, <q>This is an inline quotation, and the browser should add quotes.</q></p>

  • The inline quotation ("This is an inline quotation...") should have quotation marks automatically added around it by the browser (<q>).
