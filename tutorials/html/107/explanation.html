107. Ordered Lists (<ol>): Sequential Numbering

Let's add more items to our ordered list to observe the default sequential numbering.

Default Behavior:

When you add subsequent <li> elements inside an <ol>, the browser automatically increments the numerical marker for each item.

Code Change:

We've added two more <li> elements to the <ol> in index.html.

<ol>
  <li>First step</li>
  <li>Second step</li> <!-- Added second li -->
  <li>Third step</li> <!-- Added third li -->
</ol>

✅ Observation:

  1. Examine the three consecutive <li> elements within the <ol> in index.html.
  2. Look at the browser preview. You should now see:
    • "First step" marked with 1.
    • "Second step" marked with 2.
    • "Third step" marked with 3.
    • The numbering automatically follows a sequence based on the order of the <li> elements in the source code.

✨ Key Takeaway: By default, browsers render items in an <ol> with sequential numerical markers (1, 2, 3,...), reflecting the list's ordered nature.
