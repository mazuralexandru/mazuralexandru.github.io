213. text-decoration

Files Modified: style.css

The text-decoration property is primarily used to add lines to text (underline, overline, line-through).

  • Purpose: To decorate text, most commonly for underlining links or striking out deleted text.
  • Common Values:
    • none: Removes any text decoration (default for most elements). Often used to remove the default underline from links (a { text-decoration: none; }).
    • underline: Adds a line below the text. (Default for <a>, <ins>, <u>).
    • overline: Adds a line above the text.
    • line-through: Adds a line through the middle of the text. (Default for <s>, <del>).
  • Multiple Lines: You can specify multiple values space-separated (e.g., underline line-through).
  • Shorthand: text-decoration is actually a shorthand for several sub-properties:
    • text-decoration-line: Specifies the type of line (underline, overline, line-through).
    • text-decoration-color: Sets the color of the line.
    • text-decoration-style: Sets the style of the line (solid, double, dotted, dashed, wavy).
    • text-decoration-thickness: Sets the thickness of the line.

Example Usage:

a { 
  text-decoration: none; /* Remove default link underline */
  color: blue;
}
a:hover {
  text-decoration: underline; /* Add underline only on hover */
}
.deleted {
  text-decoration: line-through red wavy 2px; /* Fancy red wavy strikethrough */
}
