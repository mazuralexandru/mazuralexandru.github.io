281. Longhand Padding Properties: Values

Files Modified: None

Let's quickly recap the values accepted by the individual padding properties (padding-top, padding-right, padding-bottom, padding-left):

  1. Length Units:

    • px: Fixed pixel value (e.g., padding-top: 10px;).
    • em: Relative to the element's font size (e.g., padding-left: 1.5em;). Scales with text size.
    • rem: Relative to the root (<html>) element's font size (e.g., padding-bottom: 1rem;). Scales with base font size.
    • Other units like vw, vh are also possible.

  2. Percentages (%):

    • Calculated relative to the width of the element's containing block. This is true even for padding-top and padding-bottom.
    • Example: padding-left: 10%; means the left padding will be 10% of the parent container's width.

Padding values cannot be negative. The default value for all padding properties is 0.
