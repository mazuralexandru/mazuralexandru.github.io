235. Edits & Annotations (<s>, <del>, <ins>, <mark>)

Files Modified: None

HTML provides specific elements to semantically mark up edits, deleted/inserted content, and text that should be marked or highlighted for reference.

  • <s> (Strikethrough):
    • Semantic Meaning: Represents content that is no longer accurate or relevant. Distinct from <del> which implies actual removal from the document.
    • Default Styling: Renders with a line-through.
    • Example: <s>Was $99.99</s> Now $49.99!
  • <del> (Deleted Text):
    • Semantic Meaning: Represents content that has been deleted from the document. Often used in "diff" views or to show revisions.
    • Default Styling: Renders with a line-through.
    • Example: The event is on <del>Tuesday</del> Wednesday.
  • <ins> (Inserted Text):
    • Semantic Meaning: Represents content that has been added (inserted) into the document. Often used alongside <del> to show changes.
    • Default Styling: Renders with an underline.
    • Example: The event is on <del>Tuesday</del> <ins>Wednesday</ins>.
  • <mark> (Marked/Highlighted Text):
    • Semantic Meaning: Represents text marked or highlighted for reference purposes, due to its relevance in a particular context (like highlighting search terms in results). It's not for general emphasis or importance.
    • Default Styling: Renders with a yellow background (similar to using a physical highlighter pen).
    • Example: The results showed several instances of the term <mark>semantic HTML</mark>.

These tags provide specific meaning beyond simple presentation, which can be useful for assistive technologies or content processing.
