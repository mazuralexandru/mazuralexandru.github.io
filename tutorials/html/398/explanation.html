398. Basic Sizing: width, height: auto

Files Modified: style.css

Let's start by controlling the image size using basic CSS properties. A common requirement is to set a specific width and let the height adjust automatically to maintain the original aspect ratio.

  • width: value;: Sets the desired width (e.g., 300px, 50%, 20rem).
  • height: auto;: Tells the browser to calculate the height based on the specified width and the image's intrinsic aspect ratio. This prevents the image from being stretched or squashed vertically.

✅ Observe style.css:

  • We've added a rule for our .sample-image class.
  • We set width: 250px; and height: auto;.
.sample-image {
  width: 250px; /* Set a specific width */
  height: auto; /* Maintain aspect ratio */
  border: 2px solid yellow; /* Add border to see element bounds */
}

✅ Switch to index.html to recall the image element we are styling.

✨ Observe the Preview:

  • The image area (represented by the broken icon and yellow border) should now be 250px wide.
  • Its height should have adjusted proportionally based on the original 300x200 dimensions provided in the HTML attributes (so the height should be (200/300) * 250px = ~167px). The image space is no longer stretched or squashed, maintaining its correct proportions even though we only set the width in CSS.
