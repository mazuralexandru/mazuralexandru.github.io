276. Padding Fundamentals: Effect

Files Modified: None

Let's reiterate the key effects of padding:

  1. Increases Space Inside Border: Padding adds space around the content within the element's boundaries defined by the border.
  2. Background Extends: The element's background-color or background-image fills both the content area AND the padding area.
  3. Affects Overall Size (with content-box): When using the default box-sizing: content-box;, adding padding increases the total rendered width and height of the element. For example, an element with width: 100px; and padding: 10px; will actually render as 120px wide (100px content + 10px left padding + 10px right padding).
  4. Does Not Affect Size (with border-box): If you set box-sizing: border-box;, the width and height properties define the size including padding and border. Adding padding will shrink the available content area instead of expanding the overall box size. This is often more intuitive for layout.

Understanding how padding interacts with the box sizing model is essential for predictable layouts.
