347. List Style Shorthand (list-style)

Files Modified: style.css

Similar to padding, margin, and border, CSS provides a shorthand property for list styles: list-style.

  • Purpose: Allows you to set list-style-type, list-style-position, and list-style-image in a single declaration.
  • Syntax: list-style: value1 value2 value3;
    • You can provide one, two, or all three values.
    • The order of the values does not matter.
    • Any omitted properties are reset to their initial values (disc, outside, none respectively).
  • Common Use: Often used to reset all list styles with list-style: none;.

✅ Observe style.css:

  • We've modified the ul rule to use the list-style shorthand, combining the square type and inside position.
  • We've explicitly reset the ordered list styles using list-style: none;.
/* Style Unordered Lists */
ul {
  /* list-style-type: square; */ 
  /* list-style-position: inside; */
  list-style: square inside; /* Shorthand */
  padding-left: 2em; 
  margin-bottom: 1em; 
}

/* Style Nested Unordered Lists */
ul ul {
    list-style-type: circle; 
    margin-top: 0.25em; 
    margin-bottom: 0.5em; 
    /* Position is inherited as 'inside' */
}


/* Style Ordered Lists */
ol {
   /* list-style-type: lower-alpha; */
   /* padding-left: 2em; */
   list-style: none; /* Reset type and position, remove padding */
   padding-left: 0;
   margin-bottom: 1em; 
}

✨ Observe the Preview:

  • The "Favorite Fruits" list should look the same as before (square markers, positioned inside).
  • The ordered lists ("Morning Routine", "Top 3") should now have no markers and no indentation, as list-style: none; resets both the type and position (position defaults to outside but has no effect without a marker), and we also set padding-left: 0;.
