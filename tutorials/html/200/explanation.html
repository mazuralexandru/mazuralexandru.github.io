200. Reset: Removing tel: Link Example

We've now covered the mailto: and tel: special URI schemes used within href attributes.

The next crucial topic in Module 4 is the content of the anchor element itself – the part between the opening <a> and closing </a> tags – and its impact on usability and accessibility.

To focus on this, let's remove the tel: link example.

Why Remove Code?

  • Focus: Shift attention from the href attribute's value (protocols) to the content inside the <a> tags.
  • Clean Slate: Provide a neutral starting point for adding links with varying text content for analysis.

Code Change:

We have removed the <a> tag containing the tel: link example from index.html and updated the surrounding text.

<body>
  <h1>Link Content & Accessibility</h1> <!-- Updated H1 -->
  <p>Preparing to discuss link content best practices.</p> <!-- Adjusted P -->

  <p>
     <!-- Removed tel link example -->
     <!-- <a href="tel:+15551234567">Call Us at (555) 123-4567</a> -->
  </p>

  <p>The text between &lt;a&gt; tags is critical.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

  1. Verify that the tel: link example is gone from index.html.
