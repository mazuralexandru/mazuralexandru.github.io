58. Text Styling: font-style

The font-style property is primarily used to select between the normal, italic, and oblique faces within a font family.

.style-normal {
  font-style: normal; /* Default style */
  margin-bottom: 10px;
  font-size: 1.1rem;
}

.style-italic {
  font-style: italic; /* Selects the true italic face if available */
  margin-bottom: 10px;
  font-size: 1.1rem;
}

.style-oblique {
  font-style: oblique; /* Selects oblique face, often faked by slanting normal */
  font-size: 1.1rem;
}

font-style Explained

  • Purpose: Selects a font face classified as normal, italic, or oblique.
  • Values:
    • normal: Selects the standard, upright face of the font. This is the default.
    • italic: Selects the font face specifically designed as italic. True italics often have different letterforms compared to just slanting the normal characters. Requires the font family to have a dedicated italic face loaded.
    • oblique: Selects a font face classified as oblique. Oblique faces are typically just slanted (skewed) versions of the normal font face. If no specific oblique face is available, browsers can often simulate one by electronically slanting the normal face. You can optionally specify an angle, e.g., oblique 14deg, but browser support varies.
  • Italic vs. Oblique: Use italic when you want the specifically designed italic version (preferred if available). Use oblique if you specifically want a slanted version or if the font only provides an oblique variant. If a font has an italic face, oblique might still select the italic face as a fallback, or the browser might simulate the slant.

In Our Example:

  • We updated the Google Fonts link to explicitly request the italic variants (ital,wght@1,400;1,700) alongside the normal ones (ital,wght@0,400;0,700).
  • .style-normal uses the default normal.
  • .style-italic uses italic. Since Roboto has a true italic variant and we requested it, this should display the proper italic letterforms.
  • .style-oblique uses oblique. Since Roboto doesn't typically have a separate "oblique" face distinct from its "italic" face, the browser will likely either render the italic face or synthetically slant the normal face. The result might look very similar or identical to italic.
