194. Email Links (mailto:): Adding Body Content

You can also pre-fill the main body of the email message using the body parameter.

The body Parameter:

  • Key: body
  • Purpose: Pre-fills the content area of the email message.
  • Syntax: mailto:to@example.com?subject=Subj&body=Email%20body%20content%20here.

URL Encoding is Crucial for body:

  • The value of the body parameter often contains spaces and, importantly, line breaks.
  • These must be correctly URL-encoded for the email client to interpret them properly.
    • Space: %20
    • Line Break (Carriage Return + Line Feed - CRLF): %0D%0A (This is the standard way to encode a line break for email bodies in mailto: links).
  • Failing to encode line breaks will usually result in the body text appearing as one long, unbroken line in the email client.

Code Change:

We've created a new mailto: link example in index.html that includes a body parameter with encoded spaces (%20) and encoded line breaks (%0D%0A).

  </a>
</p>

Note the two %0D%0A sequences used to create a blank line in the email body.
