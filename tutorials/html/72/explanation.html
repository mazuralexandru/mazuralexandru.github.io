72. Quotations: The cite Attribute for <blockquote>

The <blockquote> element has an optional but useful attribute: cite.

The cite Attribute:

  • Purpose: Used to provide a URL that points to the source document or message from which the quotation was taken. It adds machine-readable provenance information.
  • Syntax: Added within the opening <blockquote> tag: <blockquote cite="URL_of_source">
  • Value: The value must be a valid URL (potentially surrounded by spaces). It could point to a specific web page, an article, a message ID, etc.

Important Behavior:

  • Not Visually Displayed: Browsers do not display the value of the cite attribute on the page. There's no default visual indication (like a footnote link) associated with it.
  • Machine-Readable: Its primary value is semantic and machine-readable. It can be accessed by:
    • Assistive technologies (though support may vary) to provide users with the source context if they request it.
    • JavaScript code to potentially extract and display source information dynamically.
    • Browser tools or extensions.
    • Search engines (potentially) for understanding the origin of the quoted content.

Code Change:

We've added a cite attribute to the existing <blockquote> element in index.html, pointing to the relevant section in the HTML specification as an example source.

<blockquote cite="https://www.w3.org/TR/html52/textlevel-semantics.html#the-blockquote-element"> <!-- Added cite attribute -->
  <p>This is a longer quotation, potentially spanning multiple paragraphs, taken from another source. It indicates that this content is not the original author's own words.</p>
  <p>Blockquotes are typically indented by browsers by default.</p>
</blockquote>

✅ Observation:

  1. Examine the opening <blockquote> tag in index.html and note the added cite attribute with its URL value.
  2. Look carefully at the browser preview. It should look exactly the same as in the previous step. The addition of the cite attribute has no visual effect.

✨ Key Takeaway: Use the optional cite attribute on <blockquote> to provide a URL pointing to the source of the quotation. This adds valuable machine-readable context, even though it's not displayed visually by default.
