437. background-repeat: Shorthand (1 vs 2 values)

Files Modified: None

As mentioned, the background-repeat property accepts either one or two values:

  • One Value: Sets both horizontal and vertical repetition simultaneously.
    • background-repeat: repeat; means repeat repeat
    • background-repeat: no-repeat; means no-repeat no-repeat
    • background-repeat: repeat-x; means repeat no-repeat
    • background-repeat: repeat-y; means no-repeat repeat
  • Two Values: Sets horizontal repetition first, then vertical repetition.
    • background-repeat: repeat space; (Repeat horizontally, space out vertically)
    • background-repeat: round no-repeat; (Round horizontally, don't repeat vertically)

Using one value is common when you want the same behavior on both axes (like no-repeat or repeat). Use two values when you need different horizontal and vertical tiling.
