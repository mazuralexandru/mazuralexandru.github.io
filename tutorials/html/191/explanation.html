191. Email Links (mailto:): Basic Syntax

Let's create a basic email link using the mailto: scheme.

Basic Syntax:

The simplest mailto: link includes only the recipient's email address directly after the mailto: prefix.

  • href="mailto:emailaddress@domain.com"

Code Change:

We've added an anchor element to index.html with an href attribute using the basic mailto: syntax.

<body>
  <h1>Special Link Protocols</h1>
  <p>Basic 'mailto:' link:</p>

  <p>
    <a href="mailto:contact@example.com">Email Us</a> <!-- Added basic mailto link -->
  </p>

  <p>Clicking attempts to open the default email client.</p>

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Examine the <a> tag in index.html. Note the href value starting with mailto: followed by an email address.
  2. Look at the "Email Us" link in the browser preview. It appears as a standard hyperlink.
  3. Try clicking the link. What happens will depend on your system setup:
    • If you have a default desktop email client (like Outlook, Mail, Thunderbird) configured, it should launch and open a new message composition window with "contact@example.com" pre-filled in the "To:" field.
    • If you don't have a default client configured, the link might do nothing, or your browser/OS might ask you which application to use.
    • It will not typically navigate to a web page.

✨ Key Takeaway: The basic syntax href="mailto:email@address.com" creates a link that, when clicked, tries to open the user's default email client addressed to the specified recipient.
