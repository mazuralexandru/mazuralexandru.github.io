85. CSS Declaration Block

Files Modified: None

The declaration block contains the actual style rules to be applied to the elements matched by the selector.

  • Syntax: It starts with an opening curly brace ({) and ends with a closing curly brace (}).
  • Content: Inside the braces, it holds one or more declarations.
selector { /* Start of Declaration Block */
  
  /* Declarations go here */

} /* End of Declaration Block */

Example:

body { 
  /* Everything between { and } is the declaration block */
  background-color: #2e2e2e; 
  color: #E0E0E0;            
  font-family: sans-serif; 
  padding: 15px; 
  min-height: 150px; 
  /* ... etc ... */
}

The curly braces group all the style instructions that belong to a specific selector.
