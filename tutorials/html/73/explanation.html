73. Quotations: <blockquote> Attribution

While the cite attribute provides a machine-readable link to the source document, how do you display the attribution (like the author or publication name) for the quotation visibly to the user?

Attribution Placement:

The HTML standard does not prescribe a single required way to attribute a <blockquote>, but common best practices involve placing the attribution text outside the <blockquote> element itself.

Common Patterns:

  1. Following Paragraph: Place the attribution in a separate <p> element immediately following the <blockquote>.
    <blockquote>
  <p>Quote content...</p>
</blockquote>
<p>— Author Name, Source Title</p>
  2. Using <figure> and <figcaption>: For self-contained quotes that might be considered figures, wrap the <blockquote> and its attribution within a <figure> element, using <figcaption> for the attribution. This semantically groups the quote and its caption.
    <figure>
  <blockquote>
    <p>Quote content...</p>
  </blockquote>
  <figcaption>— Author Name, <cite>Source Title</cite></figcaption>
</figure>
    (Note: We'll cover <figure> and <figcaption> in a later module, but it's relevant here. Also note the use of <cite> for the source title - see Topic 23).

What NOT to Do:

Do not put the attribution text directly inside the <blockquote> element unless it was part of the original quoted material. The <blockquote> should only contain the text exactly as quoted.

✅ Observation: No code changes in this step. We are discussing how to handle attribution, which typically involves adding other elements around the <blockquote>, not modifying the <blockquote> itself (beyond the optional cite attribute). Our current example doesn't yet include visible attribution.

✨ Key Takeaway: Display the visible attribution for a <blockquote> (author, source) in associated text outside the <blockquote> element itself, often using a following <p> or the <figure>/<figcaption> pattern. Use the cite attribute for the source URL.
