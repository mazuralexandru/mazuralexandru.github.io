290. Border Shorthand Properties: Side-specific Shorthands

Files Modified: style.css

What if you want to define the width, style, and color for just one side? You can use the side-specific shorthand properties:

  • border-top: width style color;
  • border-right: width style color;
  • border-bottom: width style color;
  • border-left: width style color;

These work just like the main border shorthand but only affect the specified side. Any sides not explicitly set will have no border (or inherit if applicable, though border properties generally don't inherit).

✅ Observe style.css:

  • We've removed the general border shorthand.
  • We've added a border-bottom shorthand to create only a bottom border.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin: 1em auto; 
  /* border: 3px dashed orange; <- Removed */
  border-bottom: 4px solid red; /* Add only a bottom border */
}

✨ Observe the Preview: The orange dashed border on all sides should be gone. Instead, the blue box should now have only a 4px solid red border along its bottom edge. The other three sides have no border.
