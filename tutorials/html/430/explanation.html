430. background-image Introduction

Files Modified: None

While solid colors are useful, CSS also allows you to set one or more images (or gradients, which are treated as images) as the background of an element using the background-image property.

  • Purpose: To apply decorative images or gradients behind an element's content.
  • Layering: You can specify multiple background images, separated by commas. The first image listed is the topmost layer, painted on top of subsequent images. The background-color is drawn beneath all images.
  • Syntax: background-image: none | <image> [, <image>]*
    • <image> is typically a url() function or a gradient function (linear-gradient(), radial-gradient(), etc.).
    • none explicitly removes any background image.

This property opens up many possibilities for visual design, from subtle patterns to full photographic backgrounds or complex layered effects.
