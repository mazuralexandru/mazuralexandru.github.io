234. Effective alt Text Example

Let's refine our example to show more realistic alt text for a non-placeholder image.

Imagine our src pointed to an actual logo file, images/logo.png. How should we write the alt text?

  • Scenario 1: Logo is just displayed (not a link).

    • The purpose is brand identification.
    • Good alt: alt="Example Corp Logo" or simply alt="Example Corp"

  • Scenario 2: Logo image is wrapped in a link to the homepage.

    • The purpose is navigation to the homepage.
    • Good alt: alt="Example Corp Homepage" or alt="Go to Example Corp Homepage"

Code Change:

We will modify the existing <img> tag in index.html. We'll change the src back to a relative path (imagining it's a logo) and update the alt text to reflect its function as a simple logo display (Scenario 1).

<body>
  <h1>Embedding Images</h1>
  <p>Example of an image with appropriate alt text:</p> <!-- Adjusted P -->

  <img src="images/logo.png"
       alt="Example Corp Logo">
       <!-- Changed src to relative, updated alt -->

  <p>The alt text should describe the image's content or function.</p> <!-- Adjusted P -->


  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Examine the updated <img> tag. Note the relative src and the new alt="Example Corp Logo".
  2. The browser preview will show a broken image icon again (because images/logo.png doesn't exist here).
  3. Crucially: If image loading failed on a real site, or for a screen reader user, the text "Example Corp Logo" provides meaningful information about what should be there. Compare this to the previous alt text which described the placeholder itself.

✨ Key Takeaway: Tailor your alt text to the specific content and function of the image within its context. Describe what it is or what it does.
