78. Preparing for Next Section: Removing <cite> Examples

We've now covered the quotation elements <blockquote> and <q>, their cite attributes, and the separate <cite> element for marking up titles.

To prepare for the next section, which focuses on other miscellaneous inline semantic elements (<small>, <sub>, <sup>, <s>), we will clean up our current examples by removing the paragraphs containing the <cite> element.

Why Remove Code?

  • Focus: Shift attention away from quotations and citations towards the next set of specific inline tags.
  • Relevance: The citation examples are not needed for understanding elements like <small> or <sub>/<sup>.
  • Clean Slate: Simplifies the page structure before introducing new inline elements.
