35. Analogy: Properly closing containers

Files Modified: None

Think about nesting HTML elements like stacking containers or using parentheses in mathematics:

  • Containers: If you put a small box (<p>) inside a larger box (<div>), you must close the lid of the small box (</p>) before you can close the lid of the large box (</div>).
  • Parentheses: In an expression like ( 5 * (3 + 2) ), you evaluate the inner parentheses (3 + 2) first before dealing with the outer ones.

HTML nesting follows the same logic: close the most recently opened element first. This ensures a clear and unambiguous structure for the browser to interpret.
