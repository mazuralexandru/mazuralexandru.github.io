109. HTML Comments: Visibility & Security

Files Modified: None

A crucial point about HTML comments:

  • Not Rendered: Browsers do not display the content of HTML comments on the web page itself.
  • Visible in Source Code: However, comments are part of the HTML source code. Anyone can view the source code of a web page (usually by right-clicking and selecting "View Page Source" or using browser developer tools).

Therefore: Do NOT put sensitive information in HTML comments!

Avoid including things like:

  • Passwords or private keys
  • Internal server details
  • Confidential notes or data
  • Anything you wouldn't want a user or competitor to see.

Comments are for developer notes and code management, not for hiding secret information.
