377. Basic Structure: Building a Row with Multiple Cells

A table row (<tr>) almost always contains multiple cells (<td> or <th>) to represent the data across different columns for that row.

Structure:

  • You simply place the required number of <td> (or <th>) elements sequentially within the <tr> tags.
  • The number of cells in each row should generally be consistent throughout the table (unless using colspan or rowspan, covered later) to maintain a regular grid structure.

Code Change:

We've added a second <td> element to our first <tr> in index.html, creating a row with two cells (representing two columns).

<!-- index.html snippet -->
  <table>
    <tr>
      <td>Cell 1, Row 1</td>
      <td>Cell 2, Row 1</td> <!-- Added second data cell -->
    </tr>
  </table>

✅ Observation:

  1. Examine the <tr> in index.html. It now contains two <td> elements.
  2. Look at the browser preview. You should see the text "Cell 1, Row 1" and "Cell 2, Row 1" displayed side-by-side on the same line. This represents the first row with two columns. The browser automatically arranges the cells horizontally within the row. (Again, no visible table borders yet by default).

✨ Key Takeaway: Add multiple <td> elements within a <tr> to define the data for each column in that specific row. The browser arranges these cells horizontally.
