544. Multimedia: Video (<video>) - controls Attribute

For users to interact with the video (play, pause, adjust volume, etc.), you need to provide controls. The simplest way is using the boolean controls attribute.

The controls Attribute:

  • Purpose: Instructs the browser to display its default set of user interface controls for video playback.
  • Syntax: <video src="..." controls> (Boolean attribute).
  • Behavior:
    • When present, the browser renders its native control bar overlayed on or positioned near the video area.
    • The specific controls included (play/pause, progress bar/scrubber, time display, volume control, fullscreen button, captions/subtitles menu if <track>s are present) and their appearance vary significantly between different browsers (Chrome, Firefox, Safari, Edge).
    • Provides essential user interaction. Appearance varies across browsers. If omitted, no controls are shown by default, requiring custom controls built with JavaScript.
  • Accessibility/Usability: Highly recommended unless you are providing your own fully custom, accessible controls via JavaScript. Without controls, users (especially keyboard-only users) cannot operate the video player.

Code Change:

We've added the boolean controls attribute to our <video> tag in index.html.

<!-- index.html snippet -->
  <!-- Added controls attribute -->
  <video src="/videos/example.mp4" controls> <!-- Hypothetical path -->
    <p>Sorry, your browser doesn't support embedded videos.</p>
    <p><a href="/videos/example.mp4">Download the video</a> instead.</p>
  </video>

✅ Observation:

  1. Locate the controls attribute added to the opening <video> tag.
  2. Look at the browser preview. You should now see the browser's default video player controls appear within or below the video area (which likely still shows an error or black box because the src file doesn't exist). You can see buttons for play/pause, volume, etc.
  3. This demonstrates that adding the controls attribute enables the browser's native player UI.

✨ Key Takeaway: Add the boolean controls attribute to <video> to display the browser's default playback controls, making the video operable by users. It's essential for usability and accessibility unless building fully custom controls.
