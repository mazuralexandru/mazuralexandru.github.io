607. Graphics: SVG (<svg>) - The <path> Element

For drawing shapes beyond basic rectangles, circles, or polygons, SVG provides the powerful <path> element.

The <path> Element:

  • Purpose: Defines a shape by specifying a sequence of drawing commands (moving, drawing lines, curves, arcs) using a special mini-language within its d attribute.
  • Versatility: Almost any 2D shape imaginable can be created using <path>. Vector graphics editors primarily generate <path> data when exporting complex drawings.

The d Attribute (Path Data):

  • Contains a string of commands and coordinates.

  • Commands (Case sensitive: Upper=Absolute, Lower=Relative):

    • M x y / m dx dy: Move To (starts a new subpath, lifts the pen).
    • L x y / l dx dy: Line To (draws a straight line).
    • H x / h dx: Horizontal Line To.
    • V y / v dy: Vertical Line To.
    • C c1x c1y c2x c2y x y / c dc1x dc1y dc2x dc2y dx dy: Cubic Bézier Curve To (needs two control points).
    • S c2x c2y x y / s dc2x dc2y dx dy: Smooth Cubic Bézier Curve To (infers first control point).
    • Q c1x c1y x y / q dc1x dc1y dx dy: Quadratic Bézier Curve To (needs one control point).
    • T x y / t dx dy: Smooth Quadratic Bézier Curve To (infers control point).
    • A rx ry rot large-arc sweep x y / a ...: Elliptical Arc To.
    • Z / z: Close Path (draws line back to start of subpath).

  • Understanding Manually: While possible to write simple paths by hand, complex paths are almost always generated by vector editing software. Understanding the basic commands (M, L, Q, C, A, Z) helps in reading or modifying generated SVG code.

✅ Observation:

  1. Locate the <path> element and its d attribute. Try to conceptually trace the commands: Move to (10,80), draw a quadratic curve towards (90,80) using (50,10) as the control point, then draw a smooth quadratic curve towards (130,80).
  2. Examine the CSS rule for .complex-path.
  3. Look at the browser preview. Inside the dashed SVG border, you should see an orange, slightly thicker curved line representing the path defined by the d attribute.

✨ Key Takeaway: The <path> element defines complex shapes using commands and coordinates in its d attribute. It's the most versatile SVG shape element, often generated by vector editors, and can be styled via attributes or CSS.
