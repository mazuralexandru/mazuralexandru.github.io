266. The height Property: Purpose

Files Modified: style.css

Similar to width, the height property controls the vertical dimension of an element's content area (by default).

  • Purpose: Defines the height of the content area.
  • Applicability: Like width, it primarily affects block-level elements, replaced elements, and elements whose display allows height control (e.g., block, inline-block, flex, grid). It is ignored by default on non-replaced inline elements.
  • Default Behavior: Most elements have a default height of auto, meaning their height expands or shrinks vertically to fit their content.

✅ Observe style.css:

  • We've added height: 100px; to the .box rule.
.box {
  width: 75%; 
  height: 100px; /* Set explicit height */
  background-color: cornflowerblue; 
  padding: 10px; 
}

✨ Observe the Preview: The blue box should now have a fixed height of 100 pixels (plus the top/bottom padding). If the text content requires more vertical space than this, it might overflow the box (depending on the overflow property, which defaults to visible). If the content requires less space, the box will still maintain its 100px height.
