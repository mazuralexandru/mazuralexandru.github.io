352. Advanced Marker Styling (::marker): Purpose & Syntax

Files Modified: None

For more direct control over the list marker itself (the bullet or number), modern CSS provides the ::marker pseudo-element.

  • Purpose: To select and apply styles directly to the marker box of a list item (<li>), separating marker styling from the list item's content styling.
  • Syntax: You target the list item (or the list container) and append ::marker.
    • li::marker { ... } (Style markers for all list items)
    • ul li::marker { ... } (Style markers only for items in unordered lists)
    • .my-list-item::marker { ... } (Style marker for items with a specific class)
  • Relationship to list-style-*: The ::marker styles are applied after the list-style-* properties. You can use ::marker to override the color or size derived from the list item, or even replace the marker entirely using the content property within the ::marker rule.

This pseudo-element offers a more robust and direct way to customize markers compared to older techniques like background images or ::before pseudo-elements, although the range of properties applicable to ::marker is somewhat limited.
