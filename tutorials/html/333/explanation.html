333. Description List Example

Files Modified: index.html

Let's add a description list to define some web technologies.

✅ Observe index.html:

  • We've added a <dl> element containing <dt> and <dd> pairs.
    <hr>

    <!-- Description List Example -->
    <h2>Web Technologies</h2>
    <dl>
        <dt>HTML</dt>
        <dd>The standard markup language for creating web pages.</dd>
        
        <dt>CSS</dt>
        <dd>A stylesheet language used for describing the presentation of a document written in HTML.</dd>
        
        <dt>JavaScript</dt>
        <dd>A programming language that enables interactive web pages.</dd>
    </dl>

✨ Observe the Preview:

  • You should see the heading "Web Technologies".
  • Below it, each term (HTML, CSS, JavaScript) appears on its own line.
  • Each corresponding description appears below its term and is typically indented by the browser's default styling for <dd> elements.
