408. Decorative Styling: border

Files Modified: style.css

We've already used the border property. It draws a line around the padding edge of an element.

  • Shorthand: border: width style color; (e.g., border: 5px double #fff;)
  • Longhand: border-width, border-style, border-color.
  • Side-Specific: border-top, border-right, border-bottom, border-left.

✅ Observe style.css:

  • We've changed the border on .block-image to be thicker, solid, and a different color.
.block-image {
    display: block; 
    max-width: 100%; 
    height: auto; 
    width: 300px; 
    margin: 2em auto 3em auto; 
    /* border: 2px solid lightgreen; <- Changed */
    border: 4px solid steelblue; /* Thicker, solid, different color */
}
