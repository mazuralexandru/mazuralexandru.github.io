82. Basic CSS Syntax

Files Modified: None

Now, let's look at the fundamental syntax of a CSS rule. CSS code is made up of rulesets.

A basic ruleset has two main parts:

  1. Selector: A pattern that identifies which HTML element(s) the rule should apply to. (e.g., body, h1, p).
  2. Declaration Block: One or more declarations enclosed in curly braces ({ }).
    • Declaration: A property-value pair that specifies the style to apply.
      • Property: The stylistic aspect you want to change (e.g., color, font-size, background-color).
      • Value: The setting for that property (e.g., red, 16px, #ffffff).
    • Declarations are separated by a colon (:) and must end with a semicolon (;).

Structure:

selector {
  property: value;
  property2: value2; 
}

Example from base_styles.css:

body { /* Selector: targets the <body> element */
  background-color: #2e2e2e; /* Declaration 1 */
  color: #E0E0E0;            /* Declaration 2 */
  /* ... more declarations ... */
}
