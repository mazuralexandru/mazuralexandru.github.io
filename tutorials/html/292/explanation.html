292. Rounded Corners (border-radius): Syntax & Values

Files Modified: style.css

The border-radius property is a shorthand that can accept one to four values to control the rounding of different corners.

  • Syntax: border-radius: value(s);
  • Values: Typically length units (px, em, rem) or percentages (%). Percentages are relative to the corresponding dimension of the element's border box (50% often creates a circle/ellipse).
  • Shorthand Application (like padding/margin):
    • One Value: border-radius: 10px; (Applies 10px radius to all four corners).
    • Two Values: border-radius: 10px 30px; (10px for top-left/bottom-right, 30px for top-right/bottom-left).
    • Three Values: border-radius: 10px 20px 30px; (10px top-left, 20px top-right/bottom-left, 30px bottom-right).
    • Four Values: border-radius: 10px 20px 30px 40px; (Top-left, Top-right, Bottom-right, Bottom-left - clockwise).

✅ Observe style.css:

  • We've added border-radius: 15px; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin: 1em auto; 
  border-bottom: 4px solid red; 
  border-radius: 15px; /* Round all corners */
}

✨ Observe the Preview: The corners of the blue box (including the background and the bottom border) should now be noticeably rounded with a 15px radius.
