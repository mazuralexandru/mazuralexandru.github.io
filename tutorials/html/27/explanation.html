27. Opening Tags (<tagname>)

Files Modified: None

The opening tag signifies the start of an HTML element.

  • Syntax: It consists of the element's name (like p, div, h1) enclosed in angle brackets (< >).
    • Example: <h1>, <p>, <div>
  • Purpose: Tells the browser, "An element of this type starts here."
  • Attributes (Optional): Opening tags can also contain attributes, which provide additional information about the element. We'll cover attributes later (e.g., <p class="intro">).

Every element that has content or wraps other elements must begin with an opening tag.
