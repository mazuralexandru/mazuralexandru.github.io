237. <s>, <del>, <ins>, <mark>: Default Styling

Files Modified: index.html

Browsers apply distinct default styles to these elements:

  • <s> and <del>: Strikethrough text (text-decoration: line-through;).
  • <ins>: Underlined text (text-decoration: underline;).
  • <mark>: Yellow background (background-color: yellow; or similar, plus often color: black;).
