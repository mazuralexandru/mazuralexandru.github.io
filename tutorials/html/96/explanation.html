96. Basic CSS Properties & Values (First Look)

Files Modified: None

We've seen a few CSS properties in action (color, font-size, background-color). Let's formally introduce a few basic ones and the types of values they accept.

We'll cover:

  • color (foreground/text color)
  • background-color (element background)
  • font-size (text size)

Understanding properties and their corresponding value types is fundamental to using CSS.
