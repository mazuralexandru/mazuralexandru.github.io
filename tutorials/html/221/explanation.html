221. white-space

Files Modified: style.css

The white-space property controls how whitespace (spaces, tabs, newlines) inside an element is handled and how lines are wrapped.

  • Purpose: To manage text wrapping and the collapsing of whitespace characters.
  • Inherited: Yes.
  • Common Values:
    • normal: (Default) Collapses multiple whitespace characters into a single space. Text wraps when necessary. Newlines in the source code are treated like spaces.
    • nowrap: Collapses whitespace like normal, but prevents text from wrapping to the next line. Text will overflow its container if it's too long.
    • pre: Preserves whitespace and newlines exactly as they appear in the HTML source code. Text only wraps on explicit newline characters in the source. Acts like the HTML <pre> tag. Often used with monospace fonts for code blocks.
    • pre-wrap: Preserves whitespace and newlines like pre, but allows text to wrap normally if it exceeds the container width.
    • pre-line: Collapses sequences of whitespace into a single space, but preserves newline characters from the source code (text wraps on newlines and also when necessary).
    • break-spaces: Similar to pre-wrap but with slightly different handling of spaces at the end of lines for wrapping.

✅ Observe style.css:

  • Let's apply white-space: nowrap; to the <span> inside the first paragraph to prevent it from wrapping.
/* Descendant Combinator Example */
.container span {
  font-weight: bold; 
  border: inherit; 
  font-size: 1.5rem; 
  white-space: nowrap; /* Prevent this span from line-breaking */
  background-color: rgba(0, 0, 255, 0.2); /* Add background to see effect */
}
  • We added a semi-transparent background to make the span's boundaries clearer.

✨ Observe the Preview:

  • If the text "This is inside the paragraph." was previously wrapping onto a new line within the first paragraph (depending on window width), it should now stay on a single line, potentially overflowing the paragraph's width if the window is narrow. The nowrap value prevents line breaks within the span.
