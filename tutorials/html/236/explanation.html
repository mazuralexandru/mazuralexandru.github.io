236. When NOT to Use alt=""

Using alt="" correctly is important, but it's equally important to know when not to use it. Misusing it can make content inaccessible.

Do NOT use alt="" if:

  1. The Image Conveys Information: If the image contains data, illustrates a concept, shows details not in the text, or contributes in any way to understanding the content, it needs descriptive alt text.
    • Example: A chart showing sales figures. alt="" is wrong. Use alt="Chart showing sales increased 20% in Q4." (or similar).
  2. The Image Contains Text: If there's text within the image that's important for the user to read, that text must be included in the alt attribute. (Again, try to avoid images of text altogether if possible).
    • Example: An image of a button saying "Submit Order". alt="" is wrong. Use alt="Submit Order".
  3. The Image Is Functional (Link/Button): If the image acts as a link or button, it must have alt text describing the destination or action. alt="" would make the link/button completely inaccessible to screen reader users.
    • Example: A shopping cart icon link. alt="" is wrong. Use alt="View Shopping Cart".
  4. The Image Sets a Mood or Tone: Sometimes images don't convey hard data but contribute to the feel or context (e.g., a landscape photo opening an article about travel). While concise, some description is usually better than alt="" in these cases. alt="Sun setting over a tranquil beach."
  5. It's a Logo: Company logos represent brand identity and should generally have alt text identifying the company (e.g., alt="Example Corp"). If the logo is also a link to the homepage, describe the destination (e.g., alt="Example Corp Homepage").

Common Mistake Review:

  • Omitting alt: Invalid HTML. Leads to unpredictable screen reader behavior (reading filename, etc.).
  • Using alt="" Incorrectly: Making informative or functional images invisible to screen reader users.

Decision Tree Recap: Always ask: "If this image was removed, would the user lose any information or functionality?" If yes, provide descriptive alt text. If no, use alt="".

✅ Observation: No code changes. Review the examples in the previous step. The first image (logo) needs descriptive alt. The second image (decorative border) is correctly marked with alt="". Consider other types of images and whether alt="" would be appropriate.

✨ Key Takeaway: Only use alt="" for images that are truly decorative and provide zero informational or functional value. When in doubt, provide descriptive alt text.
