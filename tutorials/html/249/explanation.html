249. <blockquote>, <q>, <cite>: HTML Recap

Files Modified: None

Let's recap the roles of the quotation elements:

  • <blockquote>: For long, block-level quotations. Often indented by default. Can have a cite attribute (URL).
  • <q>: For short, inline quotations. Browsers usually add quotation marks automatically. Can have a cite attribute (URL).
  • <cite>: For the title of a creative work (book, film, etc.). Typically italicized by default.

Common Pattern:
