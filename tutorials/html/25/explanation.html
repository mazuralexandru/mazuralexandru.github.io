25. What is HTML? Revisited

Files Modified: None

Let's formally define HTML again before we dive into writing it.

HTML = HyperText Markup Language

  • HyperText: Text which contains links (<a> tags) to other texts (or resources). This linking capability is fundamental to how the web works, allowing navigation between pages.
  • Markup: A way of annotating text to define its structure or presentation. In HTML, we use tags (<tagname>) to "mark up" content, telling the browser what kind of content it is (e.g., a paragraph, a heading, an image).
  • Language: It has a defined vocabulary (the set of valid tags and attributes) and syntax (the rules for how to write them correctly).

It's crucial to remember that HTML is a markup language, not a programming language. It describes the structure of content, rather than providing instructions for the computer to execute complex logic (that's JavaScript's job).
