549. Multimedia: Video (<video>) - playsinline Attribute

The playsinline attribute addresses a specific behavior primarily on iOS devices (iPhone, iPad).

The playsinline Attribute:

  • Purpose: A boolean attribute that hints to the browser (especially mobile Safari on iOS) that the video should be allowed to play "inline" within the element's playback area in the document, rather than automatically entering fullscreen mode when playback begins.
  • Syntax: <video src="..." playsinline>
  • Historical Context: By default, older versions of iOS Safari would force videos to enter fullscreen mode upon playback. playsinline was introduced to override this for use cases like background videos or videos embedded as part of the page design.
  • Modern Behavior: While still important for ensuring inline playback on iOS, default behaviors might vary on newer OS versions or other mobile browsers. Including it provides the strongest hint for inline playback.

Use Cases:

  • Essential for background videos that should remain part of the page layout while playing.
  • Necessary for any video where you want playback to occur within its container on the page (e.g., in a card component, alongside text) on iOS devices, without forcing fullscreen.

Code Change:

We've removed the preload attribute from the previous example and added the boolean playsinline attribute.

<!-- index.html snippet -->
  <!-- Removed preload, added playsinline -->
  <video controls width="480" height="270"
         src="/videos/example.mp4" <!-- Hypothetical path -->
         playsinline>
    <p>Sorry, your browser doesn't support embedded videos.</p>
  </video>

✅ Observation:

  1. Locate the playsinline attribute on the <video> tag.
  2. The browser preview on a desktop machine will likely show no difference.
  3. (Conceptual): If you viewed this page on an iPhone and played the video (if it existed), the playsinline attribute would instruct Safari to play it within the 480x270 area defined on the page. Without playsinline, Safari might automatically switch to its native fullscreen player when play is initiated.

✨ Key Takeaway: Use the boolean playsinline attribute on <video> elements to enable inline playback (preventing automatic fullscreen) on iOS devices and potentially other mobile browsers. Essential for background videos or videos integrated into page layouts on iOS.
