63. Essential Metadata: Title & Character Set

Files Modified: None

While our current index.html is structurally valid, it's missing two crucial pieces of metadata that should be present in virtually every web page:

  1. The Document Title (<title>): Defines the text that appears in the browser tab or window title bar.
  2. Character Encoding Declaration (<meta charset="UTF-8">): Ensures the browser correctly interprets the characters used in the document.

These elements belong inside the <head> section. Let's explore the <title> element first.
