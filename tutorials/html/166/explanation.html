166. Universal Selector: Syntax

Files Modified: None

The syntax is simply the asterisk character:

* {
  box-sizing: border-box;
}

It can also be used in combination with other selectors, although this is less common. For example, div * would select any element that is a descendant of a <div>.

Its specificity value is zero (0,0,0,0), meaning it's easily overridden by any other selector (type, class, ID, attribute).
