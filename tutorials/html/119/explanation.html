119. Viewing Page Source

Files Modified: None

Sometimes, you want to see the raw HTML code exactly as it was written in the .html file, including comments and the original formatting.

  • How: Right-click on the background of the web page in your browser and select "View Page Source", "View Source", or a similar option. Alternatively, use a keyboard shortcut (often Ctrl+U or Cmd+Option+U).
  • What it shows: This opens a new tab or window displaying the plain text content of the HTML file that the browser received. It shows the <!DOCTYPE>, <html>, <head>, <body>, all tags, attributes, text content, and importantly, HTML comments.
  • Difference from DevTools: This is different from the "Elements" panel in the browser's Developer Tools. "View Source" shows the original source code, while the DevTools "Elements" panel shows the live, parsed DOM tree, which might have been modified by JavaScript or corrected by the browser's error handling.

Viewing the source is useful for seeing the exact HTML structure before any browser interpretation or dynamic changes occur, and for easily seeing HTML comments.
