188. Non-Inherited Properties

Files Modified: None

Most CSS properties related to layout, dimensions, positioning, backgrounds, borders, padding, and margins are not inherited by default.

Common Non-Inherited Properties:

  • Box Model:
    • width, height (and min/max variants)
    • padding (and its longhand properties)
    • border (and its longhand properties)
    • margin (and its longhand properties)
    • box-sizing
  • Backgrounds:
    • background-color
    • background-image
    • background (shorthand)
  • Positioning:
    • position
    • top, right, bottom, left
    • z-index
  • Layout (Flexbox/Grid):
    • display
    • Flex container properties (flex-direction, justify-content, etc.)
    • Grid container properties (grid-template-columns, gap, etc.)
    • Flex/Grid item properties (flex-grow, grid-column, etc.)
  • Other:
    • float, clear
    • overflow
    • vertical-align (though it affects how an element aligns relative to its parent's line, the property itself isn't inherited)
