335. Nested Lists

Files Modified: None

Lists can be nested inside other lists to create hierarchical structures or outlines.

  • Purpose: To represent sub-items or sub-categories within a main list item.
  • Syntax: A nested list (<ul> or <ol>) must be placed inside a list item (<li>) of the parent list. It cannot be placed directly inside the parent <ul> or <ol> tags between list items.
  • Correct Nesting:
    <ul>
  <li>Parent Item 1</li>
  <li>Parent Item 2
    <ul> <!-- Nested list starts INSIDE the parent <li> -->
      <li>Child Item 2.1</li>
      <li>Child Item 2.2</li>
    </ul> <!-- Nested list ends INSIDE the parent <li> -->
  </li> <!-- Parent <li> ends -->
  <li>Parent Item 3</li>
</ul>
  • Incorrect Nesting:
    <ul>
  <li>Parent Item 1</li>
  <li>Parent Item 2</li> 
  <ul> <!-- INCORRECT: Nested list between <li> elements -->
    <li>Child Item 2.1</li>
    <li>Child Item 2.2</li>
  </ul>
  <li>Parent Item 3</li>
</ul>
  • Default Styling: Browsers typically apply different marker styles (e.g., disc -> circle -> square for <ul>) and increase indentation for nested lists.
