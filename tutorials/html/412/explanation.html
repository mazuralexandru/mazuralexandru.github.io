412. CSS Image Filters (filter) Introduction

Files Modified: None

Beyond basic box styling, CSS offers the filter property to apply graphical effects, similar to those found in photo editing software, directly to elements (including images).

  • Purpose: To modify the rendering of an element by applying effects like blur, brightness adjustment, contrast changes, grayscale conversion, hue rotation, sepia tones, and more.
  • Applicability: Can be applied to any element, but commonly used on images (<img>), videos (<video>), or sometimes containers to affect all their children.
