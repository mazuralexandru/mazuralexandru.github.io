223. overflow-wrap Example

Files Modified: index.html

To see overflow-wrap: break-word; in action, we need some text that would normally overflow its container.

✅ Observe index.html:

  • Inside the first paragraph (.highlight-text), we've added a very long, unbreakable string of characters.
    <p class="highlight-text" title="Standard Paragraph">
        This is a standard paragraph with a verylongandunbreakablewordlikethisoneaddedtoit. 
        <span>This is inside the paragraph.</span> 
    </p>

✨ Observe the Preview:

  • Look at the first paragraph. The long word "verylongandunbreakablewordlikethisoneaddedtoit" should now break onto multiple lines if necessary to fit within the paragraph's width, rather than overflowing out of the yellow background area.
  • This breaking within the word happens because we applied overflow-wrap: break-word; to the .highlight-text class in the previous step. Without this property, the long word would likely extend beyond the container's edge.
