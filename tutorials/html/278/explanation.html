278. The class Attribute - Behavior & Interactions: CSS Styling

The most common and fundamental use of the class attribute is to serve as a hook for CSS styling.

CSS Class Selector:

  • In CSS, you target elements based on their class name using the class selector, which consists of a period (.) followed immediately by the class name.
    • Syntax: .classname { ...css properties... }
    • Example: .intro-paragraph { ... } targets all elements with class="intro-paragraph" (or elements having intro-paragraph within a list of classes).

Specificity:

  • Class selectors have a medium level of specificity, lower than ID selectors (#id) but higher than type selectors (p).
  • This makes them ideal for general styling, as styles applied via classes can be reasonably overridden by more specific selectors (like IDs or more complex class combinations) if needed, but they easily override default type styles.

Applying Styles:

  • You define rules in your CSS file using class selectors to apply specific visual appearances to all elements sharing that class.
  • This promotes reusability and consistency. Define the style once in CSS, and apply the class to any number of HTML elements.

Code Change:

We will now add CSS rules to style.css targeting the classes we added to the paragraphs in the previous step.

/* style.css - Add AFTER base body style */

/* Style elements with the 'intro-paragraph' class */
.intro-paragraph {
  font-style: italic;
  margin-bottom: 2em; /* Add extra space below */
}

/* Style elements with the 'highlight' class */
.highlight {
  background-color: #444; /* Slightly lighter dark background */
  border-left: 3px solid #F0E68C; /* Khaki border */
  padding: 10px;
}

/* Style elements with the 'info-text' class */
.info-text {
  font-size: 0.9em;
  color: #BDBDBD; /* Lighter grey text */
}

✅ Observation:

  1. Verify the new CSS rules targeting .intro-paragraph, .highlight, and .info-text have been added to style.css.
  2. Look at the browser preview. You should see significant visual changes:
    • The first paragraph (with class="intro-paragraph highlight") should now be:
      • Italic (.intro-paragraph).
      • Have extra space below it (.intro-paragraph).
      • Have a different background color and a left border (.highlight).
      • Have some internal padding (.highlight). (It receives styles from both classes).
    • The second paragraph (with class="info-text") should now be:
      • Slightly smaller font size (.info-text).
      • A different shade of light grey text (.info-text).

✨ Key Takeaway: Use the CSS class selector (.classname) to apply styles defined in your CSS file to all HTML elements marked with the corresponding class attribute. This is the standard way to achieve reusable and maintainable styling.
