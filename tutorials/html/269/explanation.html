269. height Applicability

Files Modified: None

The applicability rules for height are the same as for width:

  • It affects block-level elements, replaced elements, and elements with display set to block, inline-block, flex, grid, etc.
  • It is ignored by default on non-replaced inline elements (<span>, <a>, etc.).

Remember the caveat about percentage heights requiring an explicit height on the parent container in many common layout scenarios.
