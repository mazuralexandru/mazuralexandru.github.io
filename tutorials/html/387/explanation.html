387. Horizontal Nav (Flexbox): Alignment (justify-content, align-items)

Files Modified: style.css

Flexbox provides powerful properties for aligning items within the flex container.

  • justify-content: Controls alignment of items along the main axis (the direction items flow - row by default for display: flex).
    • flex-start (Default): Items packed toward the start line.
    • flex-end: Items packed toward the end line.
    • center: Items centered along the line.
    • space-between: Items evenly distributed; first item at the start, last item at the end, space is maximized between items.
    • space-around: Items evenly distributed with equal space around each item (half-size space at ends).
    • space-evenly: Items evenly distributed with equal space between them and at the ends.
  • align-items: Controls alignment of items along the cross axis (perpendicular to the main axis). For a default row direction, this controls vertical alignment.
    • stretch (Default): Items stretch to fill the container's cross size (height in a row).
    • flex-start: Items packed toward the cross-start edge (top in a row).
    • flex-end: Items packed toward the cross-end edge (bottom in a row).
    • center: Items centered on the cross axis (vertically centered in a row).
    • baseline: Items aligned based on their text baselines.

✅ Observe style.css:

  • We've added justify-content: center; to the nav ul rule to center the navigation items horizontally within the ul container.
nav ul {
    display: flex; 
    list-style: none; 
    padding: 0;       
    margin: 0;        
    background-color: rgba(0, 0, 0, 0.2); 
    padding: 0.5em 0; 
    gap: 0.5em; 
    justify-content: center; /* Center items horizontally */
    /* align-items: center; /* Could be used for vertical centering if needed */
}

✨ Observe the Preview:

  • The navigation items ("Home", "About", etc.) should now be centered horizontally as a group within the faint dark background area of the <ul>.
  • Compare this to the text-align: center used in the inline-block method, which centered the inline content within the ul. Flexbox centers the flex items (<li>) themselves within the flex container (<ul>).
