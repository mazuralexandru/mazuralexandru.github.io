144. Type Selectors: Syntax

Files Modified: None

The syntax for a type selector is just the element's tag name.

  • To select all paragraphs (<p>):
    p {
  /* styles for paragraphs */
}
  • To select all level 2 headings (<h2>):
    h2 {
  /* styles for H2s */
}
  • To select all images (<img>):
    img {
  /* styles for images */
}
  • To select all links (<a>):
    a {
  /* styles for links */
}
  • To select the main body (<body>):
    body {
  /* styles for the body */
}

It's case-insensitive according to the specification, but convention dictates using lowercase tag names, matching the HTML convention.
