181. Specificity (Deep Dive)

Files Modified: None

When two conflicting declarations come from the same origin (e.g., both are normal author styles) and have the same importance, the browser uses Specificity to decide which one wins.

  • Purpose: Specificity is a weight or score calculated for each CSS selector. The selector with the higher specificity score wins, and its declaration is applied.
  • Concept: More specific selectors (like IDs or complex combinations) target elements more precisely than general selectors (like types). The cascade prioritizes these more specific rules.

Think of it like addressing mail: a full street address with a name (high specificity) is more precise than just sending mail to everyone on a street (lower specificity).
