190. Email Links (mailto:): Introduction

Besides http: and https:, the href attribute can use other URI (Uniform Resource Identifier) schemes to trigger specific actions. A very common one is mailto:, used to create links that open the user's email client.

The mailto: Scheme:

  • Definition: A URI scheme specifically designed for email addresses.
  • Purpose: When used in an href attribute (href="mailto:..."), it instructs the browser to attempt to launch the user's default desktop email client (e.g., Outlook, Apple Mail, Thunderbird) and pre-fill a new email message.
  • Goal: To provide users a convenient way to start sending an email to a specific address without needing to manually copy and paste it.

Basic Structure:

The simplest form just includes the email address after the mailto: prefix. We'll add parameters like subject and body later.

Behavior Note: The functionality entirely depends on the user having a default email client configured on their operating system. If no client is set up, clicking the link might do nothing or result in an error message. It doesn't reliably open webmail services like Gmail in the browser unless the user has specifically configured handlers for mailto: links.
