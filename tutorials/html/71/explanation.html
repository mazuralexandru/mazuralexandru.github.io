71. Quotations: Block Quotes (<blockquote>)

Let's begin exploring elements for marking up quotations, starting with <blockquote>, which is used for longer, block-level quotes.

The Block Quote Element (<blockquote>)

  • Purpose: Represents content that is quoted from another source. It's intended for quotations that stand apart from the surrounding text, often spanning one or more paragraphs.
  • Type: Block-level element.
  • Syntax: Uses an opening <blockquote> and closing </blockquote> tag.
  • Content Model: Can contain other block-level elements like <p> or lists, as well as phrasing content. It's common to put paragraphs (<p>) inside a <blockquote>.

Default Rendering:

Browsers typically render <blockquote> elements with indentation on both the left and right sides to visually set the quoted text apart from the main content flow. They might also add vertical margins. As always, this appearance can be controlled with CSS.
