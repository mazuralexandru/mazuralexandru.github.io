68. Text Styling: text-shadow

The text-shadow property allows you to add one or more shadow effects to text content.

  text-shadow: 2px 3px 4px #111; /* Soft dark shadow */
  color: orange; /* Make text color visible against dark shadow */
}

.multi-shadow {
  /* Multiple shadows are comma-separated */
  text-shadow:
    1px 1px 2px black, /* Base shadow */
    0 0 1em blue,     /* Blue outer glow */
    0 0 0.2em blue;   /* Sharper blue inner glow */
  color: white; /* White text for contrast */
}

text-shadow Explained

  • Purpose: Applies shadow effects directly to the text characters themselves.
  • Syntax: text-shadow: <offset-x> <offset-y> [blur-radius] [color];
    • <offset-x>: (Required) The horizontal distance of the shadow from the text. Positive values move it right, negative values move it left.
    • <offset-y>: (Required) The vertical distance of the shadow. Positive values move it down, negative values move it up.
    • [blur-radius]: (Optional) The blur distance. Larger values create softer, more spread-out shadows. 0 means a sharp, unblurred shadow. Defaults to 0 if omitted.
    • [color]: (Optional) The color of the shadow. Defaults to the element's color if omitted.
  • Multiple Shadows: You can apply multiple shadows to the same text by providing a comma-separated list of shadow definitions. Shadows are layered, with the first shadow specified appearing on top.
  • Use Cases: Creating depth, embossed/engraved effects, glows, outlines (by using multiple sharp shadows with small offsets). Often used on headings or decorative text. Overuse can harm readability.

In Our Example:

  • .simple-shadow has one shadow: offset 2px right, 3px down, with a 4px blur, and a dark gray color (#111).
  • .multi-shadow applies three comma-separated shadows to create a more complex effect resembling a neon glow: a basic black shadow for depth, and two blue shadows with different blur radii for the glow effect.
