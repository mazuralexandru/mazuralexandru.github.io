606. Graphics: SVG (<svg>) - Basic Shape Elements

Now let's draw something inside our SVG container using basic shape elements.

Common Shape Elements:

  • <rect>: Draws a rectangle.
    • Attributes: x (top-left corner x), y (top-left corner y), width, height, rx (x-radius for rounded corners), ry (y-radius for rounded corners).
  • <circle>: Draws a circle.
    • Attributes: cx (center x), cy (center y), r (radius).
  • <ellipse>: Draws an ellipse.
    • Attributes: cx, cy, rx (x-radius), ry (y-radius).
  • <line>: Draws a straight line segment.
    • Attributes: x1 (start x), y1 (start y), x2 (end x), y2 (end y).
  • <polyline>: Draws a series of connected lines (open shape).
    • Attribute: points="x1,y1 x2,y2 x3,y3 ..."
  • <polygon>: Draws a series of connected lines that automatically closes (forms a closed shape).
    • Attribute: points="x1,y1 x2,y2 x3,y3 ..."

Styling Attributes (or CSS):

  • These shapes are often styled using presentation attributes directly on the element or via CSS. Key ones:
    • fill: The color or pattern inside the shape (default is often black). Use fill="none" for no fill.
    • stroke: The color of the outline.
    • stroke-width: The thickness of the outline.

✨ Key Takeaway: Use basic shape elements like <rect>, <circle>, <ellipse>, <line>, <polyline>, and <polygon> inside <svg> to define geometric figures. Control their appearance using attributes like fill, stroke, and stroke-width either directly on the element (presentation attributes) or, preferably, via CSS.
