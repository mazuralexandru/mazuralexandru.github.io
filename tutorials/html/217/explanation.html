217. line-height Example

Files Modified: style.css

Let's adjust the line height for our highlighted paragraph.

✅ Observe style.css:

  • We've added line-height: 1.8; to the .highlight-text ruleset.
.highlight-text {
  background-color: #4d4d00; 
  color: #1a1a1a; /* Overridden by .container > p */
  padding: 5px; 
  line-height: 1.8; /* Increase line spacing */
}

✨ Observe the Preview:

  • Look closely at the first paragraph ("This is a standard paragraph..."). The vertical space between the lines of text within this paragraph should have increased compared to the previous step and compared to the second paragraph (which still uses the default line height).
  • Using the unitless value 1.8 means the line height will be 1.8 times the paragraph's own computed font size.

Adjusting line-height is crucial for text readability, especially for longer blocks of text.
