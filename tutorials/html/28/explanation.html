28. Closing Tags (</tagname>)

Files Modified: None

The closing tag signifies the end of an HTML element.

  • Syntax: It consists of a forward slash (/) followed by the element's name, enclosed in angle brackets (< >).
    • Example: </h1>, </p>, </div>
  • Purpose: Tells the browser, "The element that started with the corresponding opening tag ends here."
  • Requirement: Most HTML elements require a closing tag to define where their content ends. Forgetting a closing tag is a common source of errors and can lead to unexpected rendering by the browser.

There are exceptions called "void elements" (like <img> or <br>) that don't have content and thus don't need a closing tag, which we'll discuss separately.
