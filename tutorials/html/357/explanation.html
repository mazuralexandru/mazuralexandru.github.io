357. Styling Description List Details (<dd>)

Files Modified: None

We already added some basic styles for the description details (<dd>) in the previous step. Let's review them:

  • margin-left: Controls the indentation of the description relative to the term. Browsers usually apply a default margin here, but setting it explicitly ensures consistency. We used margin-left: 2em;.
  • margin-bottom: Controls the space after a description, before the next term (<dt>). This helps separate distinct term/description pairs. We used margin-bottom: 1em;.

You can apply any other text or box model styles to <dd> as needed, such as changing the color, font-style, or adding padding.
