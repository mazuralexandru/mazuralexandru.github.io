282. The style Attribute - Fundamentals

Now we examine the style attribute, which allows embedding CSS rules directly within an HTML element's tag.

Purpose:

  • The style attribute applies one or more CSS style declarations directly to the single element it resides on.

Core Concept:

  • It allows you to define presentation rules (like color, size, spacing) right alongside the HTML structure for that specific element instance. This is known as inline styling.

Analogy:

  • Instead of using a general style guide (external CSS file), you're writing specific formatting instructions directly on a single page or object.

Syntax:

  • The attribute is added to the opening tag: style="css-declarations"
  • The value of the style attribute is a string containing one or more standard CSS declarations.
  • CSS Declaration Syntax: Each declaration consists of a property: value pair, and multiple declarations are separated by semicolons (;).
    • Example: style="color: blue; font-weight: bold; margin-left: 20px;"

Key Difference from Stylesheets:

  • Unlike styles defined in external (<link>) or internal (<style>) stylesheets which use selectors (p, .my-class, #my-id) to target elements, inline styles apply only to the element where the style attribute is written.
