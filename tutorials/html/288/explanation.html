288. Longhand Border Properties: border-color

Files Modified: style.css

The border-color property sets the color of the border line(s).

  • Purpose: Defines the color of the border.
  • Syntax: border-color: value; (shorthand for all four sides) or border-top-color: value;, border-right-color: value;, etc.
  • Values: Any valid CSS color value (keywords, hex, rgb(a), hsl(a), currentColor, transparent).
  • Default: If omitted, the border color defaults to the value of the element's color property (its text color).
  • Shorthand Application: The border-color shorthand can accept 1, 2, 3, or 4 color values to set top, right, bottom, and left colors respectively.

✅ Observe style.css:

  • We've added border-color: orange; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin: 1em auto; 
  border-width: 3px; 
  border-style: dashed; 
  border-color: orange; /* Set border color */
}

✨ Observe the Preview: The dashed border around the blue box should now be orange, instead of the default text color it was using previously. We have now explicitly set the width, style, and color, completing the border definition using longhand properties.
