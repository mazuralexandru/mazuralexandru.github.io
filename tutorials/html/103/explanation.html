103. CSS Font Size: Pixel Values (px)

Files Modified: None

Pixel (px) values define an absolute font size. 1px is generally defined as 1/96th of an inch, but its actual appearance depends on the screen's resolution (PPI).

  • Nature: Fixed size. 16px text should appear roughly the same size regardless of the parent element's font size.
  • Use Cases:
    • Setting a base font size on the body or <html> element.
    • Situations where precise pixel control is needed and scalability is less of a concern.
    • Sometimes used for borders, padding, margins where fixed dimensions are desired.
  • Pros: Predictable size (mostly). Easy to understand initially.
  • Cons: Less flexible for users who want to adjust their default browser font size (text might not scale). Can make creating fully responsive/scalable layouts more challenging compared to relative units.

Example:

p { font-size: 14px; } 
h1 { font-size: 32px; }
