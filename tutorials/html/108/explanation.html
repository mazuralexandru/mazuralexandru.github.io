108. HTML Comments: Multi-line

Files Modified: None

HTML comments can easily span multiple lines. You simply start the comment with <!-- on the first line and end it with --> on the last line. Everything in between, across all lines, is part of the comment.

<!-- 
  This is a multi-line comment.
  It's useful for longer explanations or for commenting out
  larger blocks of HTML code during development.

  <section>
    <h2>Section Title</h2>
    <p>This whole section is currently commented out.</p>
  </section> 
-->

This is commonly used for:

  • Writing more detailed explanations.
  • Commenting out entire sections of a page temporarily.
