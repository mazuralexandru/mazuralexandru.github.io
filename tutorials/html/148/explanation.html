148. HTML class Attribute Revisited

Files Modified: None

Before using class selectors in CSS, let's quickly revisit the class attribute in HTML:

  • Purpose: The class attribute is a global attribute used to assign one or more class names to an HTML element. These names act as labels or categories for the element, primarily used as hooks for CSS styling and JavaScript manipulation.
  • Syntax: The attribute value is a space-separated list of class names.
    <p class="intro important featured">...</p>
<button class="button button-primary">Submit</button>
<div class="card user-profile">...</div>
  • Multiple Classes: An element can have multiple classes, allowing you to combine different styles (e.g., a base .button style and a modifier .button-primary style).
  • Reusability: The same class name can be applied to multiple elements on the page, even elements of different types (e.g., both a <div> and a <span> could have class="highlight").
  • Naming Conventions:
    • Use meaningful, semantic names that describe the purpose or role of the element/style (e.g., class="error-message", class="product-image") rather than describing the visual appearance (e.g., avoid class="red-text").
    • Use lowercase letters.
    • Separate words with hyphens (-) (this is the most common convention, known as kebab-case). Avoid spaces within a single class name.
    • Class names can contain letters, numbers, hyphens, and underscores, but cannot start with a digit, two hyphens, or a hyphen followed by a digit.

Assigning appropriate classes in your HTML is the foundation for effective CSS styling using class selectors.
