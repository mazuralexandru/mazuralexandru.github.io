207. Reset: Removing Link Content Examples

We've covered the importance of link content, especially descriptive link text, and how to handle image links using alt text.

Now, we'll move on to explore some other useful attributes that can be added to the <a> element besides href and target, such as rel, hreflang, and type.

To prepare, let's remove the image link example we created.

Why Remove Code?

  • Focus: Shift attention from the content of the link to additional attributes applied to the <a> tag itself.
  • Clean Slate: Provide a simple structure for adding links demonstrating these other attributes.
