33. Attribute Selectors: Adding Links for More Examples

Attribute selectors are particularly useful for styling links (<a>) based on their destinations (href) or how they open (target). Let's add a couple of links to our HTML to demonstrate this.

What's New?

  • HTML Pruning: Removed the input fields and the <br> tag to focus on links. Look for <!-- Pruned previous input examples -->.
  • HTML Addition: Added two anchor (<a>) elements:
    • One linking to an external site that opens in a new tab (target="_blank").
    • One linking to a non-existent PDF file (for demonstration purposes).
<!-- index.html snippet -->
<body>
  <h1>Attribute Selectors Demo</h1>
  <p>Styling links based on attributes.</p> <!-- Changed P text -->

  <!-- Pruned previous input examples -->

  <a href="https://example.com" target="_blank">External Link (New Tab)</a> <!-- Added link with target -->
  <br><br> <!-- Added spacing -->
  <a href="document.pdf">Link to PDF</a> <!-- Added link to PDF -->

  <!-- Commented out paragraph remains -->
  <!-- <p>This paragraph is commented out and won't appear.</p> -->

  <script src="script.js"></script>
</body>

Link Attributes

  • href: Specifies the URL the link points to.
  • target: Specifies where to open the linked document.
    • _blank: Opens in a new window or tab.
    • _self (default): Opens in the same frame.
    • Other values like _parent, _top, or a specific frame name exist but are less common.

We now have links with different target and href attributes, ready to be styled using attribute selectors.
