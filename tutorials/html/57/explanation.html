57. Structural Formatting: Horizontal Rules (<hr>)

Let's introduce the <hr> element, which represents a thematic break in the content.

The Horizontal Rule / Thematic Break Element (<hr>)

  • Purpose: Represents a paragraph-level thematic break. It signifies a shift in topic, a scene change in a story, or a separation between different subjects within a section, where a new heading might not be appropriate or necessary.
  • Semantics: Its meaning is about separation and transition in the flow of content.
  • Type: It's a void (empty) element (no content, no closing tag). It's also considered block-level, meaning it typically occupies its own line and separates other block-level elements.
  • Syntax (HTML5): <hr> (XHTML-style <hr /> is also valid).

Default Rendering:

Browsers typically render the <hr> element as a horizontal line spanning the width of its container, often with some vertical margin space around it. This visual line reinforces the idea of a separation.

Code Change:

We have added an <hr> element between the two paragraphs in index.html.

<body>
  <h1>Main Page Title (h1)</h1>
  <p>This is the introductory paragraph. We will explore thematic breaks next.</p>

  <hr> <!-- Added horizontal rule -->

  <p>This is another paragraph, following the thematic break.</p> <!-- Adjusted text -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Locate the new <hr> tag placed between the two <p> tags in index.html.
  2. Look at the browser preview. You should now see a horizontal line visually separating the first paragraph from the second paragraph. This is the default rendering of the semantic <hr> element.

✨ Key Takeaway: Use the <hr> element to signify a thematic break or topic shift between paragraph-level content blocks. It's a void, block-level element, typically rendered as a horizontal line by default.
