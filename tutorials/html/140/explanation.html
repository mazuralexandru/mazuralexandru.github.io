140. Optional type="text/css" Attribute

Files Modified: None

You might occasionally see the type attribute used on a <link> tag for stylesheets:

<link rel="stylesheet" href="styles.css" type="text/css"> <!-- type attribute -->
  • Purpose (Historical): The type attribute was used to specify the MIME type (media type) of the linked resource. For CSS files, the correct MIME type is text/css.
  • HTML5 Status: In HTML5, if the rel attribute is set to stylesheet, the type attribute is no longer required. Browsers assume the type is text/css by default when rel="stylesheet" is present.
  • Recommendation: You can safely omit the type="text/css" attribute for CSS links in modern HTML5 documents. Including it doesn't hurt, but it adds unnecessary verbosity.
<link rel="stylesheet" href="styles.css"> <!-- Preferred HTML5 syntax -->
