348. Controlling List Indentation: Default Mechanism

Files Modified: None

How does the default list indentation actually work?

While it can vary slightly between browsers, the common mechanism is that user-agent (browser) stylesheets apply a default padding-left (or sometimes margin-left, though padding is more typical for indentation that aligns with the marker) to the <ul> and <ol> elements themselves.

  • Typical Default: This default padding is often around 40px or a relative value like 2em or 2.5em.
  • Purpose: This padding creates the space needed to display the list marker (list-style-position: outside;) without overlapping the text content of the list items.
  • Nested Lists: Nested lists typically inherit or have their own default padding applied, leading to the cumulative indentation effect.

This is why setting padding-left: 0; on ul and ol (as we did in our reset and then for the <ol>) effectively removes the default indentation.
