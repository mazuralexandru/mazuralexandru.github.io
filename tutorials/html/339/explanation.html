339. list-style-type Values for <ul>

Files Modified: style.css

Let's restore some markers specifically for the unordered lists (<ul>).

Common values for list-style-type suitable for unordered lists include:

  • disc: A filled circle (often the default for top-level <ul>).
  • circle: An unfilled circle.
  • square: A filled square.
  • none: No marker is displayed.

✅ Observe style.css:

  • We've added a new rule specifically targeting ul (overriding the previous reset for ul specifically).
  • We're setting list-style-type: square;.
  • We're also adding back some padding-left to provide indentation for the markers.
/* Reset default list styles */
/* ul, ol {
  list-style: none;
  padding-left: 0;
} */ /* Commenting out the general reset for now */

/* Style Unordered Lists */
ul {
  list-style-type: square; /* Use squares for bullets */
  padding-left: 2em; /* Add indentation back */
  margin-bottom: 1em; /* Add space below list */
}

/* Keep ordered lists reset for now */
ol {
   list-style: none;
   padding-left: 0;
   margin-bottom: 1em; /* Add space below list */
}

(Note: We commented out the combined ul, ol reset and added separate rules to apply styles differently).
