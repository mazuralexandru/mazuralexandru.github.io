42. Component 1: <!DOCTYPE html>

Files Modified: None

The very first thing in any HTML document must be the DOCTYPE declaration.

  • Purpose: Its primary job is to tell the browser which version of HTML the document is written in. For modern web development, we use the HTML5 DOCTYPE. It also ensures the browser uses its "standards mode" for rendering, rather than falling back to older, inconsistent "quirks mode".
  • Syntax: <!DOCTYPE html>
    • It's case-insensitive, but uppercase DOCTYPE is the common convention.
    • There are no spaces before it. It must be the absolute first line.
  • Placement: Must appear only once, at the very beginning of the file.

This specific, simple DOCTYPE signals HTML5. Older HTML versions had much more complex DOCTYPE declarations.
