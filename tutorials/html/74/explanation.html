74. <meta charset="UTF-8">: Placement

Files Modified: None

The placement of the <meta charset="UTF-8"> tag is critical.

  • Rule: It should be placed as early as possible within the <head> element.
  • Reason: The browser needs to know the character encoding before it starts parsing content that might contain special characters, including potentially the content of the <title> element itself. If the declaration comes too late, the browser might have already misinterpreted earlier parts of the document.
  • Requirement: Technically, the declaration must appear within the first 1024 bytes of the HTML file for guaranteed detection by the browser's pre-scan algorithms. Placing it immediately after the opening <head> tag ensures this.
<head>
  <meta charset="UTF-8">  <!-- CORRECT: As early as possible -->
  <title>Page Title</title>
  <!-- Other head elements -->
</head>
