104. CSS Font Size: Keywords

Files Modified: None

CSS provides several keywords that can be used as values for font-size.

  1. Absolute Size Keywords: These map to predefined font sizes determined by the browser. The exact pixel size can vary slightly between browsers, but medium is typically the user's default font size (often 16px).
    • xx-small
    • x-small
    • small
    • medium
    • large
    • x-large
    • xx-large
  2. Relative Size Keywords: These make the font size larger or smaller relative to the parent element's font size.
    • larger
    • smaller

Pros: Simple to use. Respect user's default font size settings to some extent. Cons: Less precise control than length units. The exact rendering can vary between browsers. Not commonly used in modern development compared to rem or em units for scalability.

Example:

h1 { font-size: x-large; }
small { font-size: smaller; } /* Makes text smaller than its parent */

While available, you'll more often see px, em, or rem used for font sizes in practice.
