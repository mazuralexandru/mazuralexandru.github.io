400. Controlling Content within Dimensions (object-fit)

Files Modified: index.html, style.css

What if you need an image to fit into specific dimensions (e.g., a fixed-size square container) that don't match its original aspect ratio? The object-fit property controls how the image content is resized to fit its container.

This property applies to replaced elements like <img> and <video>.

  • Common Values:
    • fill (Default): Stretches/squashes the image to exactly match the container's dimensions, potentially distorting the aspect ratio.
    • contain: Scales the image (maintaining aspect ratio) to fit inside the container. May leave empty space (letterboxing).
    • cover: Scales the image (maintaining aspect ratio) to cover the entire container. May crop parts of the image that extend beyond the container.
    • none: Displays the image at its intrinsic size, potentially clipping it if it's larger than the container.
    • scale-down: Selects the smaller of none or contain.

✅ Observe style.css:

  • We've modified the .sample-image rule, giving it a fixed square width and height.
  • We've added object-fit: cover;.
.sample-image {
  max-width: 100%; 
  /* height: auto; <- Removed */
  width: 200px; /* Set fixed width */
  height: 200px; /* Set fixed height (different aspect ratio) */
  display: block; 
  border: 2px solid yellow; 
  margin: 1em auto; 
  object-fit: cover; /* Scale to cover, maintain aspect ratio, crop if needed */
}

✨ Observe the Preview:

  • The image container (yellow border) is now a 200x200px square.
  • Our original image aspect ratio was 300x200 (3:2). Since the container is square (1:1) and we used object-fit: cover;, the image content (represented by the broken icon) should be scaled proportionally to fill the entire square, meaning parts of the image (likely the left and right edges) would be cropped off to fit the square container while maintaining the image's aspect ratio.
  • Try changing cover to contain in DevTools. The image should shrink to fit entirely within the square, leaving empty space above and below.
  • Try changing it to fill. The image should stretch vertically to fill the square, distorting its appearance.
