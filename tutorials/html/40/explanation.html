40. CSS Combinators: Descendant Combinator (Space)

Selectors can be combined to define relationships between elements in the HTML structure. These combinations are called Combinators. The most common is the Descendant Combinator, represented by a space.

Descendant Combinator ( - Space)

  • Syntax: Two or more selectors separated by a space ( ). Example: ancestor descendant.
  • Meaning: Selects all elements matching the descendant selector that are located anywhere inside an element matching the ancestor selector in the HTML structure. The descendant doesn't have to be a direct child; it can be nested many levels deep.
  • Example: article p selects any <p> element that exists somewhere within an <article> element.

In our code:

  • The rule article p { ... } targets paragraph elements within the <article>.
  • It applies a lightgreen color, a left border, padding, and margin to these paragraphs.
