423. Styling <figcaption>: Typography

Files Modified: style.css

Let's adjust the typography of the caption. By default, it might inherit the body font, and browsers sometimes render it in italics.

✅ Observe style.css:

  • We've added a rule for figcaption.
  • We're setting the font-size to be slightly smaller, explicitly setting font-style to italic (though it might be the default anyway), and changing the color.
/* Styling Figcaption */
figcaption {
    font-size: 0.9rem; /* Slightly smaller text */
    font-style: italic; /* Ensure italics */
    color: #bbb; /* Lighter gray color */
    /* Alignment and spacing next */
}

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview: The caption text ("This is the caption...") below the image should now be slightly smaller, italicized, and a lighter gray color compared to the default body text.
