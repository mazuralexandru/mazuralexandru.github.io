221. The Image Element (<img>) - Fundamentals

The core element for embedding images in HTML is <img>.

Definition:

  • The <img> element is used to embed an image into an HTML document.

Purpose & Role:

  • Its primary role is to display graphical content, such as photographs, icons, diagrams, illustrations, logos, etc., directly within the flow of the web page content.
  • Unlike background images added via CSS (which are purely decorative), <img> elements represent content images that are part of the document's information structure.

Core Concept:

  • The <img> tag acts as a placeholder, reserving space on the page where an external image file, specified by the mandatory src attribute (which we'll cover next), will be loaded and rendered by the browser.

✅ Observation: No code changes in this step. We are defining the fundamental purpose of the <img> element before adding it to the page.

✨ Key Takeaway: The <img> element is the standard way to embed content images into an HTML document. It requires a src attribute to specify the image file.
