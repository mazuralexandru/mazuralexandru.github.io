283. The style Attribute - Core Mechanics (Syntax & Example)

Let's apply an inline style using the style attribute.

Syntax Recap:

  • style="property: value; property2: value2;"
  • The value string contains standard CSS property-value pairs separated by semicolons.

Code Change:

We've added a style attribute to the first paragraph in index.html to change its text color and add a border.

<!-- index.html -->
<body>
  <h1>Global Attributes - Style</h1>
  <p style="color: #FF6347; border: 1px solid #FF6347; padding: 5px;">Now let's look at the 'style' attribute for inline CSS.</p> <!-- Added style attribute -->

  <p>Inline styles apply directly to a single element.</p>

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>
  • We've set the color property to #FF6347 (Tomato red).
  • We've set the border property to 1px solid #FF6347.
  • We've added some padding for spacing inside the border.

✅ Observation:

  1. Locate the style="..." attribute added to the first <p> tag in index.html. Examine the CSS declarations within its value string.
  2. Look at the browser preview. You should see:
    • The text of the first paragraph is now Tomato red.
    • The first paragraph has a thin, solid Tomato red border around it, with a small amount of padding between the text and the border.
    • The second paragraph remains completely unchanged, still using the default styles inherited from the body rule in style.css.
  3. This clearly demonstrates that the inline style attribute affects only the element it is placed on.

✨ Key Takeaway: Inline styles using the style attribute allow you to apply specific CSS rules directly to an individual element, overriding other styles for that element.
