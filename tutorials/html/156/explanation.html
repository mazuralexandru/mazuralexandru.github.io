156. ID Selectors: Scope

Files Modified: None

The scope of an ID selector is inherently limited by the uniqueness requirement of the HTML id attribute.

  • Unique Target: An ID selector like #main-header is intended to match at most one element on any given page â€“ the single element that has id="main-header".
  • High Specificity: ID selectors have a higher specificity than class selectors or type selectors. This means styles applied via an ID selector are "stronger" and will override conflicting styles from class or type selectors targeting the same element. (More on specificity later).

Because an ID must be unique, ID selectors provide a way to precisely target a single, specific element for styling or JavaScript interaction.
