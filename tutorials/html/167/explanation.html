167. Universal Selector: Scope

Files Modified: None

The scope of the universal selector (*) is all-encompassing. It matches every single element within the scope it's applied (usually the entire document if used at the top level of the CSS).

This includes:

  • The root <html> element.
  • Elements within the <head> (like <meta>, <title>, <link>, <style>, <script>).
  • The <body> element.
  • All elements within the <body> (<h1>, <p>, <div>, <span>, <img>, etc.).
  • Pseudo-elements (like ::before, ::after) are generally not matched by the universal selector alone, though *::before would match the ::before pseudo-element of every element.

Because it affects everything, applying styles directly with * can have significant performance implications (the browser has to apply the style to every single node) and unintended consequences if not used carefully.
