289. Border Shorthand Properties: border

Files Modified: style.css

Similar to padding and margin, CSS provides shorthand properties for borders to make definitions more concise. The most common is the border property.

  • Purpose: Sets the border-width, border-style, and border-color for all four sides of an element simultaneously.
  • Syntax: border: width style color;
    • The order of the three values (width, style, color) doesn't strictly matter, but this is the conventional order.
    • The border-style value is required for the border to be visible.
    • If width is omitted, it defaults to medium.
    • If color is omitted, it defaults to the element's color (text color).
  • Example: border: 1px solid #ccc; (A common way to add a light gray solid border).

✅ Observe style.css:

  • We've replaced the three longhand border-* properties with a single border shorthand declaration to achieve the same result.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin: 1em auto; 
  /* Replaced longhand with shorthand */
  border: 3px dashed orange; 
}

✨ Observe the Preview: The appearance of the box's border should be identical to the previous step (3px dashed orange). The border shorthand provides a much more compact way to define simple borders that are the same on all sides.
