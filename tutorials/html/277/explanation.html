277. Longhand Padding Properties: padding-top

Files Modified: style.css

You can control padding on each side individually using the longhand properties. Let's start with padding-top.

  • Purpose: Sets the amount of padding space specifically on the top edge of the element, between the top of the content and the top border.
  • Syntax: padding-top: value;
  • Values: Accepts non-negative lengths (px, em, rem, etc.) or percentages (%). Percentages are calculated relative to the width of the containing block.

✅ Observe style.css:

  • We've removed the padding: 10px; shorthand.
  • We've added padding-top: 20px;.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  /* padding: 10px; <- Removed shorthand */
  padding-top: 20px; /* Added specific top padding */
  margin: 1em auto; 
}

✨ Observe the Preview: The blue box should now have more space (20px) between the top edge of its background and the start of the text content compared to the other sides (which currently have no explicit padding). Inspecting with DevTools will confirm the 20px top padding in the box model diagram.
