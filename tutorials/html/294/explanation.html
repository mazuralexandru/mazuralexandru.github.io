294. Rounded Corners (border-radius): Elliptical Corners

Files Modified: None

The border-radius shorthand (and the longhand properties) can also accept two values separated by a forward slash (/) to create elliptical corners instead of circular ones.

  • Syntax: border-radius: horizontal-radius / vertical-radius;

    • The value(s) before the slash define the horizontal radius (or radii) of the ellipse.
    • The value(s) after the slash define the vertical radius (or radii) of the ellipse.
    • You can use the 1-to-4 value syntax independently for both horizontal and vertical radii.

  • Examples:

    • border-radius: 10px / 20px; (All corners have 10px horizontal radius, 20px vertical radius)
    • border-top-left-radius: 50px 20px; (Alternative longhand syntax for a single corner: 50px horizontal, 20px vertical)
    • border-radius: 10px 30px / 20px 40px; (Top-left/Bottom-right: 10px horiz / 20px vert. Top-right/Bottom-left: 30px horiz / 40px vert.)

This allows for more complex corner shapes beyond simple circles.
