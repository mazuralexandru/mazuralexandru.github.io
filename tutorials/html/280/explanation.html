280. Longhand Padding Properties: padding-left

Files Modified: style.css

Finally, padding-left.

  • Purpose: Sets the amount of padding space specifically on the left edge of the element, between the left edge of the content and the left border.
  • Syntax: padding-left: value;
  • Values: Same as padding-top (non-negative lengths or percentages relative to container width).

✅ Observe style.css:

  • We've added padding-left: 30px; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding-top: 20px; 
  padding-right: 5px; 
  padding-bottom: 15px; 
  padding-left: 30px; /* Added specific left padding */
  margin: 1em auto; 
}

✨ Observe the Preview: The blue box should now have 30px of space between the left edge of the text content and the left edge of the blue background. We now have different padding values applied to all four sides using the longhand properties.
