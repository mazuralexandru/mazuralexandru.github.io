631. Escaping Reserved Characters: Less Than (&lt;)

Let's start applying entities by escaping the essential reserved characters, beginning with the less-than sign (<).

The Problem:

  • The < character signifies the beginning of an HTML tag (e.g., <p>, <div>). If you type < directly into your HTML content where you intend to display the literal character, the browser parser will expect a tag name to follow. If it doesn't find a valid tag name, it can lead to parsing errors, unexpected rendering, or the content being hidden.

The Solution: &lt;

  • Use the named entity &lt; (stands for "less than") to represent the literal < character.
  • The browser will parse &lt; and render it visually as <.

Use Cases:

  • Displaying mathematical comparisons (e.g., 5 < 10).
  • Showing code examples containing HTML tags (e.g., explaining <p> requires writing &lt;p&gt;).

Code Change:

We've added paragraphs to index.html demonstrating the use of &lt;. Note that in the explanation paragraph, we use &amp;lt; to display the literal entity code itself.

<!-- index.html snippet -->
<body>
  <h1>HTML Entities: Escaping Reserved Characters</h1> <!-- Updated H1 -->
  <p>Reason 1: Escaping characters with special meaning in HTML.</p>

  <h2>Displaying Less Than (&lt;)</h2>
  <p>To show the less than sign, use &amp;lt; like this: 5 &lt; 10.</p> <!-- Used &lt; -->
  <p>Without escaping, &lt;10 would look like a broken tag.</p> <!-- Shows escaping in explanation -->

  <!-- Example of what NOT to do (would likely render poorly or be invisible) -->
  <!-- <p>Incorrect: 5 < 10 </p> -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Locate the first content paragraph under the <h2>. Find 5 &lt; 10.
  2. Examine the second content paragraph explaining the entity. Note how &amp;lt; is used to display the entity code literally.
  3. Look at the browser preview. The first paragraph should render visually as: 5 < 10. The browser correctly replaced &lt; with the < symbol.

✨ Key Takeaway: Always use the entity &lt; to display a literal less-than sign (<) in your HTML content to prevent it from being misinterpreted as the start of a tag.
