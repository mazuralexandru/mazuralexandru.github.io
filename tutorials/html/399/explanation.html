399. Spanning Rows: rowspan Example

Let's apply rowspan to create a row header that spans multiple data rows.

Scenario: Imagine our table now groups items by type, and we want the "Type" header (like "Alpha") to apply to multiple rows of associated names/values.

Code Change:

We've modified the <tbody> in index.html:

  1. Added rowspan="2" to the first row header (<th scope="row">Alpha</th>).
  2. Added data cells for the first row (A1, A2).
  3. The second row now omits the first <th> cell because the "Alpha" cell from the row above spans down into this row's first column slot. It starts directly with the <td> for the "Name" column (B1).
  4. Added a third row to show a different type ("Beta").
<!-- index.html snippet -->
<body>
  <h1>Spanning Columns & Rows</h1>
  <p>Using 'rowspan' on a header cell (th):</p> <!-- Adjusted P -->

  <table>
    <thead>
      <tr>
        <th scope="col">Type</th>
        <th scope="col">Name</th>
        <th scope="col">Value</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <th scope="row" rowspan="2">Alpha</th> <!-- Row Header spanning 2 rows -->
        <td>A1</td>
        <td>A2</td>
      </tr>
      <tr>
        <!-- No th needed here for Type column due to rowspan -->
        <td>B1</td> <!-- This cell is in the 'Name' column -->
        <td>B2</td> <!-- This cell is in the 'Value' column -->
      </tr>
      <tr> <!-- Third Row -->
        <th scope="row">Beta</th>
        <td>C1</td>
        <td>C2</td>
      </tr>
    </tbody>
  </table>

  <p>The "Alpha" header spans two rows vertically.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Examine the <tbody> structure carefully in index.html. Note the rowspan="2" on the first <th>. Critically, observe that the second <tr> only contains two <td> elements, lacking the initial <th>.
  2. Look at the browser preview:
    • The cell containing "Alpha" visually extends downwards, covering the vertical space next to both the "A1/A2" row and the "B1/B2" row.
    • The "B1" and "B2" cells correctly align under the "Name" and "Value" column headers, respectively.
    • The "Beta" row appears below the spanned "Alpha" section.
    • The rowspan attribute created the desired merged vertical cell.

✨ Key Takeaway: Apply rowspan="number" to a <th> or <td> to make it span number rows vertically. Ensure you omit the corresponding cells from the subsequent number - 1 rows in the HTML source code to accommodate the span.
