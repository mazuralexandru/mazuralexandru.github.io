84. CSS Selectors

Files Modified: None

The selector is the part of the CSS ruleset that determines which HTML element(s) the styles inside the declaration block will apply to.

There are many types of selectors, allowing for precise targeting. We'll start with the simplest:

  • Type Selector (or Element Selector): Selects all elements of a specific type (tag name).
    • h1 selects all <h1> elements.
    • p selects all <p> elements.
    • div selects all <div> elements.
    • body selects the <body> element (as seen in base_styles.css).

Example:

p { 
  color: green; /* This will make ALL paragraphs green */
}

h2 {
  text-decoration: underline; /* This will underline ALL level 2 headings */
}

We will explore more complex selectors (like class selectors, ID selectors, attribute selectors, pseudo-classes) later, as they are essential for more targeted styling. For now, understanding the type selector is enough.
