192. Email Links (mailto:): Adding Subject

You can pre-fill other fields in the email message besides the "To:" address by adding parameters to the mailto: link, using a syntax similar to URL query strings.

Adding Parameters:

  • Start parameters after the email address with a question mark (?).
  • Use key=value pairs.
  • Separate multiple parameters with an ampersand (&) (we'll see this later).

The subject Parameter:

  • Key: subject
  • Purpose: Pre-fills the email's subject line.
  • Syntax: mailto:email@address.com?subject=Your%20Subject%20Here

URL Encoding (Percent Encoding):

  • Problem: Certain characters have special meanings in URLs (like spaces, ?, &, /, %, #) or are not allowed.
  • Solution: These characters must be URL-encoded (also called percent-encoding). This involves replacing the special character with a % sign followed by its two-digit hexadecimal ASCII value.
  • Common Encodings:
    • Space becomes %20
    • ! becomes %21
    • ? becomes %3F
    • & becomes %26
    • Line Break (CRLF) becomes %0D%0A (important for the body parameter later)
  • Requirement: Parameter values (like the subject text) must be properly URL-encoded to work reliably. Many online tools or programming functions can perform URL encoding. For short examples, we'll encode spaces as %20.

Code Change:

We've modified the mailto: link in index.html to include a subject parameter. Note the %20 used to encode the space in "Website Inquiry".

<p>
  <!-- Note: %20 represents a space -->
  <a href="mailto:contact@example.com?subject=Website%20Inquiry"> <!-- Added ?subject=... -->
    Email Us (With Subject)
  </a>
</p>

✅ Observation:

  1. Examine the href value in index.html. Note the ?subject=Website%20Inquiry part.
  2. Click the "Email Us (With Subject)" link in the preview.
  3. If your email client opens, you should see the "To:" field filled with "contact@example.com" AND the Subject field pre-filled with "Website Inquiry".

✨ Key Takeaway: Add parameters like ?subject=value after the email address in a mailto: link to pre-fill fields. Remember to URL-encode special characters (especially spaces as %20) in the parameter values.
