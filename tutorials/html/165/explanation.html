165. The Universal Selector

Files Modified: None

The Universal Selector is a special selector that matches any single element.

  • Syntax: An asterisk (*).
* {
  /* These styles apply to EVERY element on the page */
  margin: 0;
  padding: 0;
}

While powerful, it needs to be used with caution due to its extremely broad scope.
