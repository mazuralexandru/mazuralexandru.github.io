354. Other <head> Elements: The <style> Element - Core Mechanics

Let's add some embedded styles using the <style> element and observe their effect.

Syntax Recap:

  • <style> CSS rules... </style>
  • Place within <head>.
  • Content uses standard CSS syntax (selectors, properties, values).

Attributes:

  • type="text/css": Similar to <link>, this attribute was historically used but is no longer necessary for <style> tags in HTML5. Browsers assume the content is CSS. It can be safely omitted.
  • media: Just like with <link>, you can use the media attribute on a <style> tag to make the embedded rules apply only under specific conditions (e.g., media="print", media="(max-width: 600px)").

Code Change:

We've added a <style> block to the <head> of index.html containing a CSS rule to style the <h1> element.

<!-- index.html -->
<head>
  <meta charset="UTF-8">
  <title>HTML Metadata - Style Element</title>
  <link rel="stylesheet" href="style.css"> <!-- External CSS -->

  <style>
    /* Internal Styles */
    h1 {
      color: orange; /* Style for H1 in this document only */
      text-decoration: underline wavy red;
    }
  </style> <!-- Added style block -->

</head>
<body>
  <h1>Metadata: The &lt;style&gt; Element</h1> <!-- This H1 will be styled -->
  <p>Added a style block to the head to style the H1.</p> <!-- Adjusted P -->

  <p>Internal styles apply only to the current document.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>
</html>

✅ Observation:

  1. Locate the new <style> block within the <head> in index.html. Examine the CSS rule inside it (h1 { ... }).
  2. Look at the browser preview. The <h1> ("Metadata: The <style> Element") should now be styled:
    • Its text color should be orange.
    • It should have a wavy red underline.
  3. The paragraphs remain styled only by the base body rules from the external style.css file. The h1 styles only apply because they were defined within the <style> block in this HTML document.

✨ Key Takeaway: CSS rules placed inside a <style> block in the <head> are applied to elements within that specific HTML document, allowing for page-specific embedded styles. The type attribute is optional.
