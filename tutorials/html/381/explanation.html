381. Header Cells: Combining <th> and <td>

Now let's build a complete simple table by adding data rows (<tr> with <td> cells) below the header row (<tr> with <th> cells).

Structure:

  • The first <tr> (or rows within <thead>) typically contains the <th> elements defining column headers.
  • Subsequent <tr> elements (usually within <tbody>) contain the corresponding data using <td> elements.
  • The number of <td> cells in a data row should match the number of <th> cells in the header row (assuming no colspan/rowspan).

Code Change:

We've added two data rows (<tr> containing <td>s) below the header row in index.html.

<!-- index.html snippet -->
  <table>
    <tr> <!-- Header Row -->
      <th>Column Header 1</th>
      <th>Column Header 2</th>
    </tr>
    <tr> <!-- Data Row 1 -->
      <td>Data A1</td> <!-- Added td -->
      <td>Data B1</td> <!-- Added td -->
    </tr>
    <tr> <!-- Data Row 2 -->
      <td>Data A2</td> <!-- Added td -->
      <td>Data B2</td> <!-- Added td -->
    </tr>
  </table>

✅ Observation:

  1. Examine the complete table structure in index.html. Note the header row with <th> followed by data rows with <td>.
  2. Look at the browser preview. You should now see a complete 2x2 data grid plus a header row:
    • The top row shows "Column Header 1" and "Column Header 2" (likely bold and centered).
    • The second row shows "Data A1" and "Data B1" (likely normal weight, left-aligned).
    • The third row shows "Data A2" and "Data B2" (styled like the second row).
    • The structure clearly presents headers followed by corresponding data.

✨ Key Takeaway: Combine header rows (using <th>) and data rows (using <td>) within a <table> to create a standard data table structure. Ensure the number of cells aligns between header and data rows.
