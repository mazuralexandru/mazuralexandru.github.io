211. font-variant

Files Modified: None

The font-variant property is a shorthand for various font variant properties, most commonly used to enable small caps.

  • Purpose: Allows access to alternative glyphs within a font, such as small capitals, ligatures, or different numeral styles.
  • Common Value: small-caps
    • Displays lowercase letters as smaller versions of uppercase letters.
    • Requires the font to actually contain small-caps glyphs for true typographic quality; otherwise, the browser might simulate them (often poorly).
  • Other font-variant-* Properties (More Advanced):
    • font-variant-caps: Controls alternative capitalizations (small-caps, all-small-caps, petite-caps, titling-caps, etc.).
    • font-variant-numeric: Controls numeral styles (lining-nums, oldstyle-nums, proportional-nums, tabular-nums, slashed-zero, etc.).
    • font-variant-ligatures: Controls common and discretionary ligatures (common-ligatures, no-common-ligatures, discretionary-ligatures, etc.).
    • font-variant-east-asian: Controls variants for East Asian scripts.
  • Font Support: The effectiveness of these properties is highly dependent on the features supported by the chosen font-family. Many standard web fonts have limited variant support beyond basic ligatures. OpenType fonts often have richer feature sets.

Example:

h1 {
  font-variant: small-caps; /* Display H1 in small caps */
}

.price {
  font-variant-numeric: oldstyle-nums; /* Use old-style figures if available */
}

font-variant is used less frequently than basic style/weight properties but offers fine-grained typographic control when needed and supported by the font.
