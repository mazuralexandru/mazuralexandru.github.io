409. Decorative Styling: padding

Files Modified: style.css

As we know from the box model, padding adds space between the content area and the border. While less common for images than for text containers, you might use it if you want the image's border to be spaced away from the image content itself.

  • Effect: Creates space inside the border. The background color/image extends into the padding.
  • box-sizing: Remember that with the default content-box, padding adds to the total size. With border-box (which we set globally), padding takes space from the content area.

✅ Observe style.css:

  • We've added padding: 10px; to the .block-image rule.
  • We've also added a background-color to the image element itself so we can see the padding area distinctly from the page background.
.block-image {
    display: block; 
    max-width: 100%; 
    height: auto; 
    width: 300px; 
    margin: 2em auto 3em auto; 
    border: 4px solid steelblue; 
    padding: 10px; /* Add padding inside the border */
    background-color: #444; /* Add background to see padding area */
}

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview:

  • There should now be a 10px gap between the image content area (where the broken icon is) and the steel-blue border.
  • This gap should have the dark gray (#444) background color we applied.
  • Because we're using box-sizing: border-box;, the overall rendered width of the element (including padding and border) should still be dictated by the width: 300px; rule. The content area inside has shrunk to accommodate the padding and border.
