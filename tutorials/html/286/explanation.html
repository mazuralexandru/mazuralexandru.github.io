286. Longhand Border Properties: border-width

Files Modified: style.css

The border-width property sets the thickness of the border.

  • Purpose: Controls how thick the border line(s) are.
  • Syntax: border-width: value; (shorthand for all four sides) or border-top-width: value;, border-right-width: value;, etc.
  • Values:
    • Keywords: thin, medium (default), thick. The exact pixel size of these keywords can vary slightly between browsers.
    • Length Units: px, em, rem, etc. (e.g., 2px). 0 effectively removes the border width.
  • Shorthand Application: Like padding, the border-width shorthand can accept 1, 2, 3, or 4 values to set top, right, bottom, and left widths respectively.

Note: Setting only border-width will not make a border visible. You also need to set border-style.

✅ Observe style.css:

  • We've added border-width: 3px; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin: 1em auto; 
  border-width: 3px; /* Set border thickness */
  /* Still need border-style and border-color for visibility */
}

✨ Observe the Preview: There is no visible change yet, because we haven't specified a border-style. We've defined the thickness, but not the type of line to draw.
