424. Styling <figcaption>: Alignment

Files Modified: style.css

We can control the horizontal alignment of the caption text using text-align.

✅ Observe style.css:

  • We've added text-align: center; to the figcaption rule.
figcaption {
    font-size: 0.9rem; 
    font-style: italic; 
    color: #bbb; 
    text-align: center; /* Center the caption text */
    /* Spacing next */
}
