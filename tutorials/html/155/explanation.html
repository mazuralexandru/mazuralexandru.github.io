155. CSS ID Selector Syntax (#idname)

Files Modified: None

To target a specific element based on its unique ID in CSS, you use an ID selector.

  • Syntax: A hash symbol (#) followed immediately by the ID name (the exact, case-sensitive value used in the HTML id attribute, without the quotes).

  • Examples:

    • To select the element with id="main-navigation":
      #main-navigation {
  /* styles for the main navigation */
}
    • To select the element with id="error-summary":
      #error-summary {
  /* styles for the error summary box */
}
    • To select the element with id="page-footer":
      #page-footer {
  /* styles for the page footer */
}

  • Case Sensitivity: ID selectors are case-sensitive, matching the case used in the HTML id attribute value. #mainNav will not match id="mainnav". Stick to consistent casing (usually lowercase).
