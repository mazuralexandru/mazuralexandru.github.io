315. Essential <meta>: Page Description (name="description") - Core Mechanics

Let's add the <meta name="description"> tag to our page and discuss best practices for its content.

Syntax:

  • Place the tag within the <head> section. <meta name="description" content="Your brief but accurate summary...">

content Value Best Practices:

Writing effective meta descriptions is important for attracting clicks from search results:

  1. Relevance & Accuracy: The description must accurately reflect the main content and topic of the specific page it's on. Don't mislead users.
  2. Conciseness (Length): Search engines typically display only the first ~150-160 characters of the meta description in their results snippets. While there's no hard penalty for longer descriptions, ensure the most important information and keywords appear early. Aim for roughly two short sentences.
  3. Uniqueness: Every page on your website that you want indexed should have a unique meta description tailored to its specific content. Avoid using the same description across multiple pages.
  4. Compelling Language: Think of it as advertising copy for your page. Use clear, engaging language that highlights the page's value proposition and encourages users to click. Include a call to action if appropriate (e.g., "Learn more...", "Find out how...").
  5. Keyword Inclusion (Natural): Naturally incorporate 1-2 primary keywords that the page targets, as search engines often bold matching query terms within the description snippet, increasing visibility. Avoid "keyword stuffing" (unnaturally repeating keywords).
  6. Readability: Write clear, grammatically correct sentences.

Code Change:

We've added a <meta name="description"> tag to the <head> section of index.html.

<!-- index.html -->
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Learn about HTML metadata elements like meta, link, title, base, style, and script found in the document head section. Essential for SEO and page setup."> <!-- Added description meta tag -->
  <title>HTML Metadata</title>
  <link rel="stylesheet" href="style.css">
  <!-- Other meta tags will be added here -->
</head>
<body>
  <h1>Metadata: Page Description</h1>
  <p>Added a meta description tag to the head.</p> <!-- Adjusted P -->

  <p>Check the &lt;head&gt; section in the HTML code.</p> <!-- Adjusted P -->
