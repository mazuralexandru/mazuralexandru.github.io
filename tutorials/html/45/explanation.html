45. DOCTYPE: Placement

Files Modified: None

The placement rule for <!DOCTYPE html> is strict and simple: It must be the very first thing in the HTML file.

<!DOCTYPE html>  <!-- CORRECT: First line -->
<html>
  ...
</html>

<!-- INCORRECT: Comment before DOCTYPE -->
<!DOCTYPE html> 
<html>
  ...
</html>

 
<!DOCTYPE html>  <!-- INCORRECT: Whitespace before DOCTYPE -->
<html>
  ...
</html>

Any characters or whitespace before the DOCTYPE will invalidate it, potentially causing the browser to fall back into Quirks Mode. Always ensure it's right at the top, starting at column 1 of line 1.
