232. <i>, <b>, <u>: HTML Recap

Files Modified: None

Let's recap the intended HTML5 semantics:

  • <i>: Alternate voice/mood, technical term, foreign phrase, thought, ship name. (Default: Italic)
    • Example: The term <i>Homo sapiens</i> refers to modern humans.
  • <b>: Keywords, product names, attention-grabbing text without added importance. (Default: Bold)
    • Example: Click the <b>Save</b> button to continue.
  • <u>: Non-textual annotation (e.g., marking misspelling). (Default: Underline) - Avoid general use.
    • Example (rare): This word is <u>mispelled</u>. (Though CSS might be better).

Remember the difference between these and the semantic emphasis/importance tags:

  • Use <em> for stress emphasis.
  • Use <strong> for strong importance.
  • Use <i> or <b> only if their specific HTML5 semantic meaning fits, or potentially (though less recommended) as purely presentational hooks if CSS classes aren't feasible.
