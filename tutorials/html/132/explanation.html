132. Internal Stylesheets: Example

Files Modified: index.html, style.css

Let's demonstrate an internal stylesheet. We'll move the rules currently in style.css into a <style> block within index.html.

‚úÖ Observe index.html!

  • A <style> block has been added inside the <head>.
  • The CSS rules for h1 and p (including the commented-out ones) are now inside this <style> block.
    <style>
      /* CSS rules moved here from style.css */
      h1 {
          color: cornflowerblue; /* Set heading color */
          /* text-decoration: underline; */ /* Temporarily disable underlining */
      }

      p {
          /* color: limegreen; */ /* Style currently commented out */
      }
    </style>

‚ú® Observe the Preview: The appearance should be identical to the previous step. The <h1> is still cornflowerblue, and the <p> still uses the default text color. This shows that the browser applies styles from the internal <style> block just as it did from the external file.
