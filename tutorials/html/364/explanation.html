364. Link Attributes: hreflang & type

Files Modified: None

These attributes provide hints to the browser and search engines about the linked resource.

  • hreflang:

    • Purpose: Specifies the language of the linked document.
    • Syntax: hreflang="language-code" (e.g., "en", "fr", "es-MX").
    • Usage: Useful when linking to versions of a page in different languages, helping search engines understand the language variations and potentially aiding browser translation features. It's only a hint; the linked document should ideally also declare its language (e.g., via its own <html> tag's lang attribute or HTTP headers).
    • Example: <a href="page-fr.html" hreflang="fr">Version franÃ§aise</a>

  • type:

    • Purpose: Specifies the MIME type (media type) of the linked resource.
    • Syntax: type="mime/type" (e.g., "application/pdf", "image/png", "text/csv").
    • Usage: Provides a hint to the browser about the file type before it downloads it. This might allow the browser to display a specific icon, warn the user, or decide not to navigate if it can't handle the type. It's purely advisory.
    • Example: <a href="report.pdf" type="application/pdf">Download Report (PDF)</a>

These attributes add metadata but don't typically change the link's visual appearance or core functionality directly.
