428. background-color: Syntax

Files Modified: None

The syntax for background-color is straightforward:

background-color: <color>;

Where <color> can be any valid CSS color value:

  • Keyword: red, blue, transparent, lightcyan
  • Hexadecimal: #ff0000, #f0f8ff (aliceblue)
  • RGB/RGBA: rgb(255, 99, 71), rgba(46, 62, 80, 0.8) (semi-transparent version of our current background)
  • HSL/HSLA: hsl(210, 14%, 83%), hsla(210, 14%, 83%, 0.5)
  • currentColor: Uses the element's text color.
