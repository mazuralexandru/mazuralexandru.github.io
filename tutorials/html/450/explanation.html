450. background-origin

Files Modified: style.css

The background-origin property determines the origin point for positioning the background image(s) using background-position.

  • Purpose: Specifies whether background-position is relative to the content box, padding box, or border box.
  • Syntax: background-origin: value;
  • Values:
    • padding-box (Default): background-position is calculated relative to the outer edge of the padding. 0 0 places the image at the top-left corner including padding.
    • border-box: background-position is calculated relative to the outer edge of the border. 0 0 places the image under the top-left border.
    • content-box: background-position is calculated relative to the edge of the content area (inside the padding). 0 0 places the image at the top-left corner of the content itself.
  • Multiple Backgrounds: Use comma-separated values for different layers.

✅ Observe style.css:

  • We've reverted to a simpler box with a single no-repeat background image.
  • We've set background-position: top left; (or 0 0).
  • We've explicitly set background-origin: border-box;.
/* Background Origin Example */
.origin-box {
    width: 300px;
    height: 150px;
    padding: 25px; /* Significant padding */
    border: 10px dashed orange; /* Thick border */
    margin: 2em auto;
    background-color: rgba(0, 100, 0, 0.2); /* Faint green */

    background-image: url('images/placeholder-pattern.png');
    background-repeat: no-repeat;
    background-size: 40px; 
    background-position: top left; /* Position at 0 0 */
    background-origin: border-box; /* Position relative to border edge */
    /* Try changing to padding-box (default) or content-box */
}

✨ Observe the Preview:

  • You should see a box with a thick orange dashed border and significant padding.
  • The small background pattern image should appear positioned at the very top-left corner, seemingly underneath the top and left borders, because its 0 0 position is calculated relative to the border-box.
  • Experiment: Use DevTools to change background-origin to padding-box (the default). The image should shift right and down, aligning with the inner edge of the border (the padding edge). Change it to content-box. The image should shift further right and down, aligning with the inner edge of the padding (the content edge).
