270. Minimum and Maximum Dimensions: min-width

Files Modified: style.css

Sometimes you want an element to be flexible in size (e.g., using percentages or auto), but you need to prevent it from becoming too small. The min-width property addresses this.

  • Purpose: Sets the minimum width an element can shrink to. If the content or the calculated width (like a percentage) would normally make the element narrower than the min-width value, the element's width will be forced to be at least the min-width.
  • Syntax: min-width: value;
  • Values: Length units (px, em, rem), percentages (%), or keywords like auto (default, no minimum), min-content, max-content, fit-content.
  • Interaction: min-width overrides width and max-width if they conflict (i.e., if width or max-width calculates to a value less than min-width).

✅ Observe style.css:

  • We've added min-width: 200px; to the .box rule.
.box {
  width: 75%; 
  height: auto; 
  min-width: 200px; /* Ensure box is at least 200px wide */
  background-color: cornflowerblue; 
  padding: 10px; 
}

✨ Observe the Preview: The box still tries to be 75% of the container width. However, if you make the preview window very narrow (if possible in your environment), you should see that the box stops shrinking once its total width (content + padding + border, depending on box-sizing) reaches approximately 200px (plus padding/border). It won't get narrower than that, even if 75% of the container width becomes smaller than 200px.
