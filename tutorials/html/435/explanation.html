435. background-repeat: Syntax

Files Modified: None

The background-repeat property can accept one or two values:

  • One Value: Applies the specified repeat behavior in both horizontal and vertical directions.
    • background-repeat: repeat; (tiles both ways)
    • background-repeat: no-repeat; (shows once)
    • background-repeat: repeat-x; (tiles horizontally only)
    • background-repeat: repeat-y; (tiles vertically only)
    • background-repeat: space; (spaces out tiles both ways)
    • background-repeat: round; (scales and tiles both ways)
  • Two Values: The first value specifies the horizontal repeat, the second specifies the vertical repeat.
    • background-repeat: repeat no-repeat; (Tiles horizontally, shows once vertically)
    • background-repeat: round space; (Rounds horizontally, spaces vertically)
  • Multiple Backgrounds: If you have multiple background images defined with background-image, you can specify repeat behaviors for each layer, separated by commas. The order corresponds to the order in background-image.
    • background-image: url(one.png), url(two.gif);
    • background-repeat: no-repeat, repeat-y; (one.png doesn't repeat, two.gif repeats vertically)
