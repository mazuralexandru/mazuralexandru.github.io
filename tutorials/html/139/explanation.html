139. <link href="..."> Explained

Files Modified: None

The href attribute specifies the location (URL or path) of the external resource being linked. For stylesheets, this is the path to your .css file.

  • Importance: If the href path is incorrect, the browser will not be able to find or load the CSS file, and none of the styles defined within it will be applied. This is a very common source of errors when starting out.
  • Relative Paths: As discussed, these are relative to the location of the HTML file.
    • styles.css: Same directory.
    • css/styles.css: In a css subdirectory.
    • ../styles.css: In the parent directory.
  • Absolute Paths: Full URLs (https://...) or root-relative paths (/css/styles.css). Root-relative paths start from the domain root and are often useful in larger sites to avoid complex ../ sequences.
  • Debugging: If styles aren't applying, the first thing to check is often the href path in your <link> tag and the actual location of your .css file. Browser developer tools (Network panel) can also show if the CSS file failed to load (404 Not Found error).
