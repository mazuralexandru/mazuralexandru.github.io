89. Linking Lesson Stylesheet (style.css)

Files Modified: index.html

Similar to how we linked base_styles.css, we now need to link our lesson-specific stylesheet, style.css, so that the rules we write in it will affect our HTML page.

We use another <link> tag inside the <head>. It's common practice to link your specific/overriding styles after your base/reset styles.

âœ… Observe index.html! A second <link> tag has been added, pointing to style.css.

    <link rel="stylesheet" href="style.css"> <!-- Link lesson styles -->

Now, any CSS rules we add to style.css will be loaded and applied by the browser (in addition to the rules from base_styles.css). Since style.css is still empty, there's no visual change yet.
