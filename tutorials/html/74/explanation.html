74. Quotations: Inline Quotes (<q>)

While <blockquote> handles block-level quotations, HTML provides the <q> element for short, inline quotations that occur within the flow of text.

The Inline Quote Element (<q>)

  • Purpose: Represents a short quotation embedded within surrounding prose.
  • Type: Inline element.
  • Syntax: Uses an opening <q> and closing </q> tag.
  • Content Model: Contains phrasing content. Cannot contain block-level elements.

Browser Rendering & Punctuation:

  • Automatic Quotes: The key feature of <q> is that browsers are supposed to automatically add appropriate quotation marks around the content based on the document's language (defined by the lang attribute on the <html> tag or a parent element) and nesting level.
    • For English (lang="en"), this usually means adding double quotes (“ ”) for the first level, and single quotes (‘ ’) for quotes nested inside other quotes.
  • Important: You should not manually type quotation marks (" or ') inside the <q> tags. Let the browser handle adding the correct punctuation according to linguistic conventions.
