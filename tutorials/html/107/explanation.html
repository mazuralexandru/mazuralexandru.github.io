107. HTML Comments: Syntax

Files Modified: None

The syntax for HTML comments is specific:

  • Start: <!-- (Left angle bracket, exclamation mark, two hyphens)
  • End: --> (Two hyphens, right angle bracket)
  • Content: Anything between the start and end delimiters is the comment text.

Important Notes:

  • No Dashes Inside: You cannot have two consecutive hyphens (--) inside the comment text itself, as this can be misinterpreted as the end of the comment by older browsers or validators. Use single hyphens or alternative phrasing if needed.
    • Bad: <!-- This is -- problematic -->
    • Good: <!-- This is - not problematic -->
  • Can Span Multiple Lines: Comments can start on one line and end several lines later, as shown in the multi-line example in index.html.
