23. Adding More Rules to External Stylesheets

Just like with internal stylesheets, you can add multiple rules to your external style.css file. Let's add the styling rule for the paragraph (<p>) element into style.css as well.

What's New?

  • We added a new p ruleset to the style.css file.
/* style.css snippet */
h1 {
  color: lightblue; /* Change H1 text color */
}

/* Added P rule */
p {
  font-style: italic;
  color: #ccc; /* Slightly different gray than before */
}

/* CSS files can also have comments! */

External CSS: The Standard

  • Adding rules to style.css follows the same CSS syntax.
  • This reinforces the benefits: all your presentation logic is centralized in one or more .css files, separate from your HTML structure. This is the standard, professional way to manage styles for websites of any significant size.
