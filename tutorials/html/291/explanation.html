291. Rounded Corners (border-radius): Purpose

Files Modified: None

Sharp rectangular corners can sometimes feel harsh in a design. The border-radius property allows you to round the corners of an element's border and background.

  • Purpose: To create rounded corners, softening the appearance of boxes. Can range from slightly rounded to fully circular/elliptical.
  • Effect: It affects the outer edge of the border and also clips the element's background (and content if overflow is hidden) to match the rounded shape.

This is a purely presentational property widely used in modern web design.
