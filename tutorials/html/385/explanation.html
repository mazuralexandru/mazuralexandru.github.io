385. Horizontal Nav (Flexbox): display: flex, Reset

Files Modified: style.css

Now let's implement the same horizontal navigation using the modern and preferred Flexbox approach.

Flexbox (Flexible Box Layout) is a one-dimensional layout model designed for distributing items along a single axis (row or column).

  • Step 1: Enable Flexbox: Apply display: flex; to the container element whose direct children you want to arrange. In our navigation example, the container is the <ul> element.
  • Step 2: Reset List Styles: As before, we need to remove the default list styles from the <ul>.

✅ Observe style.css:

  • We've commented out the previous inline-block styles.
  • We've added a new section for Flexbox.
  • The nav ul rule now has display: flex; added.
  • We ensure list-style, margin, and padding are reset on the ul.
  • We remove the text-align: center as Flexbox uses different properties for alignment.
  • We remove the styles previously applied to nav li as Flexbox controls item layout directly.
