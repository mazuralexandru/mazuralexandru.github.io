436. background-repeat: Values

Files Modified: None

Let's clarify the different repeat values:

  • repeat (Default): Tiles the image infinitely in both directions. If the image is smaller than the container, it will be repeated to fill it.
  • repeat-x: Tiles the image horizontally only. The vertical position is determined by background-position.
  • repeat-y: Tiles the image vertically only. The horizontal position is determined by background-position.
  • no-repeat: Displays the image only once. Its position is determined by background-position.
  • space: Tiles the image as many times as possible without clipping. If there's leftover space after fitting whole tiles, that space is distributed evenly between the tiles. The first/last tiles touch the edges of the container.
  • round: Tiles the image as many times as possible. If there's leftover space, the browser stretches or squashes the image dimensions slightly so that an integer number of tiles fits perfectly within the container without gaps or clipping.

The space and round values are less common but can be useful for creating seamless patterns without hard edges or partial tiles.
