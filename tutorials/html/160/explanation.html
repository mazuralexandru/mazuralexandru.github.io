160. Attribute Selectors

Files Modified: None

Beyond types, classes, and IDs, CSS allows you to select elements based on the presence or value of any of their HTML attributes. This provides a powerful and flexible way to target elements.

  • Purpose: Attribute selectors let you apply styles based on attributes like href, target, type, disabled, required, data-*, or even custom attributes.
  • Flexibility: They offer various matching options, from simply checking if an attribute exists to matching exact values or even substrings within the value.
