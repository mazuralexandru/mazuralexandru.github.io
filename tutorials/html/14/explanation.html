14. Recommended Editor: VS Code

Files Modified: None

We recommend using Visual Studio Code (VS Code). It's free, open-source, runs on Windows, macOS, and Linux, and has a massive ecosystem of extensions.

  • Installation: Download it from the official website: https://code.visualstudio.com/ and follow the installation instructions for your operating system.
  • Basic Interface:
    • Activity Bar (Left): Icons for different views (Explorer, Search, Source Control, Run, Extensions).
    • Side Bar (Left/Right): Displays the selected view (e.g., file explorer).
    • Editor Group (Center): Where your code files are opened and edited (supports tabs and split views).
    • Panel (Bottom): Contains output, debug console, terminal, problems.
    • Status Bar (Bottom): Shows information about the current file, project, errors, etc.

Familiarize yourself with the basic layout. We'll be using it to create our HTML and CSS files soon.
