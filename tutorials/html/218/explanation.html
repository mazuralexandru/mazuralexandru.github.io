218. letter-spacing & word-spacing

Files Modified: style.css

These two properties allow you to adjust the horizontal spacing between characters or words.

  1. letter-spacing:

    • Purpose: Adjusts the space between individual characters (tracking).
    • Values:
      • normal: (Default) Normal spacing for the font.
      • <length>: Specifies extra space in addition to the default spacing. Can use px, em, rem. Negative values are allowed to make spacing tighter.
    • Use Case: Often used subtly for headings or specific typographic effects. Excessive positive or negative spacing harms readability.

  2. word-spacing:

    • Purpose: Adjusts the space between words (specifically, added to the default space after a word).
    • Values:
      • normal: (Default) Normal spacing defined by the font/browser.
      • <length>: Specifies extra space in addition to the default inter-word space. Can use px, em, rem. Negative values are allowed.
    • Use Case: Less common than letter-spacing. Can be used for justification adjustments or specific design effects.

✅ Observe style.css:

  • We've added letter-spacing: 2px; to the #main-heading.
  • We've added word-spacing: 0.5em; to the .highlight-text paragraph.
#main-heading {
    /* ... other styles ... */
    text-align: center; 
    letter-spacing: 2px; /* Add space between letters */
}

.highlight-text {
  /* ... other styles ... */
  line-height: 1.8; 
  word-spacing: 0.5em; /* Add extra space between words */
}

✨ Observe the Preview:

  • The letters in "Main Heading" should now be spaced further apart.
  • The words in the first paragraph ("This is a standard paragraph...") should have noticeably more space between them. The amount of extra space (0.5em) is relative to the paragraph's font size.

Use these properties sparingly and test thoroughly to ensure readability isn't compromised.
