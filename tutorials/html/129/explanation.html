129. Internal Stylesheets: Scope

Files Modified: None

A key characteristic of internal stylesheets defined within <style> tags is their scope.

The CSS rules written inside a <style> element apply only to the HTML document in which they are embedded. They do not affect any other HTML pages, even if those pages are part of the same website or project.

This makes them suitable for styles that are unique to a single page, but less ideal for styles that need to be consistent across multiple pages (like site-wide branding or component styles). For cross-page consistency, external stylesheets are the better choice.
