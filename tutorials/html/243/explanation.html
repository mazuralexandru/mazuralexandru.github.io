243. <code> & <pre>: HTML Recap

Files Modified: None

Let's quickly recap the roles:

  • <code>: Inline semantic marker for computer code fragments. (Default: Monospace font).
  • <pre>: Block-level element that preserves whitespace and newlines from the source. (Default: Monospace font, white-space: pre).
  • Combination: Use <pre><code>...</code></pre> for semantic code blocks where formatting matters.
  • Escaping: Use &lt; and &gt; for literal < and > characters within code examples inside <code>.

Using these elements correctly improves the semantics and default presentation of code snippets on your web pages.
