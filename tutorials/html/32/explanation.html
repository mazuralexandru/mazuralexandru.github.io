32. Nesting HTML Elements

Files Modified: None

HTML elements can be placed inside other HTML elements. This is called nesting.

  • Concept: Creates a hierarchical structure, like boxes within boxes.
  • Terminology:
    • Parent Element: An element that contains another element.
    • Child Element: An element directly contained within another element.
    • Sibling Elements: Elements that share the same parent.

Example:

<div>  <!-- Parent element -->
  <p>  <!-- Child of div, Sibling of h2 -->
    This is a paragraph.
  </p>
  <h2> <!-- Child of div, Sibling of p -->
    This is a heading.
  </h2>
</div>

Here, <div> is the parent. <p> and <h2> are children of <div> and siblings of each other.

Nesting is fundamental to building complex page layouts and structuring content logically.
