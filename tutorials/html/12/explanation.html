12. Choosing Your Tools

Files Modified: None

To build websites, you need the right tools. The most fundamental tool is something to write your code in. You'll also need web browsers to test your creations.

We'll primarily focus on:

  • Text Editor / IDE: Software for writing and editing HTML, CSS, and JavaScript code.
  • Web Browser: Software for viewing and testing your web pages (Chrome, Firefox are recommended for development).
  • Browser Developer Tools: Built-in browser features for inspecting code, debugging, and analyzing performance.
