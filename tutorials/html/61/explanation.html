61. Text Styling: text-align

The text-align property controls the horizontal alignment of text (and other inline content) within its parent block-level element.

text-align Explained

  • Purpose: Specifies the horizontal alignment of inline-level content (like text, images, inline-blocks) within the block container it's applied to.
  • Applies To: Block-level elements (like <p>, <div>, <h1>). It affects the inline content inside them.
  • Values:
    • left: Aligns content to the left edge (default for LTR languages).
    • right: Aligns content to the right edge (default for RTL languages).
    • center: Centers content horizontally.
    • justify: Stretches lines of text to fill the container width by adjusting word spacing. The last line of a paragraph is typically left-aligned. Justification can sometimes create large, awkward gaps between words ("rivers") if columns are narrow or text isn't hyphenated, so use it carefully and test thoroughly.
    • start: Aligns to the start edge based on writing direction (left for LTR, right for RTL). Logical equivalent of left/right.
    • end: Aligns to the end edge based on writing direction (right for LTR, left for RTL). Logical equivalent of left/right.
  • Inheritance: Yes, text-align is inherited by child elements.
