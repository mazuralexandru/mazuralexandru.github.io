466. Forms: Options (<option>) - selected and disabled Attributes

Two useful boolean attributes can be applied to <option> elements: selected and disabled.

1. selected Attribute:

  • Purpose: Specifies that this <option> should be pre-selected when the page loads.
  • Syntax: <option value="..." selected>...</option>
  • Behavior:
    • The text of the selected option is displayed in the <select> box by default.
    • If no selected attribute is present on any <option>, the browser typically selects the first option in the list by default.
    • In a single-choice <select> (default), only one <option> should have the selected attribute. If multiple have it, the browser usually selects the last one marked as selected in the source order.
    • In a multi-select <select multiple>, multiple options can have the selected attribute.

2. disabled Attribute:

  • Purpose: Disables an <option>, making it unselectable by the user.
  • Syntax: <option value="..." disabled>...</option>
  • Behavior:
    • The option appears in the dropdown list, but it's typically greyed out and cannot be chosen.
    • Often used for placeholder options (like "-- Please select --") or options that are temporarily unavailable.
    • A disabled option cannot be submitted. If a disabled option also has selected, the form control might behave as if no option is selected initially (depending on browser).

Code Change:

We've modified the <option> elements in index.html:

  1. Added a first, disabled, selected option as a placeholder.
  2. Removed selected from the "Canada" option (now the placeholder is selected).
  3. Added the disabled attribute to the "United Kingdom" option.
<!-- index.html snippet -->
<form action="/submit-data" method="POST">
  <p>
    <label for="country-select">Choose a country:</label>
    <select name="country" id="country-select">
      <option value="" disabled selected>-- Please select --</option> <!-- Placeholder -->
      <option value="US">United States</option>
      <option value="CA">Canada</option>
      <option value="GB" disabled>United Kingdom (Unavailable)</option> <!-- Disabled option -->
      <option value="AU">Australia</option>
    </select>
  </p>
  <p><button type="submit">Submit</button></p>
</form>

✅ Observation:

  1. Examine the attributes on the <option> tags. Note disabled selected on the first and disabled on the fourth.
  2. Look at the browser preview:
    • The text displayed in the box initially is "-- Please select --".
    • Click the dropdown. The "-- Please select --" option might be slightly greyed out or unselectable.
    • The "United Kingdom (Unavailable)" option should appear greyed out and you should not be able to select it.
    • You can select "United States", "Canada", or "Australia".

✨ Key Takeaway: Use the boolean selected attribute to pre-select a default <option>. Use the boolean disabled attribute to make an <option> visible but unselectable. A common pattern is <option value="" disabled selected>-- Please select --</option> for placeholder instructions.
