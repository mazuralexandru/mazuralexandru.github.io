429. background-color: Transparency

Files Modified: style.css

You can easily create semi-transparent backgrounds using color formats that support an alpha channel: rgba() and hsla().

  • RGBA: rgba(red, green, blue, alpha) - Alpha is 0 (transparent) to 1 (opaque).
  • HSLA: hsla(hue, saturation, lightness, alpha) - Alpha is 0 to 1.

The transparent keyword is equivalent to rgba(0,0,0,0) (or any color with 0 alpha).

✅ Observe style.css:

  • We've changed the background-color for .bg-box to use rgba() with 50% opacity.
.bg-box {
    /* background-color: #2c3e50; */ /* Dark blue-gray */
    background-color: rgba(44, 62, 80, 0.5); /* Same color, 50% transparent */
    padding: 2em;
    margin-bottom: 1em;
    border: 1px dashed #fff; 
}
