415. filter Property: Common Filter Functions

Files Modified: index.html, style.css

Let's explore some common filter functions and apply them.

  • grayscale(percentage|number): Converts the image to grayscale. 0% or 0 is original, 100% or 1 is fully grayscale.
  • sepia(percentage|number): Applies a sepia tone. 0%/0 is original, 100%/1 is full sepia.
  • brightness(percentage|number): Adjusts brightness. 100%/1 is original, <100% darkens, >100% brightens.
  • contrast(percentage|number): Adjusts contrast. 100%/1 is original.
  • blur(length): Applies a Gaussian blur. Value is a length like px, em. 0 is no blur.
  • invert(percentage|number): Inverts colors. 100%/1 is full inversion (like a photo negative).
  • opacity(percentage|number): Applies transparency. 100%/1 is fully opaque, 0%/0 is fully transparent. Similar to the opacity property.
  • saturate(percentage|number): Adjusts color saturation. 100%/1 is original, 0%/0 is grayscale, >100% increases saturation.
  • hue-rotate(angle): Rotates colors around the color wheel. Value is an angle like 90deg, 0.5turn.
  • drop-shadow(...): Similar syntax to box-shadow (h-offset v-offset blur color), but applies the shadow to the shape of the element's content (including transparency) rather than its box.

/* Image Filter Examples */
.img-grayscale {
    filter: grayscale(100%);
}

.img-sepia {
    filter: sepia(80%);
}

.img-blur-hover:hover {
    filter: blur(3px);
    transition: filter 0.3s ease-in-out;
}
