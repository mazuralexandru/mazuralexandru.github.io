240. <small>, <sup>, <sub>: Default Styling

Files Modified: index.html

Browsers apply default styling to render these elements according to their meaning:

  • <small>: Smaller font size.
  • <sup>: Smaller font size, raised baseline (vertical-align: super;).
  • <sub>: Smaller font size, lowered baseline (vertical-align: sub;).

✅ Observe index.html:

  • We've added a paragraph demonstrating these tags.
    <hr>
    <p>
      This is normal text. <small>This is small print.</small> 
      This is the 1<sup>st</sup> example. 
      Water is H<sub>2</sub>O.
    </p>

✨ Observe the Preview:

  • "This is small print." should appear smaller than the surrounding text.
  • The "st" in "1st" should be smaller and raised (superscript).
  • The "2" in "H2O" should be smaller and lowered (subscript).
