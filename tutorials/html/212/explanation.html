212. font Shorthand Property

Files Modified: None

CSS provides a shorthand font property to set multiple font-related properties in a single declaration.

  • Purpose: To set font-style, font-variant, font-weight, font-size, line-height, and font-family all at once.
  • Syntax: The order of values matters! font: [font-style || font-variant || font-weight]? font-size [/line-height]? font-family
    • font-style, font-variant, font-weight are optional and can appear in any order before font-size.
    • font-size is mandatory.
    • line-height is optional and must come immediately after font-size, separated by a forward slash (/).
    • font-family is mandatory and must be the last value(s).
  • Resetting: Any omitted optional values are reset to their initial values (e.g., if you don't specify font-weight, it resets to normal). This is important! Using the shorthand will override any previously set individual font properties.

Examples:

/* Set size and family (style, variant, weight reset to normal) */
p { font: 1rem sans-serif; } 

/* Set style, weight, size, line-height, family (variant reset to normal) */
h1 { font: italic bold 2.5rem/1.2 "Times New Roman", serif; } 

/* Set weight, size, family */
.label { font: 600 0.9rem monospace; } 

/* Using system font keywords */
button { font: caption; } /* Use OS default for captioned controls */

Recommendation: While shorthand can be concise, be mindful that it resets omitted properties. It's often clearer, especially for beginners, to set individual properties unless you intentionally want to set most of them together.
