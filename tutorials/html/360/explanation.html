360. Link Content: Text & Images

Files Modified: index.html

The content between the opening <a> and closing </a> tags is what becomes clickable. This content should be descriptive and clearly indicate the link's destination or purpose.

  • Text Links: The most common type. The text itself describes the link.
    <a href="products.html">View Our Products</a>
  • Image Links: You can wrap an <img> element within an <a> tag to make the image clickable.
    • Accessibility: When using an image as a link, it's crucial that the <img> element has meaningful alt text. This alt text serves as the link content for screen readers and users who have images disabled. If the link also contains descriptive text alongside the image, the image's alt attribute can sometimes be empty (alt="") to avoid redundancy, but it's generally better to describe the image.
    • Default Border: Older browsers sometimes added a border around linked images. Modern resets often remove this, but you can explicitly remove it with CSS (a img { border: none; }) if needed.
