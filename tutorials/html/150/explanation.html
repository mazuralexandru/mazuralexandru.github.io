150. The href Attribute: Core Function

To make an anchor element (<a>) function as an actual hyperlink, we must add the href attribute.

The href Attribute:

  • Stands For: Hypertext Reference.
  • Purpose: Specifies the destination of the hyperlink. It tells the browser where to go or what resource to access when the link is activated.
  • Syntax: href="destination_url"
    • It's added inside the opening <a> tag.
    • The value assigned to href is typically a URL (Uniform Resource Locator) or a specific protocol handler string.
  • Requirement: The href attribute is essential for creating a navigable link.

Placeholder href Value (#)

A common practice during development, or for links designed to be handled only by JavaScript, is to use href="#" as a placeholder value.

  • # normally signifies a link to the top of the current page (a fragment identifier).
  • Using href="#" creates a link that looks like a link (gets default styling) and is interactive, but clicking it typically just scrolls to the top of the page without navigating away (though JavaScript often prevents this default scroll behavior).

Code Change:

We have added href="#" to the anchor element in index.html and updated the surrounding text.

<p>Here is an anchor element with href: <a href="#">Placeholder Link</a></p> <!-- Added href="#" -->

<p>The href attribute defines the link's destination.</p> <!-- Adjusted P -->

✅ Observation:

  1. Locate the href="#" attribute within the <a> tag in index.html.
  2. Look at the browser preview. Compare the "Placeholder Link" text to the "Empty Anchor" from the previous step:
    • It should now have the default hyperlink styling (typically blue color and an underline).
    • Hovering over it should change the mouse cursor to a pointer (hand icon).
    • Clicking it might cause the page to jump to the top (or do nothing if prevented by tooling).
    • It now looks and behaves like a link, even though its destination is just a placeholder.

✨ Key Takeaway: The href attribute is essential for making an <a> tag a functional hyperlink and defines its destination. Using href="#" creates a styled, clickable link that doesn't navigate away from the page (typically scrolls to top).
