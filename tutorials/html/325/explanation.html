325. Metadata: Author (name="author") - Core Mechanics

Let's add the <meta name="author"> tag to our example.

Syntax:

  • Place the tag within the <head> section. <meta name="author" content="Creator's Name Here">

content Value:

  • The value should typically be the name of the individual author (e.g., "Jane Doe") or the organization responsible for the content (e.g., "Example Corporation Content Team").
  • Including contact information like email addresses directly in the content is possible but generally discouraged due to spam risks (similar to mailto: links).

Code Change:

We've added a <meta name="author"> tag to the <head> of index.html.

<!-- index.html -->
<head>
  <meta charset="UTF-8">
  <meta name="description" content="Learn about HTML metadata elements like meta, link, title, base, style, and script found in the document head section. Essential for SEO and page setup.">
  <meta name="author" content="LearnHTML Examples"> <!-- Added author meta tag -->
  <title>HTML Metadata</title>
  <link rel="stylesheet" href="style.css">
  <!-- Other meta tags will be added here -->
</head>
<body>
  <h1>Metadata: Author</h1>
  <p>Added the 'author' meta tag to the head.</p> <!-- Adjusted P -->

  <p>It specifies the creator of the document content.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>
</html>

✅ Observation:

  1. Locate the new <meta name="author" content="LearnHTML Examples"> tag within the <head> in index.html.
  2. Look at the browser preview. As with other informational meta tags, there is no visual change on the rendered page.

✨ Key Takeaway: Add the <meta name="author" content="Author Name"> tag to the <head> to specify the document's creator.
