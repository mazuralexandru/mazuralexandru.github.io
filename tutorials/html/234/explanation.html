234. Styling Recommendation: Prefer Classes

Files Modified: style.css

While you can directly style i, b, and u using type selectors in CSS, it's often better practice to use classes for purely presentational styling.

Why?

  • Clarity of Intent: Using a class like .text-bold or .italicize makes it clear that the styling is purely presentational, separate from any potential (and sometimes ambiguous) semantic meaning of <i> or <b>.
  • Maintainability: If you decide to change how "bold" text looks across your site (maybe a different font-weight number or a color), you only need to update the .text-bold class definition, rather than potentially overriding b and strong styles.
  • Flexibility: You can apply a presentational class to any element (<span>, <div>, etc.) using <span class="text-bold">...</span>, whereas using <b> forces you to use that specific tag.
  • Avoiding <u>: As mentioned, <u> should generally be avoided. If you need an underline for non-link text, use a class and text-decoration: underline;.

✅ Observe style.css:

  • We've added example classes .italic-text and .bold-text.
  • We've also added a style for u to explicitly discourage its use by making it visually distinct and potentially annoying.
/* Styling Recommendation: Use Classes for Presentation */
.italic-text {
  font-style: italic;
}
.bold-text {
  font-weight: bold;
}

/* Discourage use of <u> by making it obvious */
u {
  text-decoration-color: red;
  text-decoration-style: wavy;
  background-color: rgba(255, 0, 0, 0.1);
}

✨ Observe the Preview:

  • The text within the <i> and <b> tags remains unchanged as we haven't applied the new classes to them in the HTML yet.
  • However, the text within the <u> tag ("u element") should now have a wavy red underline and a faint red background, highlighting why it's generally best avoided for standard underlining due to potential confusion and the ability to make it look non-standard.

In practice, you would apply class="italic-text" or class="bold-text" to elements like <span> when you need purely visual italicization or bolding without specific semantic meaning.
