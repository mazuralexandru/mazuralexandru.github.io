421. Styling <figure>: Centering

Files Modified: style.css

Since <figure> is a block-level element and we've given it a max-width, we can center it horizontally using the familiar margin: auto; technique.

✅ Observe style.css:

  • We've added margin-left: auto; and margin-right: auto; to the figure rule. (We kept the margin-bottom from before).
figure {
  /* margin: 0; <- Remove this */
  margin-top: 0; /* Explicitly reset top */
  margin-bottom: 1.5em; 
  margin-left: auto; /* Add auto left margin */
  margin-right: auto; /* Add auto right margin */
  max-width: 500px; 
  padding: 1em; 
  border: 1px solid #555; 
  background-color: rgba(255, 255, 255, 0.03); 
}

/* ... rest of styles ... */

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview: The entire figure (the box containing the image and caption) should now be horizontally centered within the page body, provided the viewport is wider than its max-width (500px plus padding and border).
