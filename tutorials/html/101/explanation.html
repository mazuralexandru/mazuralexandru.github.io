101. CSS Property: background-color

Files Modified: None

The background-color property sets the background color of an HTML element.

Applies to: All elements. Inherited: No (backgrounds are not typically inherited, though the default is often transparent, allowing parent backgrounds to show through).

Values: Accepts the same color value types as the color property:

  • Color Keywords (e.g., lightblue, transparent)
  • Hexadecimal (e.g., #f0f0f0, #333)
  • RGB/RGBA (e.g., rgb(200, 200, 200), rgba(0, 0, 0, 0.5))

Example Usage:

body {
  background-color: #f8f8f8; /* Light gray page background */
}

.highlight {
  background-color: yellow; /* Yellow background for highlighted sections */
}

div {
  background-color: transparent; /* Explicitly transparent background */
}
