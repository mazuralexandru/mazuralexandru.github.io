09. Basic Element Syntax Example: The Paragraph

Let's apply what we've learned and create our first actual HTML element: a paragraph.

The Paragraph Element (<p>):

The <p> element is one of the most common HTML elements. It's used to define a paragraph of text.

Code Change:

We've modified the index.html file. We added a line below the comment:

<!-- index.html -->
<!-- Content will go here soon! -->
<p>This is a paragraph.</p> <!-- Added this line -->

Deconstructing the Element:

Let's break down <p>This is a paragraph.</p>:

  1. Opening Tag: <p> - This tells the browser: "A paragraph starts here."
  2. Content: This is a paragraph. - This is the actual text content that will be displayed.
  3. Closing Tag: </p> - This tells the browser: "The paragraph ends here."

Together, these three parts form a complete paragraph element.

✅ Observation:

  1. Look at the index.html file content. You see the new line with the <p> element.
  2. Look at the browser preview pane. You should now see the text "This is a paragraph." rendered on the page! The browser understood the <p> tags and displayed the content between them. Note that the comment is still not visible.

✨ Experiment: Try changing the text content between the <p> and </p> tags in index.html. Save the file (if your preview doesn't auto-update) and see the text change in the preview. Remember to keep the tags themselves intact!
