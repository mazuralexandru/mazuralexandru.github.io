145. Type Selectors: Scope

Files Modified: None

The defining characteristic of a type selector is its broad scope.

A type selector like p will match and apply its styles to every single <p> element present in the HTML document, regardless of where it is nested or what other attributes it might have.

Similarly, h1 affects all <h1> elements, li affects all <li> elements, and so on.

This makes type selectors powerful for setting baseline styles for fundamental HTML elements, but often too broad for styling specific components or variations.
