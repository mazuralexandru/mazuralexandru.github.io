456. background Shorthand Property

Files Modified: style.css

Like other box model properties, CSS provides a background shorthand to set multiple background properties in one declaration.

  • Purpose: Combines background-color, background-image, background-repeat, background-attachment, background-position, background-size, background-origin, and background-clip.
  • Syntax: While the order isn't strictly enforced for most values, a common and safe order is: background: <color> <image> <repeat> <attachment> <position>;
    • If using background-size, it must come immediately after background-position, separated by a forward slash (/).
    • background: <color> <image> <repeat> <attachment> <position> / <size>;
    • background-origin and background-clip can often be placed near the end, but check browser compatibility if combining all properties. The background-clip value should typically come after background-origin if both are specified.
  • Resetting: Any omitted properties are reset to their initial values (e.g., omitting background-image sets it to none, omitting background-color sets it to transparent, omitting background-repeat sets it to repeat, etc.). Be careful when using the shorthand not to unintentionally reset properties you wanted to keep.
  • Multiple Backgrounds: Layer properties comma-separated, with the background-color only applicable to the last layer specified.

✅ Observe style.css:

  • We've replaced the individual background properties in the .origin-box rule with a single background shorthand declaration.
.origin-box {
    width: 300px;
    height: 150px;
    padding: 25px; 
    border: 10px dashed orange; 
    margin: 2em auto;
    
    /* Replaced individual properties with shorthand: */
    background: 
        rgba(0, 100, 0, 0.2) /* color */
        url('images/placeholder-pattern.png') /* image */
        repeat /* repeat */
        scroll /* attachment (default, explicit here) */
        top left / 40px /* position / size */
        padding-box /* origin (default, explicit here) */
        content-box; /* clip */
}
