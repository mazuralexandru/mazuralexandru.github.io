427. background-color

Files Modified: index.html, style.css

Let's begin exploring CSS background properties, starting with the simplest: background-color.

  • Purpose: Sets a solid background color for an element. This color fills the element's content, padding, and border areas (unless modified by background-clip).
  • Default: The default value is transparent.
