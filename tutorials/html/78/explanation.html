78. Adding Base Styles

Files Modified: base_styles.css

Now that we've linked base_styles.css in the previous step, let's add some foundational CSS rules to it.

✅ First, look at index.html (it's unchanged from the last step, but it contains the crucial <link> tag).

✨ Next, switch to base_styles.css. You'll see we've added CSS rules that target the <body> element. These rules establish:

  • A dark background (background-color: #2e2e2e;).
  • Light default text color (color: #E0E0E0;).
  • A standard sans-serif font (font-family: sans-serif;).
  • Some padding around the body content (padding: 15px;).
  • A minimum height (min-height: 150px;) to ensure the background grid is visible even if the body is empty.
  • A subtle background grid pattern using CSS gradients and variables (--grid-size, etc.) for visual reference during development.

These styles provide a consistent starting point for our lessons. Because base_styles.css is linked in our HTML, these styles are now applied to the page preview, giving us the dark theme and grid background. We won't usually need to modify this file again; lesson-specific styles will go in style.css.
