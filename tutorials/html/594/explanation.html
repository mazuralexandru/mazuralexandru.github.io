594. Graphics: Canvas (<canvas>) - 2D Context API Overview (Conceptual)

Once you have the 2D rendering context object (e.g., const ctx = canvas.getContext('2d');), you can use its methods and properties to draw. This is primarily a JavaScript topic, but understanding the capabilities is relevant to HTML context.

Key API Concepts (What You Can Do):

  • Coordinate System: (0,0) is top-left. X increases right, Y increases down. Units are pixels based on width/height attributes.
  • Basic Shapes:
    • Rectangles: ctx.fillRect(x,y,w,h) (filled), ctx.strokeRect(x,y,w,h) (outline), ctx.clearRect(x,y,w,h) (erase).
  • Paths (Complex Shapes):
    • Define custom shapes using sequences of lines (moveTo, lineTo), arcs (arc), curves (bezierCurveTo, quadraticCurveTo), and rectangles (rect).
    • Start with ctx.beginPath().
    • Draw the defined path using ctx.fill() (fill with fillStyle) or ctx.stroke() (outline with strokeStyle and line styles).
  • Styling:
    • Colors, Gradients, Patterns: Set ctx.fillStyle and ctx.strokeStyle.
    • Line Appearance: ctx.lineWidth, ctx.lineCap, ctx.lineJoin, ctx.setLineDash().
    • Transparency: ctx.globalAlpha.
    • Shadows: ctx.shadowColor, ctx.shadowBlur, etc.
  • Text:
    • Draw Text: ctx.fillText(text, x, y), ctx.strokeText(text, x, y).
    • Style Text: ctx.font (CSS syntax), ctx.textAlign, ctx.textBaseline.
  • Images:
    • Draw Images/Videos/Canvases: ctx.drawImage(imageSource, dx, dy, ...). Allows drawing existing images, frames from videos, or other canvases onto the current canvas. Can also clip (sx, sy, sw, sh) and scale (dw, dh).
  • Transformations:
    • Modify the coordinate system before drawing: ctx.translate(x, y), ctx.rotate(angle), ctx.scale(xScale, yScale).
    • Managed using ctx.save() and ctx.restore() to isolate transform effects.
  • State Management: ctx.save() and ctx.restore() preserve and restore the entire drawing state (styles, transformations, clipping regions), crucial for complex drawings.

Example (Conceptual JS Snippet):

// Conceptual drawing - Requires ctx from previous step
if (ctx) {
  // Draw a blue rectangle
  ctx.fillStyle = 'blue';
  ctx.fillRect(10, 10, 100, 50); // x, y, width, height

  // Draw red text
  ctx.fillStyle = 'red';
  ctx.font = 'bold 20px sans-serif';
  ctx.fillText('Canvas Text!', 10, 90); // text, x, y

  // Draw an outlined circle
  ctx.strokeStyle = 'green';
  ctx.lineWidth = 3;
  ctx.beginPath();
  ctx.arc(200, 75, 30, 0, Math.PI * 2); // x, y, radius, startAngle, endAngle
  ctx.stroke();
}
