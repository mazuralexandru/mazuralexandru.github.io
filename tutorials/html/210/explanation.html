210. font-style

Files Modified: style.css

The font-style property primarily controls whether text is displayed in italics.

  • Purpose: To apply italic or oblique styling to text.
  • Values:
    • normal: Displays text normally (not italicized). This is the default for most elements.
    • italic: Displays text using the italic version of the current font-family, if available. Italic fonts are typically cursive-like, specially designed variants.
    • oblique: Displays text using an electronically slanted version of the normal font face. If the font doesn't have a true italic variant, browsers might simulate italics by applying an oblique slant. You can sometimes specify an angle (e.g., oblique 10deg), but browser support varies.
  • Font Availability: Like font-weight, applying italic works best if the selected font family actually includes a dedicated italic face. If not, the browser might apply the oblique effect instead.

✅ Observe style.css:

  • We previously applied font-style: italic; to both the heading and the first paragraph using a grouped selector.
  • Let's remove the .highlight-text from the group so only the heading remains italic.
/* --- Grouping Example --- */
#main-heading { /* Removed .highlight-text from group */
  font-style: italic; /* Apply italic only to heading now */
}
  • We also need to remove the .highlight-text selector from the group in the CSS file. Let's adjust the grouping rule.
/* --- Grouping Example --- */
#main-heading { /* Now only targets the heading */
  font-style: italic; /* Apply italic to heading */
}

/* .highlight-text is no longer targeted by the above rule */

✨ Observe the Preview:

  • The main heading (<h1>) should still be italic.
  • The first paragraph (<p class="highlight-text">) should no longer be italic, reverting to its normal style (though it retains its yellow text color, background, padding, and dotted border from other rules).

This demonstrates how font-style applies italicization and how removing a selector from a group changes which elements are affected.
