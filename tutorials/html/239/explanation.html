239. Other Inline Semantics (<small>, <sup>, <sub>)

Files Modified: None

HTML offers a few more inline elements for specific semantic nuances:

  • <small> (Side Comments):
    • Semantic Meaning: Represents side comments, small print, copyright information, legal disclaimers, or attribution. Text that is typically rendered smaller.
    • Default Styling: Renders text one size smaller than the parent element's font size.
    • Example: <p>Total price: $100. <small>Taxes not included.</small></p>
  • <sup> (Superscript):
    • Semantic Meaning: Represents superscript text, typically used for exponents, ordinal numbers (like 1st), or footnote markers.
    • Default Styling: Renders text smaller and raised above the baseline.
    • Example: E = mc<sup>2</sup> or The 4<sup>th</sup> of July.
  • <sub> (Subscript):
    • Semantic Meaning: Represents subscript text, typically used for chemical formulas (like H2O) or mathematical variables/indices.
    • Default Styling: Renders text smaller and lowered below the baseline.
    • Example: The chemical formula for water is H<sub>2</sub>O.

Styling Note: While these have default styles, if you only need text to be smaller, superscripted, or subscripted for purely presentational reasons without the specific semantic meaning, it's often better to use a <span> with CSS classes applying font-size and/or vertical-align. However, for their intended semantic purposes (like chemical formulas or exponents), using the correct HTML tags is appropriate.
