Edits & Annotations Demo

Use s tags (strikethrough) for content that is no longer accurate or relevant. It differs subtly from `del`.

Use del tags (deleted text) to mark content that has been removed from the document. It often implies tracking changes. This text was inserted.

Use ins tags (inserted text) to mark content that has been added to the document. Browsers often underline inserted text.

Use mark tags to indicate text that is marked or highlighted for reference or relevance in the current context, like search results.
