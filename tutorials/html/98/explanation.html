98. CSS Color Values: Keywords

Files Modified: None

The simplest way to specify a color in CSS is using predefined color keywords.

  • What they are: Case-insensitive identifiers that represent specific colors (e.g., red, blue, yellow, black, white, gray, orange, purple, green).
  • Extended Keywords: Modern CSS supports over 140 color names, including more descriptive ones like tomato, cornflowerblue, lightsalmon, darkseagreen.
  • Special Keyword: transparent represents a fully transparent color.

Pros: Easy to read and remember for common colors. Cons: Limited palette compared to other methods; names can be subjective.

Example:

h1 { color: steelblue; }
.warning { color: orange; }
p { color: black; }
