102. CSS Property: font-size

Files Modified: None

The font-size property controls the size of the text within an element.

Applies to: Elements containing text. Inherited: Yes (child elements generally inherit the font size from their parent, unless overridden).

Common Value Types:

  1. Length Units:
    • Pixels (px): Absolute unit, defines a fixed size (e.g., 16px). Common for base font sizes or when precise pixel control is needed.
    • Ems (em): Relative unit. 1em is equal to the computed font size of the parent element. 2em is twice the parent's size, 0.8em is 80%. Useful for scalable layouts.
    • Rems (rem): Relative unit. 1rem is equal to the font size of the root element (<html>). Generally preferred over em for overall page scaling, as it avoids compounding effects from nested elements. 2rem is twice the root font size.
    • Percentage (%): Relative to the parent element's font size (e.g., 120%). Similar to em.
  2. Keywords: Absolute sizes (xx-small, x-small, small, medium, large, x-large, xx-large) and relative sizes (smaller, larger). Less precise and less commonly used than length units.

Example Usage:

body { 
  font-size: 16px; /* Base font size for the page */
}
h1 {
  font-size: 2rem; /* Twice the root font size */
}
.small-text {
  font-size: 0.8em; /* 80% of the parent's font size */
}
