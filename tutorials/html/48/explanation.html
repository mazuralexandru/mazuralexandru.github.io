48. Component 2: <html> Element

Files Modified: None

Immediately following the <!DOCTYPE html> declaration comes the <html> element.

  • Purpose: This is the root element of the entire HTML document. All other elements (except the DOCTYPE) must be descendants of the <html> element. It acts as the main container for the entire page structure.
  • Syntax: It requires both an opening tag <html> and a closing tag </html>.
  • Content: It typically contains exactly two direct children:
    1. A <head> element.
    2. A <body> element.

Think of <html>...</html> as wrapping everything that constitutes your actual web page content and metadata.
