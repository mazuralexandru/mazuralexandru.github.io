186. Inheritance Explained

Files Modified: None

The final major concept governing how styles are applied is Inheritance.

  • Concept: Some CSS properties, when set on a parent element, are automatically passed down (inherited) by their child elements in the HTML tree, unless a style is explicitly set on the child element itself.
  • Mechanism: If no value for an inheritable property is specified directly on an element (or via the cascade winning for that element), the browser looks at the element's parent. If the parent has a computed value for that property, the child inherits it. This continues up the ancestor chain until a value is found or the root (<html>) is reached. If no inherited value is found, the property's initial default value is used.

Inheritance is a natural way for styles like font choices and text colors to flow through the document without needing to be declared on every single element.
