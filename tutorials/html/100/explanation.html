100. CSS Color Values: RGB/RGBA

Files Modified: None

RGB (Red, Green, Blue) and RGBA (Red, Green, Blue, Alpha) provide another way to specify colors, often considered more readable than hex for understanding the components.

  • RGB Syntax: rgb(red, green, blue)
    • red, green, blue are typically integer values from 0 to 255 (inclusive).
    • Example: rgb(255, 0, 0) (Pure Red), rgb(0, 0, 0) (Black), rgb(255, 255, 255) (White).
  • RGBA Syntax: rgba(red, green, blue, alpha)
    • Adds an alpha channel value for transparency.
    • alpha ranges from 0 (fully transparent) to 1 (fully opaque). Decimal values like 0.5 create semi-transparency.
    • Example: rgba(0, 0, 255, 0.7) (Semi-transparent Blue), rgba(0, 0, 0, 0.2) (Very transparent Black/dark overlay).

Pros: Explicitly shows RGB components. RGBA provides a standard, well-supported way to handle transparency. Cons: Slightly more verbose than hex.
