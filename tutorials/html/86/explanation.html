86. Other Inline Semantics: Strikethrough (<s>)

The last inline element in this section is <s>, used to represent text that is no longer accurate or relevant.

The Strikethrough Element (<s>)

  • Purpose: Represents content that is no longer relevant, accurate, or applicable. It indicates that the information is outdated or incorrect, but it's kept visible for context (e.g., to show a price change or a completed task).
  • Semantics: The meaning is about the status of the information (incorrect, obsolete), not necessarily about deletion or removal from the document's history (which is closer to <del>).
  • Type: Inline element.
  • Syntax: <s>...</s>

Default Rendering:

Browsers typically render text inside <s> tags with a line drawn horizontally through the middle of the text (strikethrough).

Use Cases:

  • Indicating discounted prices (striking out the old price).
  • Marking items in a list as completed or no longer available.
  • Showing edits or revisions where the original text is incorrect but needs to remain visible.
