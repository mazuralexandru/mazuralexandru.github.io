373. Link State: :focus-within

Files Modified: index.html, style.css

The :focus-within pseudo-class is applied to a parent element when that element itself or any of its descendants receives focus.

  • :focus-within:
    • Selects: An element when it contains the focused element.
    • Purpose: Useful for highlighting a larger container (like a form section, a card, or a menu) when one of its internal elements (like an input field or a link) receives focus. This helps users understand the context of the focused element.

✅ Code Pruning: We've cleared index.html and style.css to set up a specific example for :focus-within.

✅ Observe index.html:

  • We've created a <form> containing a <div> (acting as a field group) which holds a <label> and an <input>.

✅ Switch to style.css:

  • We add basic styling for the div (the field group).
  • We add the key rule: div:focus-within. This rule targets the div only when an element inside it (in this case, the <input>) receives focus. We apply a background color change.
  • We also add a basic focus style for the input itself for clarity.
/* Focus Within Example */
div {
  padding: 1em;
  margin-bottom: 1em;
  border: 1px solid #555;
  border-radius: 4px;
  transition: background-color 0.3s ease; /* Smooth transition */
}

/* Style the parent div when an element inside it gets focus */
div:focus-within {
  background-color: rgba(255, 255, 0, 0.15); /* Highlight container */
  border-color: yellow;
}

/* Basic focus style for the input itself */
input:focus {
  outline: 2px solid skyblue;
}

label {
    margin-right: 0.5em;
}

  • When the input loses focus (click outside or Tab away), the background highlight on the <div> should disappear.
