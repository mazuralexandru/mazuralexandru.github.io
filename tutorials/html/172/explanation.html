172. target Attribute: _blank (New Tab/Window)

A very common requirement is to open a link in a new browser tab or window, keeping the original page open. This is achieved using target="_blank".

target="_blank"

  • Value: _blank
  • Behavior: Opens the linked document in a new, unnamed browsing context. This usually means:
    • A new browser tab.
    • Sometimes, depending on browser settings, a new browser window.
  • Each click on a link with target="_blank" typically opens another new tab/window.

Use Cases:

  • External Links: Often used when linking to external websites, allowing the user to explore the external site without losing their place on your site.
  • Documents/Files: Useful for links to resources like PDFs, images, or documents that the user might want to view or download separately while keeping your page open.

Code Change:

We have added target="_blank" to the anchor element in index.html.

<p>
  Example Link (_blank):
  <a href="https://developer.mozilla.org/" target="_blank">MDN Home (New Tab)</a> <!-- Added target="_blank" -->
</p>

✅ Observation:

  1. Locate the target="_blank" attribute in the <a> tag in index.html.
  2. In the preview, click the "MDN Home (New Tab)" link.
  3. Observe that the MDN homepage opens in a new browser tab (or possibly window), leaving your original lesson page open in the previous tab.

⚠️ Important Note: As we'll discuss next, using target="_blank" has security and performance implications that require using the rel attribute as well.

✨ Key Takeaway: Use target="_blank" to make a link open in a new browser tab or window.
