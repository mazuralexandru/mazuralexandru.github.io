07. Elements & Tags: Opening, Closing Tags, and Content

Most HTML elements are defined using a pair of tags: an opening tag and a closing tag.

Opening Tag:

  • Marks the beginning of an element.
  • Syntax: <tagname>
  • Example: <p> (starts a paragraph element)

Closing Tag:

  • Marks the end of an element.
  • Syntax: </tagname> (Note the crucial forward slash / before the tag name)
  • Example: </p> (ends a paragraph element)

Content:

  • The actual information (text, or even other HTML elements) that the element contains or represents goes between the opening and closing tags.
  • Example: If you want a paragraph containing the text "Hello World", the HTML would look like this: <p>Hello World</p>
    • <p> is the opening tag.
    • Hello World is the content.
    • </p> is the closing tag.

Requirement:

Most HTML elements that act as containers for content require both an opening and a closing tag. Forgetting the closing tag (or the slash in it) is a common beginner mistake that can lead to unexpected rendering issues.
