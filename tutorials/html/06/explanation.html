06. HTML: Structure & Content (The Nouns)

Files Modified: None

Let's focus on HTML. Its primary role is to give semantic meaning and structure to the raw content of a web page.

Imagine you have a block of text. Is it a heading? A paragraph? Part of a list? HTML provides tags (like <h1>, <p>, <li>) to wrap around your content and define what it is.

  • HyperText: Refers to the ability to link documents together (using the <a> tag, which we'll see later).
  • Markup Language: Means it uses codes (tags) to annotate or "mark up" the text, describing its structure and type, rather than providing programming instructions.

HTML forms the essential backbone upon which CSS and JavaScript build. Without HTML, there's no content structure for CSS to style or JavaScript to manipulate.
