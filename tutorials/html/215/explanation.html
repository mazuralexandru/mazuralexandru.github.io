215. text-align Example

Files Modified: style.css

Let's center our main heading using text-align.

✅ Observe style.css:

  • We've added the text-align: center; declaration to the #main-heading ruleset.
#main-heading {
    color: cornflowerblue; 
    text-decoration: underline; 
    font-family: "Georgia", Times, serif; 
    font-style: italic; 
    text-align: center; /* Center the heading */
}

✨ Observe the Preview: The "Main Heading" text should now be horizontally centered within the available width of the page (or its container, though it currently spans the body width). The paragraphs remain left-aligned by default.
