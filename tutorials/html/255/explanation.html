255. <figcaption> Accessibility & Relation to alt

The primary benefit of using <figure> and <figcaption> together is semantic structure, which directly translates to better accessibility.

Accessibility:

  • Screen Reader Association: Screen readers typically recognize the <figure>/<figcaption> relationship. When navigating, they might announce something like: "Figure, [Image Alt Text], Caption: [Figcaption Text]".
  • Context: This provides crucial context for users who cannot see the image. They get both the description of the image itself (from alt) and the author-provided caption or explanation (from <figcaption>).
  • Structure: Helps users understand that the image and its caption form a single, related unit distinct from the main text flow.

<figcaption> vs. alt Text:

It's vital to understand their distinct roles:

  • alt attribute (on <img>):
    • Describes the image content itself. Answers: "What does this image show?" or "What is the function of this image (if interactive)?"
    • Crucial fallback if the image fails to load.
    • Primary accessibility description of the image visual.
  • <figcaption> element:
    • Describes the figure as a whole. Answers: "What is this figure about?", "What is its title or legend?", "What context does it provide?"
    • Provides additional context or explanation beyond what the image visually shows.
    • Is visible to all users by default.

They are NOT interchangeable:

  • A <figcaption> does not replace the need for meaningful alt text on the image within the figure (unless the image is purely decorative (alt="") and the <figcaption> fully conveys all necessary information).
  • Good alt text describes the visual; good <figcaption> explains the figure's role or context.

Example Distinction:

<figure>
  <img src="sales-chart.png" alt="Bar chart showing Q1: $10k, Q2: $15k, Q3: $12k, Q4: $20k." width="400" height="250">
  <figcaption>Figure 1: Quarterly Sales Revenue. Note the significant increase in Q4 after the marketing campaign launch.</figcaption>
</figure>
  • alt: Describes what the chart visually shows (the data points).
  • <figcaption>: Provides a title and interpretation/context for the chart.

✅ Observation: No code changes. Review the example in index.html. The alt text describes the placeholder image's appearance. The <figcaption> provides a label ("Fig. 1") and context for the figure. They serve different roles.

✨ Key Takeaway: Use both alt (on <img>) and <figcaption> (inside <figure>) appropriately. alt describes the image; <figcaption> describes/captions the figure as a whole. This combination provides the best accessibility.
