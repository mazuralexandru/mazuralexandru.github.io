420. Styling <figure>: Borders, Backgrounds, Padding

Files Modified: style.css

You can apply standard box model properties like border, background-color, and padding to the <figure> element to visually group the image and its caption.

✅ Observe style.css:

  • We've added padding, border, and background-color to the figure rule.
figure {
  margin: 0; 
  margin-bottom: 1.5em; 
  max-width: 500px; 
  padding: 1em; /* Add padding around image and caption */
  border: 1px solid #555; /* Add a border */
  background-color: rgba(255, 255, 255, 0.03); /* Slight background tint */
}

/* ... rest of styles ... */

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview: The <figure> element (containing the image and caption) should now have:

  • A faint background color, distinct from the main page background.
  • A 1px solid gray border around it.
  • 1em of padding between the border and the content (the image and the caption).
