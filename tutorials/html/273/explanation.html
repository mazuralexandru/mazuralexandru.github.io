273. Minimum and Maximum Dimensions: max-height

Files Modified: style.css

Finally, max-height prevents an element from growing too tall.

  • Purpose: Sets the maximum height an element can expand to. If the content or a calculated height would normally make the element taller, its height will be capped at the max-height. Content that exceeds this height will overflow (controlled by the overflow property).
  • Syntax: max-height: value;
  • Values: Length units (px, em, rem, vh), percentages (% - relative to container height, subject to parent-height constraints), or keywords like none (default, no maximum), min-content, max-content, fit-content.
  • Interaction: Overrides height if height calculates to a value greater than max-height. min-height still takes precedence over max-height.

✅ Observe style.css:

  • We've added max-height: 200px; to the .box rule.
  • We've also added overflow: auto; so we can see the effect if content exceeds the max-height.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; /* Prevent box from growing taller than 200px */
  overflow: auto; /* Add scrollbars if content overflows */
  background-color: cornflowerblue; 
  padding: 10px; 
  margin: 1em auto; 
}

✨ Observe the Preview:

  • The box still respects the min-height of 150px.
  • However, if the content (including padding) naturally requires more than 200px vertically, the box's height will be capped at 200px (plus padding/border).
  • Because we added overflow: auto;, a vertical scrollbar should appear inside the box if the content overflows the 200px maximum height. Without overflow: auto;, the content would simply be clipped or spill out (depending on the default overflow: visible;).
