Working with External CSS Files

  • You simply write your CSS rules directly into the .css file.
  • The browser reads this file because it's linked in the index.html via the <link> tag.
  • Any changes saved in style.css will be reflected in index.html (and any other HTML page linking to the same CSS file) upon refreshing the browser preview.

Linking Multiple Stylesheets

  • You can link multiple external CSS files to a single HTML page by adding multiple <link rel="stylesheet" href="..."> tags in the <head>.
    <link rel="stylesheet" href="reset.css">
<link rel="stylesheet" href="layout.css">
<link rel="stylesheet" href="components.css">
<link rel="stylesheet" href="theme.css">
  • Order Matters: If rules in different files target the same elements with the same specificity, the rule from the stylesheet linked later in the HTML will override the earlier ones. This is part of the CSS Cascade (which we'll cover soon).
