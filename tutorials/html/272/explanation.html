272. Minimum and Maximum Dimensions: min-height

Files Modified: style.css

The min-height property works analogously to min-width, but for the vertical dimension.

  • Purpose: Sets the minimum height an element can shrink to. If the content or a calculated height would normally make the element shorter, its height will be forced to be at least the min-height.
  • Syntax: min-height: value;
  • Values: Length units (px, em, rem, vh), percentages (% - relative to container height, subject to the same parent-height constraints as height), or keywords like auto (default, no minimum), min-content, max-content, fit-content.
  • Interaction: Overrides height and max-height if they conflict.

✅ Observe style.css:

  • We've added min-height: 150px; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; /* Ensure box is at least 150px tall */
  background-color: cornflowerblue; 
  padding: 10px; 
  margin: 1em auto; 
}

✨ Observe the Preview: Even though the text content might not require 150px vertically (especially with the width constraint), the blue box should now have a minimum height of 150px (plus top/bottom padding). If the content did require more than 150px, the box would grow taller because height is auto. min-height just sets a floor.
