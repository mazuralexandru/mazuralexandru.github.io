256. Everything is a Box: Core Concept

Files Modified: None

Welcome to the CSS Box Model! This is one of the most fundamental concepts in CSS layout. The core idea is simple: every HTML element rendered by the browser is treated as a rectangular box.

Think about the elements on the page currently: the heading (<h1>), the paragraphs (<p>), the link (<a>), even the inline elements like <em>, <strong>, and <span>. The browser draws an invisible (usually) rectangular box around each one.

Understanding this "box" nature is crucial because CSS properties primarily work by manipulating the dimensions, spacing, and appearance of these boxes. We control how big the boxes are, how much space is inside them, what kind of border they have, and how much space is around them to push other boxes away.
