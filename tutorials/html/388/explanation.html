388. Grouping Content: <tbody> - Fundamentals

Complementing <thead>, the <tbody> element groups the main data content of the table.

The Table Body Element (<tbody>)

  • Definition: Represents the block of data rows in an HTML table.
  • Purpose: To semantically encapsulate the primary content or data records of the table, distinct from the header (<thead>) and optional footer (<tfoot>) rows.
  • Core Concept: It acts as a container for the <tr> elements that hold the main table data (usually using <td> cells, but potentially <th> for row headers).

Placement & Structure:

  • Must be a direct child of the <table> element.
  • Must appear after any <caption>, <thead>, or <tfoot> elements in the HTML source code.
  • A table can have one or more <tbody> elements. Multiple <tbody> elements can be used to divide the table data into logical sections, which can then be styled or manipulated independently.
  • Content Model: Must contain one or more <tr> elements.
  • Implicit Behavior: If a table contains <tr> elements but lacks explicit <thead>, <tbody>, or <tfoot> tags, the browser implicitly wraps all those <tr>s in a <tbody>. However, if <thead> or <tfoot> are present, an explicit <tbody> must also be included for valid HTML.

Code Change:

We've wrapped the two existing data rows (<tr> containing <td>s) inside <tbody>...</tbody> tags, placing it after the <thead>.

<!-- index.html snippet -->
  <table>
    <thead>
      <tr>
        <th scope="col">Header 1</th>
        <th scope="col">Header 2</th>
      </tr>
    </thead>
    <tbody> <!-- Added tbody -->
      <tr> <!-- Data Row 1 -->
        <td>Data A1</td>
        <td>Data B1</td>
      </tr>
      <tr> <!-- Data Row 2 -->
        <td>Data A2</td>
        <td>Data B2</td>
      </tr>
    </tbody> <!-- Closed tbody -->
    <!-- tfoot could follow -->
  </table>

✅ Observation:

  1. Locate the <tbody> tags wrapping the two data rows, placed after the closing </thead> tag in index.html.
  2. Look at the browser preview. Again, there is likely no significant visual change by default from adding <tbody>. Its primary role is semantic structure, providing hooks for styling (e.g., tbody tr:nth-child(even) for zebra striping) and enabling features like body scrolling.

✨ Key Takeaway: Use the <tbody> element to wrap the main data row(s) (<tr> elements) of your table. It must appear after <thead> or <tfoot> (if present). Explicitly adding <tbody> is required for validity if <thead> or <tfoot> are used.
