254. Figure Captions (<figcaption>) - Syntax & Example

Let's add a <figcaption> to our <figure>.

Syntax Recap:

  • Uses opening <figcaption> and closing </figcaption> tags.
  • Contains the caption text (and potentially other flow content like links or inline formatting).
  • Placed as the first or last child inside the <figure>.

Code Change:

We've added a <figcaption> element as the last child within the existing <figure> in index.html.

<body>
  <h1>Figures and Captions</h1>
  <p>A figure element groups self-contained content, like an image:</p>

  <figure>
    <img src="https://placehold.co/300x200/87CEFA/FFFFFF.png?text=Figure+Content"
         alt="Placeholder image for a figure: 300x200, sky blue background, white text."
         width="300"
         height="200">
    <figcaption>Fig. 1: A placeholder image demonstrating figure and caption.</figcaption>
    <!-- Added figcaption as last child -->
  </figure>

  <p>The figcaption provides a visible label for the figure.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

✅ Observation:

  1. Locate the new <figcaption> element nested inside the <figure>, appearing after the <img>.
  2. Look at the browser preview. You should now see the caption text ("Fig. 1: A placeholder image...") displayed below the placeholder image.
  3. Notice that the caption text is visually associated with the image, contained within the logical <figure> block. Default browser styling might apply some spacing or slightly different text formatting to the caption.

✨ Key Takeaway: Add a <figcaption> element inside your <figure> (as the first or last child) to provide a visible, semantically linked caption for the figure's content.
