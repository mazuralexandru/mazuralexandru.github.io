299. Longhand Margin Properties: margin-bottom

Files Modified: style.css

Now for margin-bottom.

  • Purpose: Sets the amount of margin space specifically on the bottom edge of the element, below its bottom border.
  • Syntax: margin-bottom: value;
  • Values: Same as margin-top (lengths, percentages relative to container width, auto, negative values allowed).

✅ Observe style.css:

  • We've added margin-bottom: 3em; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin-top: 2em; 
  margin-right: 50px; 
  margin-bottom: 3em; /* Added specific bottom margin */
  border-bottom: 4px solid red; 
  border-radius: 15px; 
}
