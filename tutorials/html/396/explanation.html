396. Spanning Columns: colspan Example

Let's apply colspan to create a header that spans multiple columns.

Scenario: Imagine the first column is a label, and the next two columns represent related data (e.g., "Min" and "Max" values under a general "Range" header).

Code Change:

We will modify the header row (<thead><tr>) in index.html.

  1. Add a <th> for the row labels column (empty for the header row).
  2. Add a <th> with colspan="2" to act as a main header spanning the next two columns.
  3. Add a new second header row containing the specific sub-headers for the spanned columns.
  4. Adjust the data rows to match the new 3-column data structure (row header + 2 data points).
<!-- index.html snippet -->
<body>
  <h1>Spanning Columns & Rows</h1>
  <p>Using 'colspan' on a header cell (th):</p> <!-- Adjusted P -->

  <table>
    <thead>
      <tr>
        <th scope="col"></th> <!-- Row Header Column -->
        <th scope="col" colspan="2">Measurement Range</th> <!-- Header spanning 2 cols -->
        <!-- No third th needed here because of colspan="2" -->
      </tr>
      <tr>
        <th scope="col"></th> <!-- Empty cell aligned with Row Header Column -->
        <th scope="col">Min Value</th> <!-- Sub-header 1 -->
        <th scope="col">Max Value</th> <!-- Sub-header 2 -->
      </tr>
    </thead>
    <tbody>
      <tr>
        <th scope="row">Sensor A</th> <!-- Row header -->
        <td>10</td> <!-- Data for Min -->
        <td>100</td> <!-- Data for Max -->
      </tr>
      <tr>
        <th scope="row">Sensor B</th> <!-- Row header -->
        <td>15</td> <!-- Data for Min -->
        <td>120</td> <!-- Data for Max -->
      </tr>
    </tbody>
  </table>

  <p>The "Measurement Range" header spans two columns.</p> <!-- Adjusted P -->

  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

(Note: We added scope="row" to the new row headers for correctness).

✅ Observation:

  1. Examine the <thead> structure carefully. See the first <tr> with the colspan="2" header. Note that this row only has two <th> elements in total. See the second <tr> within <thead> providing the sub-headers.
  2. Look at the browser preview:
    • The "Measurement Range" header visually spans across the horizontal space above both the "Min Value" and "Max Value" columns.
    • The table correctly renders as a grid with effectively three main columns (Row Header, Min Value, Max Value).
    • The colspan attribute created the desired merged header cell.

✨ Key Takeaway: Apply colspan="number" to a <th> or <td> to make it span number columns. Ensure the total number of effective columns represented by the cells in that row matches the table structure.
