295. Rounded Corners (border-radius): Longhand Properties

Files Modified: None

If you need to control the radius of each corner individually, you can use the longhand properties:

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius

Each of these properties accepts either:

  • One value: Sets both the horizontal and vertical radius for that corner (circular).
    • Example: border-top-left-radius: 20px;
  • Two values: Sets the horizontal radius first, then the vertical radius for that corner (elliptical).
    • Example: border-top-left-radius: 20px 40px;

Using longhand properties gives you the most granular control but is more verbose than the shorthand when multiple corners share the same radius values.
