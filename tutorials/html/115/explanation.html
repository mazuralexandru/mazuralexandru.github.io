115. File Extension Recap

Files Modified: None

Using the correct file extension is crucial for both your operating system and web browsers/servers to understand what type of file they are dealing with.

  • .html (or .htm): Used for files containing HyperText Markup Language. This tells the browser to parse and render the file as a web page. Our main structure and content file is index.html.
  • .css: Used for files containing Cascading Style Sheets. This tells the browser (when linked via <link>) to parse the file for styling rules. Our style files are base_styles.css and style.css.
  • .js: Used for files containing JavaScript code. (We haven't used script.js yet, but it would have this extension).

Always use the appropriate extension when saving your files. Using the wrong extension (e.g., saving HTML as .txt or CSS as .html) will prevent things from working correctly.
