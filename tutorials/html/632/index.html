HTML Entities: Escaping Reserved Characters

Reason 1: Escaping characters with special meaning in HTML.

Displaying Less Than (<)

To show the less than sign, use &lt; like this: 5 < 10.

Displaying Greater Than (>)

To show the greater than sign, use &gt; like this: 10 > 5.

Crucial for displaying code like <p> (using &lt;p&gt;).
