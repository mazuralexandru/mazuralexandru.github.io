314. Universal border-box Reset (Common Practice)

Files Modified: style.css

Because border-box is so much more intuitive for layout, it's standard practice in modern CSS to apply it to all elements on the page using a universal reset.

The most common and recommended way to do this involves inheritance:

html {
  box-sizing: border-box; /* 1. Set the base */
}

*, *::before, *::after { /* 2. Select all elements and pseudo-elements */
  box-sizing: inherit; /* 3. Make them inherit from their parent */
}

/* Rest of your CSS... */
.some-element {
  /* Now uses border-box sizing by default */
}

Explanation:

  1. We set box-sizing: border-box; directly on the root <html> element.
  2. We use the universal selector (*), plus selectors for common pseudo-elements (*::before, *::after), to target everything.
  3. We set box-sizing: inherit; on these elements. This tells them to inherit the box-sizing value from their parent. Since all elements ultimately descend from <html>, they will inherit the border-box value set in step 1.

Why this approach?

  • It applies border-box globally and consistently.
  • Using inherit makes it slightly easier to override box-sizing back to content-box for specific components if ever needed (though this is rare), compared to directly setting border-box on *.
