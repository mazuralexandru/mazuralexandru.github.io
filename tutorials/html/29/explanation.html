29. Using ID Selectors in CSS

With the <div id="main-content"> element in place, we can now target it using an ID Selector in our style.css.

What's New?

  • CSS Pruning: We've commented out the previous .highlight and .important class rules in style.css as those elements were removed from the HTML for focus.
  • CSS Addition: We added a new CSS ruleset using the ID selector #main-content.
/* style.css snippet */
/* Removed Class Selector rules for focus */
/*
.highlight {
  background-color: rgba(255, 255, 0, 0.2);
  border: 1px solid yellow;
  padding: 5px;
}
.important {
  font-weight: bold;
  color: tomato;
}
*/

/* Added ID Selector rule */
#main-content {
  border: 2px dashed cyan;
  padding: 10px;
  margin-top: 20px; /* Add some space above this div */
}

/* CSS files can also have comments! */

ID Selectors Explained

  • Syntax: A hash symbol (#) followed immediately by the ID name (e.g., #main-content, #primary-navigation). No space between the hash and the ID name.
  • Scope: Selects the single element in the document that has the matching id attribute. Because IDs must be unique, this selector will target at most one element.
  • Specificity: ID selectors have high specificity. They are more specific than class selectors, attribute selectors, and type selectors. This means a rule using an ID selector will generally override conflicting rules using those other selector types.
  • Use Cases & Caution:
    • Good for targeting unique page landmarks (header, footer, main content area, navigation) for layout or specific styling.
    • Often used as reliable hooks for JavaScript (document.getElementById()).
    • Avoid using IDs for general styling. Because they are unique and highly specific, they create styles that are not reusable and can make overriding styles later more difficult (leading to "specificity wars"). Prefer using classes for reusable styling components. Use IDs sparingly for CSS styling, primarily for unique structural elements or JS interaction points.

In our example, the rule #main-content { ... } applies a dashed cyan border, some padding, and top margin specifically to the <div> with id="main-content".
