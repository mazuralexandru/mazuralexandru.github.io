24. Introduction to Browser Developer Tools

Files Modified: None

Every modern web browser includes a powerful suite of Developer Tools (often called "DevTools"). These are indispensable for web development.

Accessing DevTools:

  • Keyboard Shortcut: Press F12 (on most Windows/Linux machines) or Option+Command+I (on macOS).
  • Right-Click Menu: Right-click anywhere on a webpage and select "Inspect" or "Inspect Element".
  • Browser Menu: Usually found under "More Tools" -> "Developer Tools" or similar.

Key Panels (Overview):

  • Elements (or Inspector): View the live HTML structure (DOM), inspect and modify CSS styles applied to elements.
  • Console: View messages logged by JavaScript, run JavaScript commands, see errors and warnings.
  • Sources (or Debugger): View loaded files (HTML, CSS, JS), set breakpoints to debug JavaScript.
  • Network: Analyze network requests (loading of files, API calls).
  • Application (or Storage): Inspect cookies, local storage, session storage.
  • Performance (or Profiler): Analyze runtime performance and identify bottlenecks.

We'll explore these tools, especially the Elements panel, as we start building.
