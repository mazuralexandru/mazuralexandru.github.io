330. Ordered Lists (<ol>)

Files Modified: None

The second main type of list is the ordered list, created using the <ol> element.

  • Purpose: Represents a list of items where the order is significant. Think of steps in a recipe, ranking items, or outlining sections in a document. Changing the order would change the meaning or outcome.
  • Structure: Similar to <ul>, the <ol> element acts as a container, and the actual list items are placed inside using <li> elements.
  • Default Styling: Browsers typically render ordered lists with sequential markers (usually numbers: 1, 2, 3...) preceding each list item, along with default indentation.
  • Attributes: <ol> has specific attributes to control the numbering:
    • type: Specifies the marker type ("1" (default), "a", "A", "i", "I").
    • start: Specifies the starting number/letter of the list (e.g., start="5").
    • reversed: A boolean attribute that reverses the numbering order (e.g., 3, 2, 1).
  • <li> value Attribute: Individual list items within an <ol> can have a value attribute to manually set their number, affecting the numbering of subsequent items.
