418. Styling <figure>: Default Margins

Files Modified: style.css

Browsers typically add default vertical margins (and sometimes horizontal indentation via margin) to <figure> elements. It's common practice to reset these if you want precise control over spacing.

✅ Observe style.css:

  • We've added a rule for figure to reset its default margins.
/* Styling Figure Elements */
figure {
  margin: 0; /* Reset browser default margins */
  margin-bottom: 1.5em; /* Add some space below the figure */
  /* Other styles will be added */
}

/* Add styles for the image inside the figure */
figure img {
    display: block; /* Remove extra space below image */
    max-width: 100%; /* Make image responsive */
    height: auto;
}

/* Styles for figcaption will be added */
  • We also added margin-bottom to create space below the figure and basic responsive styles for the image within it.

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview: The figure might shift slightly as the default browser margins are removed and replaced by only our explicit margin-bottom. The primary effect of margin: 0; is to remove any unwanted default spacing around the figure, giving us a clean slate.
