We've encountered several CSS units for properties like font-size, padding, and margin-top. Let's formally revisit the main categories of units used in CSS. Understanding the difference, especially between absolute and relative units, is crucial for creating scalable and responsive designs.

We'll focus on:

  • Absolute Units (px)
  • Relative Units (%, em, rem, Viewport Units)
