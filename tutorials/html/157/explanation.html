157. href URL Types: Relative URL (Same Directory)

Let's look at the simplest form of a relative URL: linking to a file located in the same directory (folder) as the current HTML document.

Syntax:

To link to a file (e.g., about.html) that resides in the exact same folder as the current page (let's assume our current page is index.html), you simply use the filename directly in the href attribute:

  • href="about.html"

You might also sometimes see ./ preceding the filename (href="./about.html"). The ./ explicitly means "start in the current directory," but it's usually optional for files in the same directory; just the filename typically works.

How it Resolves:

If the current page's URL is, for example, http://www.mysite.com/products/index.html, and it contains the link <a href="specifications.html">, the browser resolves this to http://www.mysite.com/products/specifications.html. It takes the directory path of the current page and appends the relative filename.

Code Change:

We've added a link to index.html using a same-directory relative URL. Assume our current file is index.html and we want to link to about.html in the same folder.

<body>
  <h1>Links and Navigation</h1>
  <p>Relative URL: Linking to a file in the same directory.</p>

  <p>Imagine this page is 'index.html'. This link points to 'about.html' in the same folder:</p>
  <p><a href="about.html">About Us</a></p> <!-- Added relative link (same directory) -->

  <p>The browser combines the current path with the relative path.</p>

</body>

✅ Observation:

  1. Examine the <a> tag in index.html. Note the simple href="about.html".
  2. Look at the browser preview. You see the "About Us" link, styled as a hyperlink.
  3. Behavior: Since we don't actually have an about.html file in our project structure for this tutorial, clicking this link will likely result in a "File Not Found" error in the preview environment. However, the structure of the link itself is correct for linking to a file in the same directory.

✨ Key Takeaway: To link to a file in the same directory using a relative URL, simply use the target filename in the href attribute (e.g., href="filename.html").
