586. Introduction to Module 12: Graphics

Welcome to Module 12! So far, we've learned to structure content with text, lists, tables, and semantic sectioning elements; link resources with <a>; embed static images with <img> and time-based media with <video> and <audio>; handle metadata in the <head>; use global attributes; and collect user input with forms.

This module focuses on Graphics in HTML. While <img> is perfect for displaying existing image files, HTML offers two powerful native technologies for creating graphics directly within the browser:

  1. The <canvas> Element: Provides a scriptable bitmap drawing surface.
  2. Scalable Vector Graphics (SVG): An XML-based language for describing vector graphics, which can be embedded directly or linked.

We'll explore the fundamentals, use cases, advantages, and disadvantages of both approaches.
