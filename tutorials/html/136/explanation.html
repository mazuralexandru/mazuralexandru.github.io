136. <link> Syntax

Files Modified: None

The basic syntax for linking an external stylesheet is:

<link rel="stylesheet" href="styles.css">

Let's break this down:

  • <link: The element tag itself. It's a void element, so no closing </link> tag is needed.
  • rel="stylesheet": The relationship attribute, set to stylesheet to indicate a CSS file. This is mandatory for linking stylesheets.
  • href="styles.css": The hypertext reference attribute, providing the path to the .css file. The path can be:
    • Relative: (Most common for project files) Relative to the location of the current HTML file.
      • href="style.css" (CSS file in the same directory as the HTML file)
      • href="css/theme.css" (CSS file in a subdirectory named css)
      • href="../shared/common.css" (CSS file in a different directory up one level)
    • Absolute: A full URL, typically used for linking to stylesheets hosted on other domains or CDNs (Content Delivery Networks).
      • href="https://cdn.example.com/library.css"
      • href="/assets/main.css" (Root-relative path, starts from the website's root directory)

Getting the href path correct is essential for the browser to find and load the styles.
