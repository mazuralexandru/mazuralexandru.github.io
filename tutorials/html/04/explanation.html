04. Role of Web Browsers (Rendering Engines)

Files Modified: None

Web browsers (Chrome, Firefox, Safari, Edge) are essential software that act as your window to the web. Their primary job is to:

  1. Fetch resources (HTML, CSS, JavaScript, images) from servers using HTTP/HTTPS.
  2. Interpret and render these resources into a visual web page you can interact with.

The core component responsible for this interpretation and rendering is the Rendering Engine (also called the layout engine or browser engine). Different browsers use different engines:

  • Blink: Used by Chrome, Edge, Opera, Brave. (Originally a fork of WebKit).
  • Gecko: Used by Firefox.
  • WebKit: Used by Safari. (Also the basis for Blink).

These engines parse the HTML to build a structure (the DOM tree), parse the CSS to determine styles, and execute JavaScript for interactivity. They then combine this information to paint the final pixels on your screen.

Minor differences between engines can sometimes lead to websites appearing slightly differently across browsers, which is a challenge web developers manage.
