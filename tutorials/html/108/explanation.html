108. Ordered Lists (<ol>): The type Attribute

While the default marker for <ol> is decimal numbers (1, 2, 3...), HTML provides the type attribute to change this marker style.

The type Attribute:

  • Purpose: Specifies the kind of sequential marker to use for the list items.
  • Syntax: <ol type="value">
  • Common Values:
    • 1: Decimal numbers (1, 2, 3...) - This is the default.
    • a: Lowercase letters (a, b, c...).
    • A: Uppercase letters (A, B, C...).
    • i: Lowercase Roman numerals (i, ii, iii...).
    • I: Uppercase Roman numerals (I, II, III...).

Important Consideration: Styling vs. Semantics

  • Using the type attribute directly mixes presentational concerns (how the marker looks) with the structural markup (HTML).
  • Best Practice: For purely visual styling of markers, it's strongly recommended to use the CSS list-style-type property instead. This keeps presentation rules separate in your CSS file.
  • When to Use type?: You might use the HTML type attribute if the specific numbering style (e.g., using Roman numerals for top-level sections in a legal document) has inherent semantic meaning in that context, beyond just visual appearance. However, even then, CSS is often preferred for consistency and control.
