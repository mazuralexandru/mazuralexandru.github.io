86. CSS Declarations (property: value;)

Files Modified: None

A declaration is a single CSS rule consisting of a property and a value, separated by a colon (:) and ending with a semicolon (;).

property: value;

  • Property: The specific aspect of the element you want to style (e.g., color, font-size, margin, border, background-color). CSS defines hundreds of available properties.
  • Value: The setting or value you want to apply to that property (e.g., blue, 1.2em, 10px, solid 1px black, #333). Each property accepts specific types of values (keywords, lengths, colors, etc.).
  • Colon (:): Separates the property from its value.
  • Semicolon (;): Marks the end of the declaration. This is crucial! Forgetting the semicolon can invalidate the current declaration and potentially the next one too.

Example declarations:

  • color: red;
  • font-size: 16px;
  • background-color: lightgray;
  • padding: 10px 20px;
