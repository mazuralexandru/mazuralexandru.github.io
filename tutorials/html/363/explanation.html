363. Link Attribute: download

Files Modified: index.html

The download attribute instructs the browser to download the linked URL instead of navigating to it.

  • Purpose: To trigger a file download prompt when the user clicks the link.
  • Syntax:
    • <a href="path/to/file.pdf" download>Download PDF</a> (Boolean attribute - browser suggests filename based on URL).
    • <a href="path/to/image.jpg" download="my-vacation-photo.jpg">Download Image</a> (With value - suggests a specific filename for the download).
  • Behavior: When clicked, the browser should initiate a download of the resource specified in the href attribute, using the optional value of the download attribute as the suggested filename.
  • Same-Origin Policy: This attribute generally only works reliably for same-origin URLs (files hosted on the same domain as the HTML page). For cross-origin URLs, browser behavior might vary due to security policies (it might navigate instead of downloading).
