353. ::marker: Stylable Properties & Use Cases

Files Modified: style.css

While not all CSS properties can be applied to the ::marker pseudo-element, several useful ones are typically supported:

  • color: Changes the color of the marker text or symbol.
  • font-size, font-weight, font-family, etc.: Allows changing the font characteristics of the marker.
  • content: Allows you to replace the default marker with custom text or symbols (including emojis or Unicode characters). Use content: ""; to effectively hide the marker.
  • Animation/Transition Properties: Can often be applied for effects.

Properties related to the box model (padding, margin, border, background) generally cannot be applied directly to the ::marker itself.

Use Cases:

  • Changing the color of bullets or numbers.
  • Using custom symbols (like arrows →, checkmarks ✓, or emojis ✅) instead of standard markers via the content property.
  • Adjusting the size or weight of the marker independently from the list item text.
