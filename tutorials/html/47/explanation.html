47. DOCTYPE: Historical Context

Files Modified: None

The simplicity of <!DOCTYPE html> is a welcome feature of HTML5. It's worth knowing that previous versions of HTML and XHTML had much more complex DOCTYPE declarations.

For example:

  • HTML 4.01 Strict:
    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
  • XHTML 1.0 Transitional:
    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

These older DOCTYPEs included references to Document Type Definitions (DTDs), which were formal specifications defining the allowed elements and structure for that specific HTML/XHTML version. They were long, hard to remember, and easy to get wrong.

HTML5 simplified this dramatically. <!DOCTYPE html> is sufficient to identify the document as HTML and trigger Standards Mode in all modern browsers. You generally don't need to worry about the older, complex DOCTYPEs unless you're working on very old legacy projects.
