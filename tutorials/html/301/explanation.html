301. Longhand Margin Properties: Values

Files Modified: None

Let's recap the values accepted by the individual margin properties (margin-top, margin-right, margin-bottom, margin-left):

  1. Length Units:

    • px, em, rem, etc. Define fixed or relative spacing. Negative values are allowed (e.g., margin-top: -10px; pulls the element up).

  2. Percentages (%):

    • Calculated relative to the width of the element's containing block, even for margin-top and margin-bottom.
    • Example: margin-left: 5%;

  3. auto:

    • The browser calculates the margin.
    • Most significantly, when applied to horizontal margins (margin-left: auto; margin-right: auto;) on a block element with an explicit width, it centers the element horizontally within its container.
    • On its own (e.g., just margin-left: auto;), it typically pushes the element as far as possible in the opposite direction (e.g., margin-left: auto; pushes the element to the right).
    • For vertical margins, auto usually resolves to 0.

The default value for all margin properties is 0.
