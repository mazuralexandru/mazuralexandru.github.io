324. Metadata: Author (name="author") - Fundamentals

Another standard metadata name is author, used to specify the author of the HTML document.

Purpose:

  • The <meta name="author"> tag is used to formally state the name of the person or organization responsible for creating the content of the web page.

Core Concept:

  • Provides a simple way to embed authorship information directly into the document's metadata layer.

Attribute Pairing:

  • Uses the name/content pair.
    • name="author": Identifies the metadata type as the author.
    • content="Author Name or Organization Name": Contains the name string.

Semantic Value:

  • Provides clear, machine-readable attribution within the document itself.

✅ Observation: No code changes yet. This step defines the author meta tag's purpose: specifying the document creator's name.

✨ Key Takeaway: Use <meta name="author" content="..."> to specify the name of the document's author or creator in the metadata.
