Background Attachment: Local

This box has a fixed height and overflow: scroll.

The background-attachment is set to 'local'.

Therefore, as you scroll the content inside this box...

...the background pattern should scroll along with the text.

Try changing background-attachment to 'scroll' in DevTools.

With 'scroll', the background stays fixed relative to the box itself.
