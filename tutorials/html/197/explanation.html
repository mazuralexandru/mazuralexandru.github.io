197. Telephone Links (tel:): Introduction

Similar to mailto:, the tel: URI scheme allows you to create links that interact with other applications – in this case, telephony applications for making phone calls.

The tel: Scheme:

  • Definition: A URI scheme specifically designed for telephone numbers.
  • Purpose: When used in an href attribute (href="tel:..."), it creates a "click-to-call" link. Activating the link instructs the browser or operating system to initiate a phone call using a capable device or application.
  • Target Devices: Primarily effective on:
    • Smartphones (opens the default phone dialer app).
    • Tablets with cellular capabilities.
    • Desktop computers with installed telephony software (like Skype, FaceTime, or VoIP apps) configured to handle tel: links.

Goal:

To provide users (especially on mobile devices) a very convenient way to initiate a call to a specific phone number directly from a web page without manual dialing.
