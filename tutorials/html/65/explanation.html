65. Text Styling: text-indent

The text-indent property allows you to indent the very first line of text within a block container.

text-indent Explained

  • Purpose: Specifies the indentation applied to the first line of text in a block container.
  • Applies To: Block container elements (<p>, <div>, etc.).
  • Values:
    • <length>: px, em, rem, etc. Specifies a fixed indentation amount. em units are often good here as they scale with the font size.
    • <percentage>: Calculates the indentation based on the width of the containing block.
    • hanging (Keyword, experimental): Inverts the indentation (creates a hanging indent).
    • each-line (Keyword, experimental): Indents every line (not just the first).
  • Negative Values: Can be used to create "outdents" or hanging indents, where the first line starts to the left of the rest of the block (often requires adjusting padding/margin on the container).
  • Inheritance: Yes, text-indent is inherited.
