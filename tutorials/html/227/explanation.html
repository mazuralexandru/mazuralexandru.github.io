227. src URL Types: Absolute URL Example

Let's modify our example to use an absolute URL for the src attribute.

Use Case: Linking to an image hosted on a different domain or a Content Delivery Network (CDN).

Code Change:

We've changed the src value in index.html to an absolute URL pointing to a placeholder image service (placehold.co). We still lack the required alt attribute.

<body>
  <h1>Embedding Images</h1>
  <p>Using an absolute URL for the image source:</p> <!-- Adjusted P -->

  <img src="https://placehold.co/100x50/E0E0E0/2e2e2e.png?text=IMG"> <!-- Absolute URL src (still INVALID without alt) -->

  <p>Absolute URLs point to external resources (like CDNs).</p> <!-- Adjusted P -->


  <!-- <p>This paragraph is temporarily commented out.</p> -->
</body>

Breakdown of the URL:

  • https://placehold.co/100x50/E0E0E0/2e2e2e.png?text=IMG
    • Scheme: https
    • Domain: placehold.co
    • Path: /100x50/E0E0E0/2e2e2e.png (specifies size, colors)
    • Query: ?text=IMG (adds text to the placeholder)

✅ Observation:

  1. Examine the <img> tag and its src attribute containing the full absolute URL.
  2. Look at the browser preview. This time, you should actually see an image! The browser fetched the placeholder image from the external placehold.co server. It should be a small (100x50 pixels) light grey rectangle with the dark text "IMG" inside.
  3. This demonstrates that when src points to a valid, accessible external resource, the browser can display it. (The image is still technically invalid HTML due to the missing alt attribute).

✨ Key Takeaway: Absolute URLs in src allow you to embed images hosted on external servers or CDNs. Ensure the URL is correct and the resource is accessible.
