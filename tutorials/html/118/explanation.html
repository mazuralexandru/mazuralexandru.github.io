118. Viewing Local Files (file:///)

Files Modified: None

How do you view your .html file in a browser when it's just saved on your computer (not on a web server)?

  1. Double-Click: In your operating system's file explorer, simply double-clicking the .html file will usually open it in your default web browser.
  2. Drag and Drop: You can often drag the .html file icon from your file explorer directly onto an open browser window or tab.
  3. Browser's "Open File" Menu: Most browsers have an option in their File menu (e.g., File > Open File... or Ctrl+O/Cmd+O) that lets you navigate to and select your .html file.

When you open a local file this way, you'll notice the URL in the browser's address bar starts with file:/// followed by the path to the file on your computer's hard drive.

Example URL: file:///C:/Users/YourName/Documents/MyWebsite/index.html

This indicates you are viewing a local resource directly, not one fetched from a web server via HTTP. This is perfectly fine for simple HTML/CSS/JS testing during development.
