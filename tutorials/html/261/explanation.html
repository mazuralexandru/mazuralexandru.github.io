261. The Four Components of the Box: Margin Area

Files Modified: None

The outermost component of the box model is the Margin Area.

  • Purpose: Margin creates transparent space outside the element's border. It's used to push other elements away, creating space between adjacent boxes.
  • Control: Set using the margin shorthand property or the longhand properties (margin-top, margin-right, margin-bottom, margin-left).
  • Appearance: Margin is always transparent; an element's background color/image does not extend into the margin.
  • Interaction: Margins of adjacent elements can sometimes collapse (vertically), which we'll discuss later.

Margins are fundamental for controlling the spacing and layout between different elements on the page.
