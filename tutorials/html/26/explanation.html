26. HTML Elements and Tags

Files Modified: None

The core components of HTML are elements and tags.

  • Element: The conceptual unit representing a piece of content and its structure. For example, a paragraph element, a heading element, a list element.
  • Tag: The syntactic marker used in the HTML code to define the start and end of an element. Tags are enclosed in angle brackets (< >).

Most HTML elements are defined using two tags:

  1. Opening Tag: Marks the beginning of the element (e.g., <p>).
  2. Closing Tag: Marks the end of the element. It's the same as the opening tag but with a forward slash before the tag name (e.g., </p>).
  3. Content: Everything between the opening and closing tag is the content of the element (e.g., the text inside the paragraph).

So, the basic structure is: <tagname>Content</tagname>.

The combination of the opening tag, the content, and the closing tag forms the complete element.
