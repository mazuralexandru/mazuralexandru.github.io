185. The download Attribute: Introduction

While server headers (like Content-Disposition: attachment) are the most reliable way to force a download, HTML provides the download attribute on the <a> element as a hint to the browser that the linked resource should be downloaded rather than navigated to or displayed inline.

The download Attribute:

  • Definition: A boolean attribute that indicates the resource pointed to by the href attribute is intended to be downloaded by the user.
  • Purpose: To suggest that the browser should trigger a "Save As..." prompt for the linked file, even for file types it might normally display inline (like images or PDFs).
  • Syntax: Can be used as a boolean attribute (presence enables it) or with an optional value specifying a suggested filename.
    • Boolean: <a href="..." download>
    • With Value: <a href="..." download="suggested-filename.ext">
