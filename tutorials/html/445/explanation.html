445. background-attachment

Files Modified: index.html, style.css

The background-attachment property determines whether a background image's position is fixed within the viewport or scrolls along with its containing element.

  • Purpose: To control the scrolling behavior of background images.
  • Syntax: background-attachment: value;
  • Applies To: Each background layer individually (use comma separation).
  • Common Values:
    • scroll (Default): The background scrolls with the element. If the element itself has scrollbars, the background does not scroll with the element's content.
    • fixed: The background is fixed relative to the viewport. It does not move when the page or the element is scrolled. Creates a parallax-like effect. Can have performance implications.
    • local: The background is positioned relative to the element's content. If the element has scrollbars, the background scrolls along with the content inside the element.

body {
  /* Ensure enough height to scroll */
  min-height: 150vh; /* Taller than viewport */

  /* Background properties */
  background-image: url('images/placeholder-landscape.jpg'); /* Use larger image */
  background-repeat: no-repeat;
  background-position: center center;
  background-size: cover;
  background-attachment: scroll; /* Default - background scrolls with page */

  /* Reset base styles we might have removed */
  background-color: #2e2e2e; 
  color: #E0E0E0; 
  font-family: sans-serif; 
  padding: 15px; 
}
