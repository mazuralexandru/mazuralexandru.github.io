338. List Markers (list-style-type): Purpose

Files Modified: None

After potentially resetting list styles, you might want to re-apply or customize the markers (bullets or numbers). The primary property for this is list-style-type.

  • Purpose: Specifies the type of marker to be used for items in a list (<li> elements).
  • Applicability: Applies to elements with display: list-item; (which <li> elements have by default).
  • Inheritance: Yes. If set on a <ul> or <ol>, it's inherited by the child <li> elements.

This property allows you to choose from various standard marker styles or explicitly set it to none to hide markers.
