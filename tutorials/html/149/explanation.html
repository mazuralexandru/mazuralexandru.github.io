149. Anchor Element (<a>): Basic Syntax Structure

Let's write our first (though not yet functional) anchor element.

Basic Syntax:

The anchor element uses an opening tag (<a>) and a closing tag (</a>). The content that the user sees and interacts with goes between these tags.

<a>Link Content Goes Here</a>

Attributes are Key:

While this structure creates the element, an anchor element is only a functional hyperlink when it includes the crucial href attribute to specify the destination. Without href, it's considered a placeholder link.

Code Change:

We've added a very basic <a> element containing text to index.html. Notice that it does not have an href attribute yet.

<p>Here is an anchor element: <a>Empty Anchor</a></p> <!-- Added simple <a> tag -->

✅ Observation:

  1. Locate the <a>Empty Anchor</a> code in index.html.
  2. Look at the browser preview. You'll see the text "Empty Anchor". How does it look?
    • It will likely appear as plain text, indistinguishable from the surrounding paragraph text.
    • It will not have the typical link styling (like blue color and underlining) because it lacks the href attribute that makes it a true hyperlink.
    • It might be focusable if you use the Tab key (depending on browser), but clicking it will do nothing.

✨ Key Takeaway: The basic structure of an anchor element is <a>...</a>. Without an href attribute, it acts as a non-functional placeholder link and typically lacks default link styling.
