454. background-clip: Syntax

Files Modified: None

The syntax is:

background-clip: <box> [, <box>]*;

Where <box> is one of the keywords:

  • border-box (Default)
  • padding-box
  • content-box
  • text (Experimental)

If multiple background images are defined, use a comma-separated list to specify the clipping region for each layer independently.
