95. Inline Styles: Example Recap

Files Modified: None

Let's quickly recap the inline style example we added:

<!-- Inside index.html -->
<p style="color: tomato; font-size: 22px;">This paragraph uses inline styles.</p>
  • We applied styles directly to the <p> tag using the style attribute.
  • The value of the attribute contains standard CSS declarations (color: tomato; and font-size: 22px;).
  • These styles affect only this specific paragraph.

While functional, remember the drawbacks discussed previously. We'll soon move this styling logic into our style.css file.
