55. Text Styling: Using Web Fonts (Google Fonts Example)

Relying only on fonts installed on a user's computer ("web safe fonts") is limiting. We can use Web Fonts to load custom fonts alongside our web page. The easiest way is often using a service like Google Fonts.

What's New?

  • HTML Addition: Added a <link> tag in the <head> to load the "Roboto" font family from Google Fonts. Note there are two link tags now - one for our style.css and one for the Google Font.
  • CSS Modification: Changed the font-family for the .sans-serif-text class to use the newly imported "Roboto" font first in its stack.
<!-- index.html snippet -->
<head>
  <meta charset="UTF-8">
  <title>My First Web Page</title>
  <!-- Link to Google Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet"> <!-- Added Google Font Link -->

  <link rel="stylesheet" href="style.css"> <!-- Our Stylesheet -->
</head>
<body>
  <h1>Font Family Demo</h1>
  <p class="serif-text">This text uses a serif font stack (Times New Roman, Georgia, serif).</p>
  <p class="sans-serif-text">This text uses the Roboto web font (via Google Fonts), then Arial, Helvetica, sans-serif.</p> <!-- Updated text -->

/* Font Family Demo */

Using Google Fonts (or similar services)

  1. Find a Font: Go to the Google Fonts website (fonts.google.com), choose a font (e.g., Roboto), and select the styles/weights you need (e.g., Regular 400, Bold 700).
  2. Get the Code: Google Fonts provides code to embed in your HTML. It usually involves one or more <link> tags that you place in the <head> of your HTML document.
    • The <link rel="preconnect"> tags help the browser establish an early connection to the font servers, potentially improving loading speed.
    • The main <link href="..."> tag points to a special CSS file generated by Google. This file contains the necessary @font-face rules (which we'll discuss briefly later) to define and load the font files (like WOFF2 format) from Google's servers. The URL often includes the font family name (Roboto) and requested weights (wght@400;700). The &display=swap parameter tells the browser to show fallback text immediately while the font loads, then swap it out (good for perceived performance).
  3. Use in CSS: Once the font is linked in the HTML, you can use its specified font-family name (e.g., "Roboto") in your CSS font stacks, just like any other font.

@font-face (Briefly)

The CSS file linked from Google Fonts contains @font-face rules. This is the standard CSS mechanism for defining custom fonts. It allows you to specify a name for the font, point to the font file(s) (e.g., myfont.woff2), and define its weight/style. This allows self-hosting fonts, but using a service like Google Fonts often simplifies the process and leverages optimized font delivery.
