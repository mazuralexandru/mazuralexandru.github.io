168. Linking to Fragments: #top and Use Cases

Let's discuss linking specifically to the top of the page and common use cases for fragment links.

Linking to the Top (#top)

  • Historically, href="#" was commonly used as a "back to top" link. While it often works, its primary meaning is just "the current document's base URL plus an empty fragment".
  • Some browsers interpret href="#top" as an instruction to scroll to the top, even without an explicit <... id="top"> element.
  • Best Practice: For clarity and reliability, if you want a dedicated "Back to Top" link, it's often better to add an empty element with a specific id near the beginning of your <body> (e.g., <div id="top-of-page"></div>) and link explicitly to that (href="#top-of-page"). This avoids ambiguity.

Common Use Cases for Fragment Links:

  1. Table of Contents: Creating a list of links at the beginning of a long document, where each link (href="#section-id") points to the corresponding section heading (id="section-id").
  2. "Back to Top" Links: Placing links (href="#top-of-page") at the end of long sections or in the footer to allow users to quickly return to the top without manual scrolling.
  3. FAQs: Linking directly from a question list to the corresponding answer element further down the page.
  4. Contextual Linking: Linking from one part of the text to a related definition, figure, or explanation elsewhere on the same page.

Code Change:

We've added:

  1. An empty <div> with id="top-of-page" at the beginning of the <body>.
  2. "Back to Top" links (<a href="#top-of-page">) within each section.
<body>
  <div id="top-of-page"></div> <!-- Optional explicit top target -->
  <h1>Linking to Page Fragments</h1>
  ...
  <h2 id="section1">Section One</h2>
  ...
  <p><a href="#top-of-page">Back to Top</a></p> <!-- Added Back to Top link -->
  ...
  <h2 id="section2">Section Two</h2>
  ...
  <p><a href="#top-of-page">Back to Top</a></p> <!-- Added Back to Top link -->
  ...
</body>

✅ Observation:

  1. Locate the new <div id="top-of-page"> and the "Back to Top" links in index.html.
  2. In the preview, scroll down to Section 1 or Section 2. Click the "Back to Top" link within that section. The page should scroll back up to the very top.

✨ Key Takeaway: Fragment links are widely used for creating tables of contents and "Back to Top" functionality, significantly improving navigation within long documents. Linking explicitly to an element with id="top-of-page" (or similar) is often more robust than relying on href="#".
