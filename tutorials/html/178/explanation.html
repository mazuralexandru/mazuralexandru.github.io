178. General Sibling Combinator (~)

Files Modified: style.css

The General Sibling Combinator is represented by the tilde symbol (~) between two selectors.

  • Syntax: element1 ~ element2
  • Meaning: Selects any element matching element2 only if it follows (not necessarily immediately) an element matching element1, and both elements share the same parent.

  • We've added a new rule: p ~ a. This selects any <a> element that follows any <p> element, provided they share the same parent. In our case, the <a> ("External Link") follows the <div> container, they don't share the same parent (<body> is the parent of both). Let's modify the selector to be more illustrative for our current structure. We'll change it to h1 ~ p which selects any <p> following an <h1> if they share the same parent (.container in this case).
/* General Sibling Combinator Example */
/* Selects any <p> that follows an <h1> within the same parent */
h1 ~ p {
  text-decoration: line-through; /* Add line-through for demonstration */
}
