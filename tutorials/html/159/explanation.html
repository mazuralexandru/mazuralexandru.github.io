159. ID Selectors: Example

Files Modified: index.html, style.css

Let's add an ID to our heading and style it using an ID selector.

✅ Observe index.html:

  • The <h1> element now has an id attribute with the value "main-heading".
<h1 id="main-heading">Main Heading</h1> <!-- Added ID -->

✅ Now, switch to style.css:

  • We've replaced the h1 type selector with the ID selector #main-heading.
  • The color property remains the same (cornflowerblue).
  • We've uncommented the text-decoration: underline; rule.
/* Replaced h1 type selector with ID selector */
#main-heading {
    color: cornflowerblue; /* Set heading color */
    text-decoration: underline; /* Added underline via ID */
}

✨ Observe the Preview:

  • The <h1> ("Main Heading") is still cornflowerblue, but now it also has an underline.
  • This style is applied because the ID selector #main-heading in style.css specifically targets the element with id="main-heading" in the HTML.
  • If we had other <h1> elements without this ID, they would not receive this styling (unlike when we used the h1 type selector).
