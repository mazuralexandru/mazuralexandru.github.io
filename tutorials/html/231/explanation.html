231. The alt Attribute - Definition & Purpose

Alongside src, the alt attribute is the other essential attribute for the <img> element.

The alt (Alternative Text) Attribute:

  • Definition: Provides a textual alternative that describes the content or function of the image.

  • Purpose (Multi-faceted and CRITICAL):

    1. Accessibility (Primary): This is the most important reason. Screen readers read the alt text aloud to users who are visually impaired, allowing them to understand what the image conveys. Without meaningful alt text, images are inaccessible barriers.
    2. Fallback Content: If the image file fails to load for any reason (broken URL, network error, user has images disabled), browsers display the alt text in place of the image. This provides context even when the visual is missing.
    3. SEO: Search engine crawlers cannot interpret the visual content of images directly. They rely heavily on the alt text (along with filenames and surrounding content) to understand what an image is about and index it correctly for image search results. Using relevant keywords appropriately in alt text can improve image SEO.

  • Requirement: The alt attribute is mandatory according to the HTML specification. Every <img> tag MUST have an alt attribute present for the HTML to be valid. (As we'll see, the value can sometimes be empty - alt="" - but the attribute itself must be there). Omitting alt entirely results in invalid HTML.
