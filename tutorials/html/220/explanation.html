220. text-indent

Files Modified: style.css

The text-indent property specifies the indentation applied to the first line of text within a block-level element.

  • Purpose: To create traditional paragraph indentation.
  • Applies To: Block container elements (like <p>, <div>).
  • Inherited: Yes.
  • Values:
    • <length>: Specifies a fixed indentation using units like px, em, rem. em is common as it scales with the font size. Negative values are allowed to create a "hanging indent" (where the first line sticks out).
    • <percentage>: Specifies indentation as a percentage of the containing block's width.
  • Keywords:
    • each-line: Indents every line (not just the first) - part of CSS Text Level 4, support may vary.
    • hanging: Inverts the indentation (like a negative value) - part of CSS Text Level 4, support may vary.

✅ Observe style.css:

  • We've added text-indent: 2em; to the .container > p ruleset. This will apply to both paragraphs inside the container.
/* Child Combinator Example */
.container > p {
  border-left: 3px solid orange; 
  padding-left: 10px;
  color: yellow; 
  text-indent: 2em; /* Indent first line of these paragraphs */
}

✨ Observe the Preview: The first line of both paragraphs inside the container ("This is a standard..." and "This paragraph follows...") should now be indented from the left edge (by an amount equal to twice their respective font sizes). Subsequent lines within each paragraph will wrap normally without the extra indent.
