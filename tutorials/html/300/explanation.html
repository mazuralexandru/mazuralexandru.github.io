300. Longhand Margin Properties: margin-left

Files Modified: style.css

Finally, margin-left.

  • Purpose: Sets the amount of margin space specifically on the left edge of the element, outside its left border.
  • Syntax: margin-left: value;
  • Values: Same as margin-top (lengths, percentages relative to container width, auto, negative values allowed).

✅ Observe style.css:

  • We've added margin-left: 10px; to the .box rule.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  margin-top: 2em; 
  margin-right: 50px; 
  margin-bottom: 3em; 
  margin-left: 10px; /* Added specific left margin */
  border-bottom: 4px solid red; 
  border-radius: 15px; 
}

✨ Observe the Preview: The blue box should now be pushed slightly away from the left edge of its containing area (the body) by 10 pixels. We now have different, explicit margins set on all four sides using the longhand properties.
