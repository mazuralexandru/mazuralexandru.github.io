233. <i>, <b>, <u>: Default Styling

Files Modified: index.html

Browsers render these elements with distinct default styles:

  • <i>: Italic text.
  • <b>: Bold text.
  • <u>: Underlined text.

These defaults often lead developers to use them purely for styling, but remember the semantic distinctions and the recommendation to prefer CSS classes for purely presentational effects.
