25. CSS Selectors: Introduction to Class Selectors

Type selectors apply to all elements of a certain type. What if you want to style some paragraphs differently from others, or apply a specific style to different types of elements? This is where Class Selectors become essential.

What's New?

  • We added a second <p> element to index.html.
  • Crucially, we added the class attribute with a value of highlight to this second paragraph.
<!-- index.html snippet -->
<body>
  <h1>Styling with External CSS</h1>
  <p>This paragraph will be styled externally.</p>
  <h2>This is a Sub-heading</h2>
  <p class="highlight">This paragraph has a highlight class.</p> <!-- Added P with class -->

  <!-- Commented out paragraph remains -->
  <!-- <p>This paragraph is commented out and won't appear.</p> -->

  <script src="script.js"></script>
</body>

The HTML class Attribute

  • Purpose: The class attribute allows you to assign one or more "class names" to an HTML element. These names act as labels or categories that you can then target with CSS (and JavaScript).
  • Syntax: <element class="classname"> or <element class="name1 name2 name3"> (multiple class names are separated by spaces).
  • Reusability: The same class name can be applied to many different elements, even elements of different types (e.g., you could have <p class="highlight"> and <span class="highlight">). This is key for creating reusable styles.
  • Naming Conventions:
    • Use meaningful, semantic names that describe the purpose or role of the element (e.g., error-message, main-navigation, product-card) rather than describing its appearance (e.g., avoid red-text, big-bold-title).
    • Use lowercase letters.
    • Separate words with hyphens (-), often called "kebab-case". (e.g., button-primary). Underscores (_) or camelCase (buttonPrimary) are less common in CSS class naming but technically valid.

In this step, we've added class="highlight" to the second paragraph. This doesn't change its appearance yet, because we haven't defined any CSS rules targeting the .highlight class.
