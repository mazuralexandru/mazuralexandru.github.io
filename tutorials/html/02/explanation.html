02. Declaring the Document Language

Let's add an important attribute to our <html> tag: the lang attribute.

What's New?

We added lang="en" to the opening <html> tag.

<!-- index.html snippet -->
<!DOCTYPE html>
<html lang="en"> <!-- Added lang attribute -->
<head>

Why Declare the Language?

The lang attribute specifies the primary language of the content within the <html> element (and usually, the whole document).

  • Accessibility: Screen readers use this attribute to pronounce the content correctly. Imagine a screen reader trying to read English text with French pronunciation rules â€“ it wouldn't work well!
  • Search Engines (SEO): Search engines can use this information to filter search results by language.
  • Translation Tools: Browsers might offer translation services based on this attribute.
  • Browser Behavior: Some browser features, like hyphenation or quotation marks, might be language-dependent.

Syntax:

  • lang is the attribute name.
  • en is the value, a standard language code (ISO 639-1 code). en stands for English. Other examples include es (Spanish), fr (French), ja (Japanese). You can find lists of these codes online (search for "BCP 47 language codes").
