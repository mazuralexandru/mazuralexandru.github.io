603. Graphics: SVG (<svg>) - Fundamentals

Let's introduce Scalable Vector Graphics (SVG).

Definition & Purpose:

  • SVG is an XML-based markup language for describing two-dimensional vector graphics.
  • It defines graphics using geometric elements like paths, shapes (circles, rectangles), and text, rather than pixels.
  • Purpose: To create graphics that are resolution-independent (scalable without quality loss), often have small file sizes (for non-photographic content), and can be easily styled with CSS and manipulated with JavaScript.

Core Concept:

  • You write SVG using declarative markup (XML tags and attributes) that describe the scene.
  • The browser parses this markup, builds an internal DOM tree for the SVG graphic (similar to the HTML DOM), and then renders the visual representation based on the defined shapes and styles. This is retained mode rendering.

Analogy Recap:

  • Describing a drawing with instructions ("Draw a 50px radius red circle at coordinate 100,100") rather than providing a grid of colored pixels.

Key Advantages Recap:

  • Scalability: Perfect quality at any zoom level or display resolution (essential for responsive design, high-DPI screens).
  • Small File Size: Often very compact for logos, icons, diagrams compared to equivalent bitmap formats.
  • Stylability: Can be styled using standard CSS (targeting elements by type, ID, class).
  • Scriptability: SVG elements are part of the DOM, accessible and modifiable via JavaScript. Event listeners can be attached directly to shapes.
  • Accessibility: Text within SVG is real text (searchable, selectable, readable by screen readers). Structure and semantics can be enhanced with elements like <title> and <desc>.
  • Text-Based Format: SVG files are XML text files, meaning they can be easily edited, gzipped effectively, and indexed by search engines.

✅ Observation: No code changes yet. Focus on the core nature of SVG: XML-based, vector description, retained mode rendering, and its key advantages, especially scalability and DOM integration.

✨ Key Takeaway: SVG is an XML-based vector graphics format for the web, offering resolution independence, CSS styling, DOM manipulation, accessibility benefits, and often small file sizes, making it ideal for logos, icons, and illustrations.
