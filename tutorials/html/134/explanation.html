134. The .css File

Files Modified: None

An external stylesheet is simply a plain text file containing only CSS rulesets (selectors, properties, values) and CSS comments.

  • Extension: It must be saved with the .css file extension (e.g., main.css, layout.css, theme.css).
  • Content: It contains only CSS code. You do not include any HTML tags like <style> inside a .css file.
  • Encoding: Like HTML files, CSS files should also be saved using UTF-8 encoding to ensure correct character interpretation, especially if using non-ASCII characters in comments or content properties (like content: 'Â©';).

You can create multiple .css files to organize your styles logically (e.g., one for base styles/resets, one for layout, one for specific components, one for theme variations).
