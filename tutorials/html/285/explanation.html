As mentioned, a visible border requires three key pieces of information:

  1. border-width: Defines the thickness of the border line(s). Can be set using keywords (thin, medium, thick) or length units (px, em, etc.). You can control the width of each side individually (border-top-width, etc.).
  2. border-style: Defines the appearance of the line. This is essential â€“ without a style other than none or hidden, the border won't render even if width and color are set. Common values include solid, dashed, dotted, double, groove, ridge, inset, outset. You can control the style of each side individually (border-top-style, etc.).
  3. border-color: Defines the color of the border line(s). Accepts standard color values (keywords, hex, rgb(a), hsl(a), currentColor). Defaults to the element's color property if not specified. You can control the color of each side individually (border-top-color, etc.).
