149. CSS Class Selector Syntax (.classname)

Files Modified: None

To target elements based on their class name in CSS, you use a class selector.

  • Syntax: A period (.) followed immediately by the class name (the exact name used in the HTML class attribute, without the quotes).

  • Examples:

    • To select all elements with class="highlight":
      .highlight {
  /* styles for highlighted elements */
}
    • To select all elements with class="button-primary":
      .button-primary {
  /* styles for primary buttons */
}
    • To select all elements with class="user-profile":
      .user-profile {
  /* styles for user profiles */
}

  • Case Sensitivity: While HTML class names themselves are technically case-sensitive in some contexts (like JavaScript document.getElementsByClassName), CSS class selectors in standards-compliant browsers typically treat them as case-sensitive. Best practice: Always use lowercase for class names in both HTML and CSS to avoid confusion and ensure consistency.
