331. Ordered List Example

Files Modified: index.html

Let's add an ordered list to show steps in a sequence.

✅ Observe index.html:

  • We've added an <ol> element containing three steps as <li> elements.
  • We've also added an example using the type and start attributes.
    <!-- Ordered List Example -->
    <h2>Morning Routine</h2>
    <ol>
        <li>Wake up</li>
        <li>Brush teeth</li>
        <li>Make coffee</li>
    </ol>

    <h2>Top 3 (Starting from 3)</h2>
    <ol type="I" start="3"> <!-- Roman numerals, starting at III -->
        <li>Third Place</li>
        <li>Fourth Place</li>
        <li>Fifth Place</li>
    </ol>

✨ Observe the Preview:

  • You should see the "Morning Routine" list with items numbered 1, 2, 3.
  • Below that, the "Top 3" list should appear with Roman numeral markers (III, IV, V) because we specified type="I" and start="3".
  • Both lists are indented by default.
