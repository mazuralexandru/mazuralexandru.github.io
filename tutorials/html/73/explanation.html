73. <meta charset="UTF-8">: Syntax

Files Modified: index.html

The character encoding is declared using a <meta> element with the charset attribute, placed inside the <head>.

  • Syntax: <meta charset="UTF-8">
  • Element: <meta> is a void element used for various types of metadata.
  • Attribute: charset specifically declares the character encoding.
  • Value: "UTF-8" is the value specifying the encoding to use. (Case-insensitive, but UTF-8 is standard).
