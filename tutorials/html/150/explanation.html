150. Class Selectors: Scope

Files Modified: None

Unlike type selectors that target all elements of a certain type, class selectors target only those elements that have been explicitly assigned the matching class name in their class attribute.

  • Specificity: A class selector (.my-class) is more specific than a type selector (p). This means if both a type selector and a class selector apply conflicting styles to the same element, the style from the class selector will generally win (we'll cover specificity in detail later).
  • Targeting: A single class selector (e.g., .important) will apply its styles to any element that has class="important", whether it's a <p>, <span>, <li>, or <div>.
  • Multiple Elements: The same class can be used on many different elements throughout the page, allowing you to apply a consistent style to all of them.

This targeted scope makes class selectors ideal for creating reusable styles.
