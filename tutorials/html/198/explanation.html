198. Telephone Links (tel:): Syntax & Formatting

Let's look at the syntax for creating telephone links using the tel: scheme.

Basic Syntax:

  • href="tel:phonenumber"

Phone Number Formatting (RFC 3966 Recommendation):

  • Best Practice: For maximum compatibility and clarity, especially for international users, format the phone number in the href value according to RFC 3966. This generally means:
    • Start with a plus sign (+).
    • Follow with the country code (e.g., 1 for North America, 44 for UK).
    • Follow with the rest of the phone number digits only, with no spaces, hyphens, parentheses, or other separators.
  • Link Text Formatting: The visually displayed link text between the <a>...</a> tags should include standard formatting (spaces, hyphens, parentheses) for readability. The strict digit format applies only to the href value itself.

Example:

  • Number: (555) 123-4567 (North America)
  • href Value: tel:+15551234567
  • Link Text: (555) 123-4567 or Call (555) 123-4567

Code Change:

We've added an anchor element with a tel: link to index.html, following the recommended formatting in the href and using human-readable formatting in the link text.

<p>
  <!-- Recommended format: +countrycode(number) -->
  <a href="tel:+15551234567">Call Us at (555) 123-4567</a> <!-- Added basic tel link -->
</p>

✅ Observation:

  1. Examine the <a> tag in index.html. Note the href="tel:+15551234567" format (digits only after +1).
  2. Compare this to the link text displayed in the preview: "Call Us at (555) 123-4567", which includes formatting for readability.
  3. Clicking the link will attempt to trigger a call handler (phone app, desktop software). The behavior depends entirely on your device and setup. It won't navigate to a web page.

✨ Key Takeaway: Use href="tel:phonenumber" for click-to-call links. Format the number in the href using the international standard (+countrycodedigitsonly) for best results. Use readable formatting in the visible link text.
