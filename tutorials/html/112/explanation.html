112. Ordered Lists (<ol>): The reversed Attribute

HTML5 introduced the reversed attribute for <ol> elements, allowing you to easily create lists that count down instead of up.

The reversed Attribute:

  • Purpose: Specifies that the list items should be numbered in descending order.
  • Syntax: <ol reversed>
  • Type: It's a boolean attribute. Its presence enables the reversed numbering; its absence means normal ascending order. You don't need to set a value (like reversed="true").
  • Behavior: The browser numbers the items starting from the total number of items in the list down to 1. (This interacts with the start attribute, which we'll see next).

Use Cases:

  • Creating countdown lists (e.g., "Top 3 reasons... 3. ..., 2. ..., 1. ...").
  • Displaying rankings in descending order (e.g., 10th place down to 1st).

Code Change:

We've removed the start attribute from the previous step and added the reversed attribute to the <ol> tag in index.html. We've also reordered the <li> text content conceptually to match a countdown, although the numbering is determined by the attribute, not the content order.
