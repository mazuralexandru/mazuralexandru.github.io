195. Relative Units Recap

Files Modified: None

Relative length units define a size based on some other value, such as the parent element's font size, the root element's font size, or the size of the viewport (the browser window's visible area). They are key to creating flexible, scalable, and responsive designs.

We'll recap the main types:

  • Percentage (%): Relative to a value from the parent element.
  • Em (em): Relative to the font-size of the element itself (or its parent for font-size).
  • Rem (rem): Relative to the font-size of the root (<html>) element.
  • Viewport Units (vw, vh, vmin, vmax): Relative to the dimensions of the viewport.

Using relative units allows layouts and text to scale more naturally, adapting better to different screen sizes and user preferences (like adjusted default font sizes).
