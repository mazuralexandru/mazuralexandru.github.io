411. Decorative Styling: box-shadow

Files Modified: style.css

The box-shadow property adds shadow effects to an element's box.

  • Purpose: Creates drop shadows or inner shadows to give elements a sense of depth or emphasis.
  • Syntax: box-shadow: [inset]? h-offset v-offset blur-radius spread-radius color [, ...];
    • inset (Optional): Changes the shadow from an outer drop shadow (default) to an inner shadow.
    • h-offset: Horizontal offset (positive = right, negative = left). (Required)
    • v-offset: Vertical offset (positive = down, negative = up). (Required)
    • blur-radius (Optional): Larger values increase blur; 0 = sharp shadow. Defaults to 0.
    • spread-radius (Optional): Positive values expand the shadow size, negative values shrink it. Defaults to 0.
    • color (Optional): Color of the shadow. Defaults to the element's color if omitted, but often specified explicitly (e.g., semi-transparent black rgba(0,0,0,0.5)).
    • Multiple Shadows: You can apply multiple shadows by comma-separating the definitions.
  • Difference from text-shadow: box-shadow applies to the element's entire box, following its border-radius, while text-shadow applies only to the text content itself.

✅ Observe style.css:

  • We've added a box-shadow to the .block-image rule.
.block-image {
    display: block; 
    max-width: 100%; 
    height: auto; 
    width: 300px; 
    margin: 2em auto 3em auto; 
    border: 4px solid steelblue; 
    padding: 10px; 
    background-color: #444; 
    border-radius: 15px; 
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.6); /* Add drop shadow */
}
