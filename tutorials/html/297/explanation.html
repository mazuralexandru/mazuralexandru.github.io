297. Longhand Margin Properties: margin-top

Files Modified: style.css

Similar to padding, you can control margins on each side individually. Let's start with margin-top.

  • Purpose: Sets the amount of margin space specifically on the top edge of the element, above its top border.
  • Syntax: margin-top: value;
  • Values: Accepts lengths (px, em, rem, etc.), percentages (% - relative to the width of the containing block), or the keyword auto. Negative values are allowed and will pull the element (or subsequent elements) upwards.

✅ Observe style.css:

  • We've removed the margin: 1em auto; shorthand.
  • We've added margin-top: 2em;.
.box {
  width: auto; 
  height: auto; 
  min-width: 200px; 
  max-width: 400px; 
  min-height: 150px; 
  max-height: 200px; 
  overflow: auto; 
  background-color: cornflowerblue; 
  padding: 20px 5px 15px 30px; 
  /* margin: 1em auto; <- Removed shorthand */
  margin-top: 2em; /* Added specific top margin */
  border-bottom: 4px solid red; 
  border-radius: 15px; 
}

✨ Observe the Preview: The blue box should now have more space above it compared to the previous step (2em instead of 1em). Since we removed the auto for left/right margins, the box is likely no longer centered horizontally and defaults to the left. We only have top margin applied currently.
