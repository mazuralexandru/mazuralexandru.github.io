446. background-attachment: Syntax

Files Modified: None

The syntax is straightforward:

background-attachment: scroll | fixed | local [, scroll | fixed | local]*;

  • You provide one of the keywords (scroll, fixed, or local).
  • If multiple background images are defined, you can provide a comma-separated list of attachment values, corresponding to each image layer.
    • Example: background-attachment: fixed, scroll; (First layer is fixed, second layer scrolls).
