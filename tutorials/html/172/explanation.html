172. Grouping Selectors: Example Recap

Files Modified: None

Let's quickly recap the grouping example:

HTML (index.html):

    <p class="highlight-text" title="Standard Paragraph">...</p>
    <h1 id="main-heading">Main Heading</h1>
    <a href="https://example.com" target="_blank">External Link</a>

CSS (style.css):

/* Individual Styles */
#main-heading {
    color: cornflowerblue; 
    text-decoration: underline; 
}
.highlight-text {
  background-color: #4d4d00; 
  color: #1a1a1a; 
  padding: 5px; 
}
/* ... attribute selectors ... */

/* Grouped Style */
#main-heading,
.highlight-text {
  font-style: italic; /* Apply italic to both */
}
  • The comma (,) separates the #main-heading selector and the .highlight-text selector.
  • The declaration block { font-style: italic; } applies to any element matching either selector.
  • Therefore, both the <h1> with the ID main-heading and the <p> with the class highlight-text are rendered in italics.

This demonstrates how grouping allows applying shared styles efficiently.
