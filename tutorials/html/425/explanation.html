425. Styling <figcaption>: Spacing

Files Modified: style.css

Finally, let's add some space between the image and the caption using margin-top.

✅ Observe style.css:

  • We've added margin-top: 0.5em; to the figcaption rule.
figcaption {
    font-size: 0.9rem; 
    font-style: italic; 
    color: #bbb; 
    text-align: center; 
    margin-top: 0.5em; /* Add space above the caption */
}

✅ Switch to index.html to see the HTML structure.

✨ Observe the Preview: There should now be a noticeable vertical gap (0.5em) between the bottom of the image and the top of the centered caption text within the figure.
