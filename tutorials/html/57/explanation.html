57. <head>: Permitted Content Overview

Files Modified: None

What kind of elements can you put inside the <head> section? Only specific metadata content is allowed. The main ones include:

  • <title>: (Required in valid HTML5). Defines the document's title. Must contain only text. Exactly one is allowed.
  • <meta>: Used for various kinds of metadata that don't have their own dedicated element (e.g., character set, viewport settings, description, keywords). Can appear multiple times.
  • <link>: Specifies relationships between the current document and external resources (most commonly CSS stylesheets, but also favicons, preloads, etc.). Can appear multiple times.
  • <style>: Contains CSS style information directly embedded within the HTML document. Can appear multiple times.
  • <script>: Embeds executable script (usually JavaScript) or links to an external script file. Can appear multiple times. (Note: Often better placed before the closing </body> tag for performance reasons, but allowed in <head>).
  • <base>: Specifies the base URL/target for all relative URLs in a document. At most one is allowed.
  • <noscript>: Defines content to be shown if scripting is disabled or not supported by the browser. Can contain <link>, <style>, and <meta> elements when used in the <head>.

You cannot put elements meant for visible content (like <h1>, <p>, <img>, <div>) directly inside the <head>.
